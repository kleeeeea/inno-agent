# 即时纪律处置与冲突降级

## When to use
- 讲课过程中学生持续交头接耳、传纸条、玩物品等干扰教学行为
- 学生顶嘴、接话、公开质疑教师指令（如"我又没说"）
- 教师感到情绪失控、想要提高音量吼叫或讽刺学生
- 2-3名带头学生反复试探边界，影响全班注意力
- 需要课堂内即时处置且快速恢复教学节奏

## When not to use
- **开学第一课立规矩、进班30-60秒压场** → 转 `classroom-rule-establishment` 技能处理
- **学生出现自伤/他伤倾向、肢体冲突、严重欺凌** → 立即暂停授课，保护现场安全，呼叫班主任/德育处介入
- **涉及特殊教育需求学生（疑似ADHD/自闭症/情绪障碍）** → 仅提供通用策略，明确建议转介学校特教资源教师或要求家长提供医学诊断
- **教师出现失眠、焦虑、逃避上课等职业倦怠症状** → 建议寻求心理支持，非单纯技术问题
- **家校冲突涉及法律纠纷** → 建议咨询学校法务或上级主管部门

## Instructions
1. **非语言优先**：处置顺序为"站位靠近→眼神凝视→停顿沉默→短句话术"，避免隔空喊话
2. **话术标准**：短句（不超过15字）、语速慢（正常1/2速）、落点重、不解释、不争辩
3. **三段式结构**：所有方案必须包含"具体动作+精准话术+明确后果"，缺一不可
4. **冷处理原则**：面对顶嘴不接招，重复规则句一次后立即回归教学，避免"对话拉长变舞台"
5. **一致性底线**：说过的规则必须执行，不随意加码，绝不空转警告

## Workflow
1. **干扰分级**：判断当前为轻度（偶发）/中度（持续）/重度（对抗）/情绪危机（教师或学生失控）
2. **层级处置**：
   - 第一层：无声靠近，站旁继续授课
   - 第二层：暂停讲课，凝视目标，全班随目光安静
   - 第三层：点名并触发课前公布的后果（记录/补做/约谈/座位调整）
   - 第四层：冷处理降级，避免情绪对抗
3. **即时执行**：按"动作-话术-后果"模板现场操作，不拖堂处理
4. **记录归档**：简要记录时间、行为、处置方式，用于课后跟进与家校沟通

## Output Format
```markdown
**场景判断**：[干扰类型] + [风险等级：绿/黄/橙/红]

**处置方案**：
- 动作：[站位/走位/手势/停顿时长]
- 时机：[何时开口/何时沉默]
- 目标：[预期达到的状态]

**可用话术**（选1句，标注语速与重音）：
- "说完了吗？"（慢速，重音在"吗"）
- "我给你时间、给你空间：上讲台说。"（停顿在冒号处）
- "你现在有两个选择：停止并继续听课；或者按规则处理。"（重音在"选择""规则"）

**后果说明**：
- 校内允许措施：[记录/补做/课后约谈/座位调整等]
- 执行时限：[当场/课后/次日]
- 记录方式：[登记地点与反馈路径]

**风险提示**：
- 避免：[情绪化语言/肢体拉扯/当众羞辱/翻旧账]
- 红线：[学生握拳对峙/涉及身份歧视/教师声音颤抖]→立即暂停，保护安全

**后续评估**：
- 课后跟进：[约谈时间/沟通要点/家校联络必要性]
- 迭代建议：[规则是否需要调整/下次类似情况优化点]
```

## Examples

**示例1：后排两人持续聊天**
- 场景判断：中度干扰（持续说话），黄色等级
- 处置方案：边讲课边走近→站在两人旁继续讲10秒→仍不改则停顿凝视→全班安静后说"按第一课规则处理"
- 可用话术："（走近后停顿）说完了吗？"（慢速，重音"吗"，等待3秒）
- 后果：当场记录姓名，课后约谈，次日课前确认改正

**示例2：学生顶嘴"我又没说"**
- 场景判断：轻度对抗（语言试探），橙色等级
- 处置方案：面无表情注视3秒→重复规则句一次→立即回归教学内容→课后单独处理
- 可用话术："（注视）你现在有两个选择：停止并继续听课；或者按规则处理。"（不解释"说没说"）
- 后果：课后约谈，明确"课堂不辩论"原则，记录备案

**示例3：教师忍不住想吼叫**
- 场景判断：情绪危机（教师失控前兆），红色等级
- 处置方案：立刻闭嘴沉默2秒→降低语速至1/2→用规则后果替代情绪→必要时短暂停课凝视全班
- 可用话术："（深呼吸）我给你们30秒。30秒后我们继续，或者按规则处理。"（极慢速，重音"30秒""规则"）
- 后果：课后教师自我复盘，使用记录表统计触发点，调整下周执行策略

## Common Issues
- **话术冗长谈判**：错误："你为什么说话？你不知道这样影响别人吗？你再这样我就..." → 正确："说完了吗？"
- **隔空喊话无效**：错误：在讲台前大声点名 → 正确：走到学生身旁再开口
- **只警告不执行**：错误："再这样就惩罚"（实际不罚）→ 正确：第一课公布的规则必须次次落实
- **陷入辩论陷阱**：错误：回应"我又没说"的细节 → 正确：重复规则句，不解释，课后处理
- **忽视非语言信号**：错误：边讲课边警告 → 正确：先停顿、凝视、站位压制，再开口
