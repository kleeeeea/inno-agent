# 课堂规则建立与开场控场

## When to use
- **开学第一课**：需制定并宣告课堂规则、建立奖惩机制、设计班规可视化内容（PPT/板书/海报）
- **课前进班**：学生嘈杂打闹，需通过30-60秒无声压场快速建立秩序，为教学开场铺垫
- **前两周执行规划**：制定规则落地的分层处置节奏（提醒→记录→约谈→家校沟通）

## When not to use
- **课堂进行中的即时干扰**（学生交头接耳、顶嘴接话、教师情绪失控）→ 转交 **即时纪律处置与冲突降级** 技能
- **安全红线情况**：学生自伤/他伤倾向、肢体冲突、严重欺凌、情绪激烈对峙 → 立即暂停授课，呼叫班主任/德育处，必要时建议心理转介
- **特殊教育需求**：涉及疑似ADHD/自闭症等未确诊学生的个别化教学调整 → 建议联系学校特教资源教师或要求家长提供医学诊断
- **家校法律纠纷**：涉及教育惩戒的法律争议或投诉 → 建议咨询学校法务或上级主管部门

## Instructions
1. **规则前置**：第一课必须完成"允许/禁止/后果"的可视化契约，避免后续口头重复无效
2. **低语言高气场**：优先使用非语言信号（站姿挺拔、面无表情扫视、停顿沉默）建立权威，减少解释性语言
3. **合规底线**：班规后果必须与《中小学教育惩戒规则（试行）》及校规一致，严禁罚款、羞辱性惩罚、超时限罚站
4. **一致性至上**：前两周执行节奏重于强度，说过的必须做到，不轻易加码但绝不空转

## Workflow
1. **场景判断**：确认当前处于"开学第一课规则建立期"还是"课前进班开场控场期"
2. **方案生成**：按"动作-话术-后果"三段式输出，动作优先非语言，话术短句慢速重音，后果具体可执行
3. **合规校验**：检查班规是否与校规冲突，惩戒措施是否在《教育惩戒规则》允许范围内
4. **执行规划**：制定前两周高一致性执行日历，明确分层处置触发条件（第几次违规对应何种措施）

## Output Format
- **场景判断**：[开学第一课规则建立 / 课前进班压场 / 前两周执行规划]
- **处理方案**：
  - **动作**：[具体站位、走位、眼神、手势、板书等非语言行为步骤]
  - **话术**：[短句、慢速、落点重、不解释，可直接背诵的脚本]
  - **后果**：[与校规一致的即时处置或课后跟进措施，分层明确]
- **可用话术库**：[3-5个短句模板，如"说完了吗？""按规则处理。"]
- **风险提示**：[班规与校规冲突检查 / 违法惩戒预警 / 特殊学生识别提醒]
- **后续评估**：[课堂纪律记录表模板 / 前两周执行检查点 / 量化指标（如违规次数/压场所需时间）]

## Examples
**示例1：开学第一课1分钟规则宣告**
- **场景判断**：开学第一课规则建立期
- **处理方案**：
  - **动作**：板书三条规则（允许/禁止/后果）；转身扫视全班15秒；指向规则逐条解释；张贴可视化班规
  - **话术**："三条底线，说清执行。第一，允许举手发言，禁止随意插话，违反先提醒再记录。第二... 这不是商量，是契约。前两周，无一例外。"
  - **后果**：首次提醒→第二次课堂记录→第三次课后约谈并告知家长（依据校规第X条）
- **可用话术库**：["三条规则，说过就执行。", "前两周，没有例外。", "这不是针对谁，是课堂运行的底线。"]
- **风险提示**：确认"记录""约谈"在校规中有明确依据；禁止"罚款""罚抄课文50遍"等违法惩戒
- **后续评估**：记录每日每节课违规次数，目标第一周下降至≤3次/节课

**示例2：课前进班30-60秒压场**
- **场景判断**：课前进班开场控场
- **处理方案**：
  - **动作**：进门→上讲台→放书→双手背后/自然收拢→面无表情扫视全班30-60秒→等待学生翻书拿材料后开口
  - **话术**："（停顿2秒）准备好了，开始上课。"
  - **后果**：未准备材料者按第一课公布的规则当场记录姓名，课后补做并约谈
- **可用话术库**：["（无声凝视）", "我给你们时间准备。", "材料准备好，我们开始。"]
- **风险提示**：压场期间避免说话或表情愤怒；若学生持续打闹不动，记录姓名课后处理，避免当众吼叫升级冲突
- **后续评估**：记录压场所需时间，目标3-5次课后30秒内全班安静

## Common Issues
1. **班规过于笼统无法执行** → 后果必须具体、可量化、与校规条款对应，避免"表现不好就罚"等模糊表述
2. **前两周执行不一致导致权威崩塌** → 制定分层处置清单（提醒→记录→约谈），严格执行，不因"不好意思"而空转
3. **压场时教师尴尬或笑场** → 提前默数30秒，保持面无表情，不开口不解释，用身体姿态而非情绪施压
4. **班规涉及违法惩戒** → 对照《教育惩戒规则》第十二条，将"罚款""羞辱性惩罚"改为"记录""补做""座位调整""课后约谈"等合规措施
