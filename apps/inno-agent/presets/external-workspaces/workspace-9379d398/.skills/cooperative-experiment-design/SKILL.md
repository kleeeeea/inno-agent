# 小组合作实验方案设计

## When to use

- 设计特定实验的4人小组合作流程与分工方案
- 制作"良好合作"与"不良合作"的具象化示范脚本或视频方案
- 制定预防材料争抢、保障实验安全的规则与站位图
- 设计符合低年级认知的实验任务卡与角色描述（如"材料小管家""操作小能手"）
- 规划"示范→分工→实验→复盘"完整教学闭环的课前准备清单

## When not to use

- **课中即时干预**：学生正在争吵、抢材料或出现行为冲突时，转交 `classroom-behavior-guidance` 处理
- **学科内容判断**：涉及实验本身的科学性评价（如测量方法是否正确），应由学科教师自主判断
- **高危实验安全**：涉及酒精灯、尖锐器具等高危器材，必须优先遵循《小学科学实验安全规范》，建议改为教师演示
- **纪律与心理问题**：学生长期不参与、情绪障碍或班级管理问题，转交班主任或心理老师处理

## Instructions

1. **具象化原则**：将"合作"转化为可见动作（如【手指放在嘴唇上】表示倾听，【传递材料时双手交接】表示轮流），禁止抽象概念教学（如"讲解合作的重要性"）
2. **三层分工架构**：第一层（自主竞选/猜拳/轮流）→ 第二层（小组协商解决）→ 第三层（教师兜底指定），每层需明确触发条件与话术
3. **闭环完整性**：确保方案覆盖"示范→分工→实验→复盘"四环节，标注各环节时间分配（课前5分钟/课中20分钟/课后3分钟）
4. **安全前置**：材料发放规则必须先于实验操作明确，⚠️标注易碎品（温度计、玻璃器皿）的管控策略

## Workflow

1. **情境诊断**：确认学段（1年级/2年级）、实验类型、学生合作基础，识别核心痛点（分工不清/行为失范/复盘缺失）
2. **具象化示范设计**：制定可见动作脚本（如"一人拿恐龙模型→一人拉软尺→一人记录→一人汇报"），设计对比案例（良好vs不良合作）
3. **分层分工方案**：为4人小组设计具体角色（材料员/操作员/记录员/汇报员），明确三层决策路径与兜底指令
4. **材料管控规则**：制定"规则先行→站位固定→分批发放"的安全流程，设计材料回收信号
5. **复盘框架预设**：设计实验后3分钟的快速复盘问题（如"谁摸到了材料？""谁的主意被采用了？"），确保全员参与

## Examples

**输入**：设计二年级"磁铁吸什么"4人小组合作流程，学生无合作基础，常抢材料。

**输出**：

【场景判断】
二年级/对比观察实验/零合作基础/材料争抢与分工不清

【处理方案】
示范脚本：【教师与3名学生演示】→【材料员从篮子取磁铁（双手接）】→【操作员悬空持磁铁（不触碰物品）】→【记录员贴贴纸标记】→【汇报员举手示意】→【四人对视点头换角色】
分工策略：第一层（学生自主猜拳决定谁先当材料员）→第二层（小组内商量"谁最喜欢画画谁记录"）→第三层（教师指定："小明先拿材料，小红先操作"）
材料规则：【U型围坐，材料篮放中间】+【教师喊"材料员来领"后分批发放（先磁铁，后记录纸）】+【听到音乐响起放回材料】
时间分配：示范（5分钟）→分工（3分钟）→实验（15分钟）→复盘（3分钟）

【可用话术】
示范阶段：「看老师的手，拿磁铁要像捧鸡蛋一样轻」
分工冲突：「如果吵得没办法，老师来当裁判，这次小红先操作，下次小明先」
材料管控：「我数到3，只有材料员站起来，1——2——3——」

【风险提示】
⚠️ 安全红线：磁铁含小零件需防吞咽，实验前检查无破损
💡 边缘化预防：设置"轮流触摸"规则，确保每人至少操作2次

【后续评估】
下次实验需强化的衔接点：强化"听到音乐就交换角色"的自动切换习惯

## Common Issues

1. **"时间不够，完不成教学任务"** → 建议初期牺牲课时进度，用1-2节课培养习惯，重点完成"示范+分工"闭环，实验可简化
2. **"学生都想当操作员，拒绝记录"** → 设计角色具象化描述（"记录员是给科学家画画的画家"），明确"轮流"机制（下次交换）
3. **"分工词太抽象，学生听不懂"** → 替换"协调员""观察员"等抽象词，改用"材料小管家""眼睛亮亮员"等动作化称呼
4. **"材料一发放就混乱"** → 执行"规则冻结"：发放前全班复述"谁拿→怎么拿→放哪里"，违规立即收回重新训练
5. **"复盘时只有组长发言"** → 预设"随机抽查"机制（抽学号而非举手），设计具体回忆问题（"刚才谁发现了磁铁不能吸木头？"）
