# 课堂实施与行为引导

## When to use

- **材料冲突**：学生争抢温度计、磁铁等实验器材，出现安全隐患或秩序混乱
- **分工失衡**：组内2人操作、2人旁观，或都想当"操作员"拒绝记录/汇报
- **行为失范**：讨论时全员抢话无人倾听，或学生声称"自己做更快"拒绝协作
- **边缘化干预**：内向学生持续躲在后排不操作、不说话，或优秀生垄断全部任务
- **复盘失效**：汇报时仅组长发言，其他成员低头沉默，或遗忘前次合作规则

## When not to use

- **课前方案设计**：设计示范脚本、分工方案、任务卡、材料规则制定 → 转 **cooperative-experiment-design**
- **学科内容判断**：评价实验设计科学性（如"测量方法是否正确"）
- **非课堂管理**：班级纪律处分、学生心理问题、家校沟通事宜
- **高危实验**：涉及酒精灯、尖锐器具等需优先执行《小学科学实验安全规范》并改为教师演示

## Instructions

1. **具象化干预**：用可见动作替代抽象说教，如【手指放嘴唇】表示倾听，【指向材料】明确归属，避免"要好好合作"等空泛指令
2. **三层递进**：第一层（自主竞选/猜拳）→ 第二层（组内协商）→ 第三层（教师兜底指定），每层配有明确话术
3. **即时稳控**：冲突发生时先【收回材料】稳定情绪，再发指令，严禁在争抢中继续发放器材
4. **公平锚点**：满足"重公平、求归属"心理，强调"轮流""每个人都有任务""刚才XX的方法很好"
5. **安全红线**：⚠️出现器材挥舞、争抢玻璃制品时，立即暂停实验，全体收回材料重新定规则

## Workflow

1. **情境扫描**（10秒内）：判断冲突类型（材料/分工/行为/边缘化），确认是否需立即收回材料
2. **即时干预**（30秒）：使用【动作+短指令】稳定秩序，如"【拍手】材料回盒，【指耳朵】听规则"
3. **行为重塑**（1-2分钟）：示范具体合作动作（如"轮流触摸"手势），重启实验并指定下一轮操作者
4. **课末复盘**（3分钟）：用"谁来说说你刚才做了什么"替代"谁来说说结论"，确保全员发言
5. **衔接强化**（预告下次）：指出本次合作亮点，预告下节课将使用的相同合作规则

## Output Format

- **场景判断**：冲突类型（材料/分工/行为/边缘化）+ 紧急程度（高/中/低）
- **处理方案**：
  - 第一层（自主）：学生自主解决路径
  - 第二层（协商）：组内调解话术与动作
  - 第三层（教师兜底）：直接指令与角色指定
- **可用话术**：原样输出教师可说的短句（标注【动作】），如"「我数到3，材料组长来领」【竖手指】"
- **风险提示**：⚠️安全预警（器材损坏风险）+ 💡边缘化学生干预要点
- **后续评估**：下次课需强化的合作行为点，及前测（上次习惯是否保留）

## Examples

**输入**：学生抢温度计差点打碎，另外2人站着看，我喊"要合作"没用

**输出**：

- **场景判断**：材料争抢+分工失衡，紧急程度：高（玻璃器材）
- **处理方案**：
  - 第一层：材料已混乱，跳过自主
  - 第二层：【收回材料】"温度计回盒，【双手交叉】暂停30秒"
  - 第三层："【指向A】你来拿温度计，【指向B】你来读数，【指向C】你记录，【指向D】你检查。A测完→传给B，B测完→C记录，轮流来"
- **可用话术**：
  - 「材料太危险，先回老师这里」【伸手收材料】
  - 「这次A先拿，下次实验B先拿，公平轮流」【手指轮流比划】
  - 「站着看的两位，【做出耳朵手势】你们当监督员，看谁做到了轮流」
- **风险提示**：⚠️玻璃温度计必须在教师视线内，下次建议改用塑料测温计；💡原旁观的2人赋予"监督员"角色而非强制操作，降低参与门槛
- **后续评估**：下次课前需检查"轮流规则"是否保留，若遗忘需重新示范一次

## Common Issues

- **"要合作"口号无效**：避免抽象词汇，改用"【指】你先拿，【指】你后拿，轮流"的具体动作指令
- **猜拳协商失败**：若猜拳超过30秒未决，立即执行第三层："老师指定，这次A操作，下次B操作"
- **复盘时间不足**：即使只有1分钟，也必须完成"每人说一句话"，牺牲汇报结论也要保证合作行为复盘
- **内向学生持续不参与**：不强制发言，先赋予低风险角色（如"拿材料的""递东西的"），逐步过渡到操作员
- **"自己做更快"抵触**：回应"这次练习轮流，就像排队滑滑梯，必须等前面的人"，强调过程价值而非速度
