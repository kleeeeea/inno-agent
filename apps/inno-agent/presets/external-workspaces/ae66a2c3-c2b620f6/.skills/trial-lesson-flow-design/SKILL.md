# 试听首课流程设计

## When to use
- 用户需要**精确到分钟**的40分钟或60分钟首课时间表（诊断-讲练-反馈三段式）
- 涉及**课堂环节设计**：学情核验提问、小切口选题、同型变式练习、当堂验证闭环
- 处理**分类型教学策略**：主动型学生（高密度）vs 被动型学生（安全感与配合意愿）
- 应对**课堂突发要求**：家长要求讲多道题、学生抗拒配合、家长旁听导致紧张等现场调整
- 备课内容选择：基于有限信息准备两套备选方案及选择规则

## When not to use
- 涉及**课后家长沟通话术**（如承诺边界"能提高多少分"、转化失败复盘、续费原因解释）→ 转 **diagnostic-feedback-communication**
- 需要**学情诊断工具设计**（梯度出题验证真实水平、区分"本来会"vs"被教会"的方法）→ 转 **diagnostic-feedback-communication**
- 需要**结构化课后反馈报告模板**（问题定位+原因+计划+预期）→ 转 **diagnostic-feedback-communication**
- 学生出现**严重情绪崩溃或心理问题**（超出教学范围）→ 建议暂停教学并转介专业心理咨询

## Instructions
1. **三段式刚性结构**：首课必须包含"学情核验（5-7分钟）→ 小切口讲练（25-40分钟）→ 当堂验证+结课（8-10分钟）"，禁止单段超时导致结构坍塌
2. **少讲多练原则**：每讲1道代表题，必须配2-3道同型变式，且要求学生**口述思路**完成闭环
3. **学生类型适配**：主动型突出效率与密度；被动型优先"听得懂"与"愿意配合"，允许降低难度换取参与度
4. **合规审查**：自动检查输出是否包含"保提分""保及格"等违规承诺，禁止建议羞辱性评价

## Workflow
1. **需求解析**：确认课时长度（40/60分钟）、学生类型（主动/被动）、家长是否旁听、信息完整度
2. **时间分配**：按三段式分配具体分钟数，标注每段必须产出的**交付物**（诊断结论/掌握点/家长反馈）
3. **内容选择**：基于学情信息缺失程度，提供"备两套方案"的决策规则（题材/难度差异）
4. **分支策略**：针对家长要求讲多道题、学生抗拒、家长旁听等场景，提供课堂内的即时调整方案
5. **风险检查**：确认无保分承诺、无超时讲课建议、无羞辱性话术

## Output Format
每次输出必须包含以下固定字段：

**场景判断**：课时长度/学生类型/特殊约束（旁听/信息缺失/家长要求）

**时间轴（精确到分钟）**：
- 诊断段（5-7分钟）：动作+产出物
- 讲练段（25-40分钟）：选题策略+讲练配比+验证动作
- 结课段（8-10分钟）：当堂验证方式+面向家长的交付物

**处理方案**：
- 标准流程（线性流）
- 分支应对（条件A-D：家长要求讲错题/自带备课/主动型/被动型）

**备选策略**：若主方案受阻（如学生严重抗拒）的降级方案

**风险提示**：合规边界（禁止保分）、教师权益边界（禁止超时讲课）、学生心理安全

### 字段缺失风险提示
若输出中缺少以下任一字段，方案完整性将受损：
- **无场景判断**：教师无法确认该方案是否适配当前学生类型
- **无时间轴**：课堂节奏易失控，导致前松后紧或无法完成验证
- **无备选策略**：遇突发情况（如学生突然抗拒）时教师无应对预案
- **无风险提示**：可能产生合规风险或教师权益受损

## Examples

**示例1：40分钟被动学生流程**
- 场景判断：40分钟/被动抗拒/家长旁听/信息缺失
- 时间轴：
  - 0-7分钟：轻量学情核验（教材版本+兴趣点）→ 产出：建立安全感
  - 7-32分钟：选1道基础代表题，讲透后做2道同型变式，学生口述思路 → 产出：确认掌握1个点
  - 32-40分钟：独立练习时段（让家长暂时离开）+ 正向反馈 → 产出：学生配合意愿评估
- 处理方案：优先体验感，密度让位于参与度
- 风险提示：禁止强行推进进度，禁止在家长面前指出"基础差"

**示例2：家长要求讲10道错题**
- 场景判断：60分钟/家长提出超量要求
- 时间轴：
  - 0-5分钟：肯定家长准备，提出"讲透比讲多更重要"原则
  - 5-10分钟：从10道中快速挑选1-2道最具代表性的（知识交叉点/高频失分点）
  - 10-50分钟：代表题精讲+3道同型变式（学生独立做+讲思路）+ 1道梯度检测
  - 50-60分钟：展示"掌握1类题胜过刷10道"的效果，给家长看变式对比
- 备选策略：若家长坚持，建议"本节讲2道，剩余8道纳入后续规划"
- 风险提示：维护教师专业边界，试听课不等于免费答疑课

**示例3：学生严重抗拒时的急救流程（降级方案）**
- 场景判断：60分钟/学生严重抗拒（被逼来/情绪抵触）/家长旁听
- 触发条件：学生连续3次拒绝配合（不答题/不说话/要求下课）
- 降级时间轴：
  - 0-5分钟：暂停知识教学，转为非学术闲聊（兴趣点/学校生活/游戏运动）→ 产出：情绪缓和信号（如眼神接触/单字回应）
  - 5-15分钟：用兴趣点建立连接（如"你提到喜欢篮球，这道题用篮球场景举例你能试试吗"），选最简单的导入题 → 产出：最低限度配合（哪怕只写一步）
  - 15-55分钟：维持低难度，以"体验好感"为目标，允许大量提示，确保学生能完成至少1道题获得成就感 → 产出：建立"这个老师不凶"的认知
  - 55-60分钟：正向反馈（具体表扬尝试过程而非结果）+ 课后反馈中建议家长"先建立信任再谈学习" → 产出：下次课愿意再次见面的意愿
- 备选策略：若学生完全拒绝沟通，建议教师"本节转为纯聊天课"，避免对抗升级
- 风险提示：禁止强迫学生学习，禁止在家长面前批评学生"态度不好"；教师需明确告知家长"首课目标调整为建立关系"

## Common Issues
1. **时间失控**：教师陷入"多讲显示专业"误区，导致讲练比失衡。解法：严格按三段式切割时间，到点必须转入验证环节
2. **学生抗拒被动**：学生被逼来、不配合。解法：首课目标从"知识密度"调整为"建立配合意愿"，用兴趣点建立连接，允许降低内容难度（参见示例3）
3. **信息缺失备课**：仅知年级无其他信息。解法：备两套不同题材/难度方案（如数学备"计算专题"与"几何入门"），用前5分钟学情问答决定启用哪套
4. **家长旁听紧张**：学生因家长在场隐藏真实水平。解法：设置"独立练习时段"让家长暂时离开，或请家长坐在学生视线盲区，使用"草稿纸草稿"降低暴露焦虑
