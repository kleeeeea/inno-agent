# 学情诊断与转化沟通

## When to use
- 需要设计5-7分钟学情核验提问清单及基于诊断的选题决策规则
- 需要验证学生"本来会"还是"被教会"（梯度出题、变式检测、口述思路）
- 家长旁听导致学生紧张，需要保护性话术与独立练习安排
- 家长追问提分效果，需要合规回应与"规划承诺"话术
- 需要生成结构化课后反馈报告（问题定位+原因+计划+预期）
- 试听课后复盘诊断失败原因并制定改进动作

## When not to use
- 需要40/60分钟试听课的精确时间分配表或三段式教学流程设计 → 使用 trial-lesson-flow-design
- 需要被动型学生的课堂活动设计或主动型学生的高密度教学策略 → 使用 trial-lesson-flow-design
- 遇到学生严重情绪崩溃或心理障碍 → 建议转介专业心理咨询，不强行推进诊断
- 仅需一般性教学知识问答而无特定首课转化场景 → 使用普通问答

## Instructions
1. **诊断验证优先**：禁止基于猜测给出确定性学情判断，必须通过当堂梯度测试验证（中→难→基础）
2. **合规底线**：严禁提供"保提分""保及格"承诺，区分"给目标"与"给规划"的边界
3. **学生保护**：家长旁听时提供"独立练习时段"让学生脱离家长视线，避免羞辱性评价
4. **工具交付**：提供可直接复用的提问清单、反馈模板、话术脚本，而非泛泛建议
5. **闭环思维**：诊断必须指向可验证的下一步动作，避免信息收集后无后续跟进

## Workflow
1. **学情核验**：通过教材/进度/失分点/习惯四维度提问，3分钟内完成信息缺口填补
2. **梯度出题**：基于诊断结果设计"中等→稍难→基础回补"三级题目，定位真实薄弱点
3. **水平验证**：采用"同型变式+限时口述+逆向提问"组合拳，区分"本来会"与"被教会"
4. **反馈生成**：输出三段式报告（现状定位/归因分析/阶段规划），明确可承诺与不可承诺边界
5. **复盘诊断**：针对未转化案例，从"诊断精度/验证方法/家长感知"三维度定位失败点

## Output Format
每次输出必须包含以下字段：
- **场景判断**：当前属于学情核验/水平验证/家长沟通/转化复盘中的哪一类
- **诊断工具**：提问清单/出题梯度/验证动作的具体内容（可直接复用）
- **处理方案**：针对该场景的操作步骤与分界点（如何时降阶/何时停止追问）
- **可用话术**：教师可直接使用的沟通脚本（含家长旁听保护话术、合规承诺话术）
- **风险提示**：合规红线（保分承诺/羞辱评价）与教师权益边界（防超时讲课）
- **后续评估**：如何验证本次诊断/沟通的有效性（如下次课检验点/家长满意度指标）

## Examples
**例1：学生声称"我都会"的验证流程**
- 给中档题（学生做对）→ 给变式题（改变数字/情境）→ 要求限时口述思路
- 若变式错或卡壳：定位到"方法迁移弱"而非"知识点会"
- 反馈话术："孩子对基础题型熟练，但在变式迁移上需要训练，这是下一阶段重点"

**例2：家长追问"能提高多少分"的合规回应**
- 阻断："成绩受多重因素影响，我无法承诺具体提分幅度"
- 转向："但我可以承诺的是：4周内我们会完成XX知识模块的系统梳理，建立XX解题步骤的标准化流程"
- 交付：提供《阶段学习规划表》替代口头保证

**例3：家长旁听保护性干预**
- 现场安排："接下来5分钟请同学独立完成这道题，家长可以到休息区稍等，这样能看到孩子真实的解题状态"
- 若家长坚持陪同：教师转向"引导式提问"而非"直接纠正"，避免学生当众出丑

## Common Issues
1. **诊断信息过载**：试图在5分钟内问完所有问题，导致学生疲惫 → 改为"3个关键问题+课堂观察补充"
2. **验证题过简单**：全对但无诊断价值 → 必须包含至少一道变式题或综合题
3. **反馈过于笼统**：只说"基础薄弱"无具体定位 → 必须细化到"知识断层/方法缺失/习惯问题/心态障碍"
4. **承诺边界模糊**：被家长诱导给出"应该能提20分"的暗示 → 坚持"规划承诺"替代"结果承诺"
5. **复盘归因错误**：将未转化简单归因为"家长没钱"而非"诊断未展示专业性" → 使用"诊断-验证-交付"三维度复盘清单
