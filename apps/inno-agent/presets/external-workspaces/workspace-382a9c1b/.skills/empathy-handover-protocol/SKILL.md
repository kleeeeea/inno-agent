# 共情安抚与专业移交规范

## Instructions

- 安抚阶段只表达理解与陪伴，禁止追问原因、质问对错、贴标签（如"你怎么这么暴躁"）
- 区分两类对象：①情绪崩溃/冲突发起学生——轻声陪伴、轻拍后背；②被攻击学生——优先检查伤情，再给予情绪支持
- 移交时只陈述客观事实（时间、行为、现场状态），不评价品行、不猜测家庭背景、不使用心理诊断性语言
- 学科教师不做心理干预，严重情绪与肢体冲突必须移交班主任或心理教师，不拖延、不硬扛
- 全程保护学生隐私，事后不在办公室、走廊或其他班级传播事件细节

## Workflow

1. **识别安抚对象**：判断当前需要安抚的是冲突发起者、被攻击者，还是两者皆需
2. **生成安抚话术**：根据对象类型输出符合"只共情不问责"原则的具体话术与动作要领
3. **提示移交时机**：班主任/心理教师到场即刻移交，给出移交话术模板
4. **生成移交汇报**：按"时间 + 客观行为 + 现场状态 + 伤情"结构输出汇报内容，过滤主观评价
5. **确认禁止项**：逐一核查输出内容是否含追问、贴标签、猜测隐私等违规表述

## Output Format

**安抚话术输出**
- 动作要领（1-2条，如蹲下/轻拍后背）
- 推荐话术（引号标注，直接可用）
- 禁止示例（❌标注，对比说明）

**移交汇报输出**
> 时间：[具体时间]
> 行为：[客观描述，如"A推了B，B摔倒在地"]
> 现场：[器材状态、其他学生反应]
> 伤情：[是否受伤及初步情况]

## Examples

**示例1——安抚情绪崩溃学生**
- 动作：走近，蹲下与学生视线齐平，轻拍后背
- 推荐话术：
  > "老师在这里，不用怕，慢慢来，不着急。"
  > "我看到你现在很难受，老师陪着你。"
- ❌ 禁止："你冷静一点！""说，为什么要这样做？"

**示例2——安抚被攻击学生**
- 动作：先检查伤情，若有外伤立即启动校医流程，再蹲下陪伴
- 推荐话术：
  > "让我看看有没有受伤，你很勇敢。"
  > "我知道你很疼，也很委屈，老师在这里陪着你。"
- ❌ 禁止："谁先动手的？""你有没有惹他？"

**示例3——移交汇报**
> "今天上午第三节，大约10点05分，A和B在实验台旁发生推搡，A推了B一下，B摔倒，我已让两人分开，B说手腕有点疼，其他同学已回座位。"

## Common Issues

- **误区：安抚时顺带了解事情经过** → 安抚阶段严禁追问，了解事实是班主任的职责
- **误区：移交时补充主观判断**（如"这孩子平时情绪就不稳定"）→ 只说观察到的行为，删除所有评价性语言
- **误区：认为轻度冲突可以自行处置不用移交** → 只要涉及肢体冲突或严重情绪崩溃，必须请班主任到场，学科教师不独立处理
- **误区：事后回应学生追问时透露细节** → 统一回应："相关老师已在处理，大家专注自己的学习。"
