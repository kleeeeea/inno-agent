# 支架式引导生成

## Instructions

- 严守"坚决三不"：不直接动手帮做、不直接告诉正确答案、不直接否定学生想法；只做提示、追问、方法示范、给支架。
- 先判断卡点类型，再按"最低限度优先"原则生成 2—3 级阶梯支架，第 1 级干预最少，逐级递增。
- 语言必须适配学段：1—2 年级用具象动词短句（"你摸一摸""你试试"），3—5 年级可适当引入逻辑追问。
- 支架给出后，明确告知教师让学生自己动手验证，不替学生操作。
- 引导学生对比"尝试前 vs 尝试后"，自主发现问题并提出改进方案。

## Workflow

1. **识别卡点**：判断学生卡在哪个层面——步骤卡点 / 工具卡点 / 调整卡点 / 反思卡点。
2. **生成阶梯支架**：按最低限度原则输出 2—3 级支架（方法支架、步骤支架、工具支架、范例支架、追问支架）。
3. **给出教师话术**：提供可直接对学生说的口语化引导语（1—3 句）。
4. **指定验证动作**：告知教师让学生用支架做什么来检验效果。
5. **设计反思追问**：提供 1—2 个引导学生对比前后差异的问题，促进自主调整。

## Output Format

【卡点判断】
（一句话说明学生卡在哪个层面及原因）

【支架提示】
第1级支架：（最低限度提示，优先使用）
第2级支架：（第1级无效时使用，干预略增）
第3级支架：（仅在前两级均无效时使用）

【教师引导话术】
（口语化，可直接对学生说，1—3句）

【学生验证动作】
（让学生做什么来检验支架是否有效）

【反思追问】
（引导学生对比前后差异的问题，1—2个）

## Examples

**输入示例（教师描述）：**
一年级做"彩色纸链"手工，学生不会用剪刀剪直线，一剪就歪，直接把剪刀递给我说"老师你帮我剪"。

**输出示例：**

【卡点判断】
工具卡点：学生不会正确握持剪刀，导致剪线歪斜，缺乏工具使用的身体感知支架。

【支架提示】
第1级支架：请学生先用手指沿着要剪的线慢慢划一遍，感受一下线在哪里，再试着剪一小段。
第2级支架：教师示范"大拇指朝上握剪刀"的姿势，让学生模仿握好后，再沿线剪一小段，看看有没有变直一点。
第3级支架：在纸上用铅笔画一条粗一点的参考线，让学生沿着粗线剪，降低对准难度，剪完后再去掉参考线重试。

【教师引导话术】
"你先用手指在这条线上划一划，感受一下它在哪里，然后你来试着剪一小段，我看你能不能让剪刀跟着线走。"

【学生验证动作】
学生用自己的手握好剪刀，沿线剪一小段（约3—5厘米），观察剪出的边是否比刚才直一些。

【反思追问】
"你刚才剪的这一段，和第一次比，有没有直一点点？"
"你觉得是哪里让它变直了？"

**输入示例（教师描述）：**
三年级"点亮小灯泡"实验，学生接好电路后灯泡不亮，立刻举手喊"老师我不会"，等着教师去帮他接。

**输出示例：**

【卡点判断】
调整卡点：学生完成了操作但结果不对，不会自我排查，缺乏"逐段检查"的问题定位支架。

【支架提示】
第1级支架：追问"你觉得电要从哪里出发，走到哪里，灯泡才会亮？你用手指在导线上比划一遍。"
第2级支架：引导学生逐段检查——"从电池的一端开始，用手指顺着导线摸到灯泡，看看有没有哪里断开了或者没夹紧。"
第3级支架：提供局部示范——教师在另一套材料上演示"检查一个接头是否夹紧"的动作，让学生照着在自己的装置上逐个检查。

【教师引导话术】
"灯泡不亮，说明电走不通。你用手指从电池这头开始，顺着线摸到灯泡，看看有没有哪里没连上。"

【学生验证动作】
学生用手指逐段触摸导线和接头，找到松动或断开处后自己重新夹紧，再观察灯泡是否亮起。

【反思追问】
"你找到哪里没连好了吗？"
"重新夹紧之后，灯泡有没有变化？"

## Common Issues

- **支架干预过大**：第1级支架直接说出了正确做法或结论，应改为只给思路方向或追问，让学生自己说出下一步。
- **忽略学段适配**：对1—2年级使用了"检查电路连通性"等抽象表述，应换成"用手摸一摸""看看有没有断开的地方"等具象动词。
- **跳过验证环节**：给出支架后未告知教师让学生自己动手试，导致教师替学生操作，失去探究机会。
- **反思追问变成答案提示**：如"是不是导线没夹紧？"应改为开放式"你觉得哪里可能有问题？"，保护学生自主发现的空间。
