# 班级霸凌结构化干预专家评价准则

## 包信息
- **Package Name**: 班级霸凌结构化干预专家
- **Package Description**: 针对小学至初中阶段班级霸凌事件，为班主任提供从隐蔽行为识别、五步法结构化干预到长效跟踪的全流程专业支持，确保教育闭环与班级生态修复。

## 整体评价
在回应"班级霸凌现象的结构化干预方案"这一复杂议题时，回复展现了高度的专业洞察力和实践指导性。回复深入理解教师在处理班级欺凌时，面临的识别困境、干预流程、多方协作、关系修复、效果评估及日常预防等核心环节，提供了系统且层层递进的解决方案。语言严谨，富有同理心，充分考量教师操作需求和情感支持。

## 具体评价维度

| 维度 | 评价标准 |
|------|----------|
| **① 识别精准性** | 应提供精准、易参照的欺凌行为观察指南。关注学生情绪、学业变化，身体伤痕及群体互动模式，助教师从细微之处精准识别潜在欺凌行为。 |
| **② 干预方案逻辑性** | 应提供逻辑严密、步骤清晰的实施路线图。不仅列举零散方法，更指导教师从识别、介入、调查、评估、计划、实施、修复到跟踪，确保系统干预。 |
| **③ 协作机制有效性** | 应明确各方职责与协作路径，提供沟通协调策略。如案例研讨、信息共享、任务分配，确保教师高效联动，整合资源，形成反欺凌合力。 |
| **④ 调解策略实用性** | 应提供实用且技巧性对话指南。涵盖前期准备、环境营造、情绪引导、同理心培养、责任厘清、行为协商，助教师有效控场，促积极对话与关系修复。 |
| **⑤ 效果评估多维性** | 应指导教师进行客观全面评估。不仅关注事件停止，更提供情绪、社交、学业、行为转变、班级氛围等多维度指标，确保评估科学性。 |
| **⑥ 预防策略前瞻性** | 应提供前瞻性、创新性预防策略。包含品格教育、同理心游戏、冲突训练、反欺凌班会、班级公约，助教师融入文化，构建友爱安全环境。 |
