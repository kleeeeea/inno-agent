# 结构化干预与家校协调

## When to use
- 已确认发生霸凌事件，需要启动正式干预流程
- 需要召集家长沟通并达成赔偿/道歉共识
- 需要设计反霸凌主题班会及划定班级行为红线
- 需要协调心理老师、科任教师等多方协作分工
- 需要组织涉事学生调解对话促进关系修复
- 需要对干预效果进行长效跟踪与评估

## When not to use
- **转识别评估**：当事件性质尚不明确，仅处于疑似观察阶段（转交 `bullying-identification-assessment`）
- **转危机干预**：涉及严重身体伤害、自杀自残风险、刑事犯罪、校外黑恶势力介入（立即上报学校管理层，启动校园危机预案）
- **转安全处置**：家长在校内发生激烈冲突或威胁教师安全（优先保障人身安全，暂停常规干预流程）

## Instructions
- 严格遵循"阻断-沟通-调解-班会-跟踪"五步法，确保教育闭环
- 家长沟通遵循"先受害者→后施暴者→再双方协商"的顺序，避免直接对峙
- 班会环节必须匿名化处理案例，聚焦行为教育而非个人审判
- 建立多方协作机制，明确班主任主导、心理老师支持、科任教师观察的职责边界

## Workflow
1. **阻断与定性**：立即物理隔离冲突双方，清晰界定"欺凌性质"，打破施暴者认知盲区
2. **家校沟通**：分别约谈双方家长掌握心理预期，作为中立裁判推动达成赔偿与和解一致
3. **责任闭环**：施暴者向受害者正式道歉完成教育闭环，同步筹备班级公共议题素材
4. **群体警示**：召开主题班会展示真实案例与后果，明确行为红线，教授非暴力冲突解决技能
5. **长效跟踪**：持续2-4周观察涉事双方情绪与行为，建立"班级观察员"机制预防复发

## Output Format
- **场景判断**：当前所处五步法阶段及关键风险点
- **处理方案**：对应步骤的具体操作清单、时序安排与注意事项
- **可用话术**：与家长、学生沟通的关键话语模板（界定/安抚/施压/引导）
- **协作分工**：班主任、心理老师、科任教师的具体任务分配与信息共享机制
- **风险提示**：家长对抗、二次伤害、关系反弹等风险及应急应对策略
- **后续评估**：跟踪观察的五大维度指标（情绪状态、社交互动、学业表现、行为转变、班级氛围）

## Examples
- **家长沟通（施暴方）**："您孩子的行为已构成校园欺凌，需承担相应责任。我们希望与您共同推动孩子认识错误，真诚道歉并赔偿。您对此有什么想法？"
- **班会引导（群体警示）**："班级里不允许任何故意伤害他人的行为，无论以玩笑还是冲突为借口。这不仅是校规，更是对生命的基本尊重。"
- **学生调解（同理心激发）**："如果你每天被人这样对待，持续一个月，你会有什么感受？你希望对方怎么做？现在轮到你选择成为怎样的人。"

## Common Issues
- **施暴者家长否认**：出示具体行为证据链，引用《未成年人保护法》及校规，强调法律后果与学校立场
- **受害者家长过度维权**：先共情安抚情绪，再引导关注孩子心理修复需求而非单纯惩罚对方
- **班会匿名失效**：提前审查PPT素材去标识化，禁止同学猜测或暗示具体当事人身份
- **干预后隐蔽报复**：建立"班级观察员"（非班干部）机制，每日课后、午休等高发时段重点巡查
- **教师情绪耗竭**：明确班主任负责协调而非独自承担所有工作，及时转介心理老师处理创伤辅导
