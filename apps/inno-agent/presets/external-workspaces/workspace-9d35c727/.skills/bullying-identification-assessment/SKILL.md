# 霸凌识别与性质界定

## When to use
- 发现学生情绪/学业突然变化、身体出现不明伤痕或群体互动模式异常，需判断是否为霸凌
- 难以区分"同学玩笑"与"言语/心理霸凌"的边界，需要性质界定标准
- 需要建立系统的行为观察清单（Checklist）和风险评估维度
- 处于事件初期发现阶段，尚未确认是否为霸凌，需专业指导进行初步定性

## When not to use
- **已确认霸凌事件**：若事件性质已明确为霸凌且需启动家长沟通、班会教育等干预措施，应转交`structured-intervention-coordination`技能执行五步法
- **高危紧急情况**：发现严重身体伤害（骨折、器官损伤、需就医的眼部/头部伤害）、自杀自残风险、刑事犯罪特征（管制刀具、性侵害）或教师无法控制的人身安全威胁时，立即停止识别流程，优先启动校园紧急预案（上报学校管理层、送医、报警）
- **非适用学段**：本技能仅适用于小学至初中阶段，不适用于高中及以上

## Instructions
- **多维度观察**：结合情绪指标（突然沉默、焦虑、逃避）、学业指标（成绩骤降、缺课）、身体指标（不明伤痕、衣物破损）、社交指标（刻意孤立、群体排斥）进行交叉验证
- **隐蔽行为识别**：重点关注非教师在场时段（课间、午休、上下学途中）及线上社交平台（班级群、私信）的互动动态
- **性质界定标准**：区分"玩笑"与"霸凌"的核心在于是否同时具备**故意伤害性**、**力量不对等性**（人数/地位/体力优势）、**重复发生性**及**受害者痛苦感**
- **风险分级**：根据行为频率、伤害程度、权力差异，将事件划分为"一般冲突""轻度霸凌""严重霸凌"三级，严重霸凌立即建议转介干预

## Workflow
1. **线索收集**：通过日常观察、科任教师反馈、匿名问卷、班级观察员等渠道收集异常行为信号
2. **初步评估**：对照霸凌核心特征（故意性、不对等性、重复性）评估疑似行为
3. **性质界定**：明确事件属于"同学冲突""玩笑过度"还是"霸凌行为"，判定严重程度
4. **风险筛查**：识别高危信号（自伤倾向、严重外伤、刑事特征），必要时立即转介安全处理
5. **方案建议**：根据评估结果建议"继续观察""启动干预"或"紧急上报"，生成观察记录

## Output Format
```
### 场景判断
- 当前阶段：疑似识别阶段 / 建议转交干预阶段
- 风险等级：一般冲突 / 轻度霸凌 / 严重霸凌 / 高危紧急
- 行为类型：[肢体/言语/关系/网络]霸凌 或 非霸凌冲突

### 观察证据
- 情绪行为指标：[具体异常表现]
- 学业社交指标：[成绩变化、同伴关系变化]
- 身体痕迹指标：[伤痕、财物损坏]
- 时空特征指标：[发生时段、地点、频率]

### 界定结论
- 性质判定：[霸凌/非霸凌/需进一步观察]
- 判定依据：[符合/不符合 故意性、不对等性、重复性标准]
- 严重程度：[轻度/中度/重度]

### 风险提示
- 高危信号：[列出风险点或"无"]
- 立即行动：[紧急处理措施或"继续监测"]

### 下一步建议
- 即时措施：[保护受害者、隔离双方等]
- 观察计划：[周期、重点关注维度]
- 转介建议：[是否启动五步法干预或转介心理老师]
```

## Examples
**示例1：言语霸凌识别**
用户：班上有同学总是给某个孩子起外号，对方很不高兴，但起外号的同学说只是开玩笑，这算霸凌吗？

输出：
```
### 场景判断
- 当前阶段：疑似识别阶段
- 风险等级：轻度霸凌
- 行为类型：言语霸凌（待确认）

### 观察证据
- 情绪行为指标：被起外号学生表现出不高兴（痛苦感确认），回避相关同学
- 学业社交指标：需观察是否回避上学
- 身体痕迹指标：无明显伤痕
- 时空特征指标：需确认发生频率（偶发还是持续）、是否有旁观者起哄

### 界定结论
- 性质判定：疑似言语霸凌（需验证重复性）
- 判定依据：具备故意伤害性（对方已表示不悦仍继续）、受害者痛苦感明确，需验证力量不对等性及重复性
- 严重程度：轻度（若无肢体威胁）

### 风险提示
- 高危信号：无自伤或严重外伤报告
- 立即行动：暂时观察并记录频率，无需紧急上报

### 下一步建议
- 即时措施：私下询问受害学生感受，明确告知施扰学生停止该行为
- 观察计划：连续一周记录发生频次及旁观者反应
- 转介建议：若一周内重复发生超过2次或出现升级，立即转交`structured-intervention-coordination`启动五步法干预
```

**示例2：隐蔽关系霸凌识别**
用户：最近发现小明总是一个人吃饭，以前他都有朋友一起的，问他就说没事，但我注意到有几个同学在旁边笑，这是不是被孤立了？

输出：
```
### 场景判断
- 当前阶段：疑似识别阶段
- 风险等级：轻度至中度霸凌（待确认）
- 行为类型：关系霸凌（社交孤立）

### 观察证据
- 情绪行为指标：社交回避（独自用餐）、口头否认（可能因恐惧隐瞒）
- 学业社交指标：社交圈突然缩小，从群体活动转为孤立状态
- 身体痕迹指标：暂无
- 时空特征指标：午餐时段、课间，需观察是否有组织性排斥

### 界定结论
- 性质判定：疑似关系霸凌
- 判定依据：具备力量不对等性（群体对个体）、受害者痛苦感（行为突变），需验证故意伤害性（是否故意排斥）及重复性
- 严重程度：中度（若确认为有组织孤立）

### 风险提示
- 高危信号：需警惕是否伴随网络霸凌（班级群排斥）、抑郁情绪
- 立即行动：启动密切观察，无需紧急医疗介入

### 下一步建议
- 即时措施：安排班级观察员关注午餐/课间动态；通过周记等非直接方式了解心理状态；不经意安排合作学习小组重建社交连接
- 观察计划：两周内每日记录社交互动频率
- 转介建议：若确认存在有组织排斥或持续一周无改善，转交`structured-intervention-coordination`启动家长沟通与班会干预
```

## Common Issues
- **过度敏感**：将偶发、力量对等的正常同学冲突误判为霸凌，导致不必要的标签化
- **证据不足**：仅凭单一事件就定性，忽略"重复性"和"持续性"是霸凌的核心标准
- **忽视隐蔽性**：仅关注肢体冲突，忽视关系霸凌（冷落、排斥）和网络霸凌（班级群言语攻击）
- **受害者隐瞒**：因恐惧或羞耻，受害者否认受害，需通过学业变化、身体痕迹等间接观察交叉验证
- **延误转介**：在识别阶段停留过久，错过干预黄金期，一旦确认霸凌性质（具备核心三特征）应立即转交干预流程
