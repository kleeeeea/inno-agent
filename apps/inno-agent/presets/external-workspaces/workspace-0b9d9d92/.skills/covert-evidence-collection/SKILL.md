# 隐秘调查与证据固定

## When to use

- 接到学生私下举报班干部"针对自己"或"执法不公"，但尚无客观证据证实
- 发现考勤记录、纪律评分等数据异常（如特定同学被记录频率异常偏高），怀疑存在选择性执法
- 观察到班级氛围微妙变化（如同学对班干部产生抵触情绪），需要核实是否存在权力滥用
- 班干部管理行为出现矛盾点（如前后标准不一），需要确认是否夹杂私人恩怨

## When not to use

- **已确认违规需处理后果** → 应使用 **restorative-intervention**（修复性干预与生态重塑）
- **涉及身体暴力、校园霸凌或学生有自伤/抑郁倾向** → 立即启动校园安全预案并转介心理老师，不再进行隐蔽调查
- **家长已威胁法律诉讼或已公开冲突** → 需先经学校管理层/法务介入，避免单独调查
- **纯制度设计需求**（如制定选拔标准、职责清单）→ 通过普通问答提供规范化建议

## Instructions

1. **隐蔽优先原则**：调查过程必须避免"打草惊蛇"，防止班干部销毁证据或产生防御心理，确保观察到的是自然状态下的行为模式。
2. **程序合法合规**：监控调取需通过学校正规流程，学生谈话需在私密空间进行，严禁诱导式提问或侵犯隐私的跟踪行为。
3. **证据链闭环**：单一证据不足以定论，必须通过"客观记录（监控/书面记录）+ 当事人陈述 + 旁证"形成交叉验证。
4. **教育性立场**：调查目的是为了教育引导而非"抓把柄"，过程中需保持对学生的基本信任与保护意识。

## Workflow

1. **双轨观察期**：首日亲自到场正常巡视，次日故意制造"管理真空"（短暂缺席或隐蔽观察），对比班干部在不同情境下的执法差异。
2. **客观证据固定**：通过正规渠道调取关键时段监控（如早读前、课间），核实考勤记录与实际行为的吻合度；保存书面记录的拍照或复印件。
3. **侧面核实取证**：以"优化管理"或"核对数据"为由，私下约谈被记录的学生，了解其被记录时的具体情境及与班干部的关系背景。
4. **行为模式分析**：整理时间轴，标记是否存在"卡点记名"、"选择性记录"（对同类行为不同处理）等滥用模式，形成初步结论报告。

## Output Format

```markdown
### 场景判断
- 涉及主体：[班干部角色] + [疑似受害者]
- 怀疑依据：[数据异常/学生举报/氛围观察]
- 风险等级：[低/中/高，高需转介]

### 取证方案
- 观察策略：[具体观察方式与时间安排]
- 证据来源：[监控时段/书面记录/访谈对象]
- 验证逻辑：[如何确认或排除怀疑]

### 可用话术
- **约谈被管理者**："[中性理由]，老师想核实一下你当天的具体到校时间及原因，请你客观陈述，老师会结合监控进行综合研判..."
- **调取监控申请**："[向学校保卫科/德育处]因班级管理优化需要，申请调取X月X日X时段教室门口监控..."

### 风险提示
- 可能遇到的阻力：[班干部察觉/家长质疑/证据不足]
- 终止调查条件：[确认无事/确认违规/安全风险出现]

### 后续评估
- 证据充分性：[是否足以支撑干预]
- 建议移交方向：[修复性干预/继续观察/制度调整]
```

## Examples

**案例：学习委员针对性考勤调查**

班主任接到学生A家长反映，孩子本周被学习委员记录迟到3次，但其他同样卡点到校的同学未被记录。班主任首先进行了两天的对比观察：第一天亲自站在教室门口，发现学习委员对A格外关注；第二天提前躲藏在楼梯拐角观察，发现学习委员确实只对A进行"卡点拦截"。随后调取本周监控，确认A有2次实际并未迟到（在铃声前已进入），但学习委员在事后补录了名单。最后以"核对考勤系统数据"为由私下询问A："老师最近在盘点考勤数据，注意到你周二被记录迟到，想核实一下当天的具体到校时间..."结合A的陈述，确认了学习委员存在"选择性执法"行为。证据固定后，转入修复性干预流程。

## Common Issues

1. **过早暴露调查意图**：教师在观察期表现得过于关注班干部，或询问问题时直接质问"你是不是针对某某"，导致对方警觉并改变行为模式，证据链断裂。
2. **单一证据定案**：仅凭一次观察或单方面陈述就认定违规，忽视可能存在的信息偏差（如被举报学生确实存在其他违纪行为）。
3. **忽视隐私边界**：在调查过程中向无关学生透露怀疑内容，或在公开场合调取监控引起围观，造成不必要的舆论压力。
4. **调查周期过长**：因犹豫不决导致观察期拖得过久，受害者持续遭受不公正对待，或班干部违规行为变本加厉。
