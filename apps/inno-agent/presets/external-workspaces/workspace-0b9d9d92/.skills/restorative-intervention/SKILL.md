# 修复性干预与生态重塑

## When to use
- 已掌握确凿证据（监控、记录等），确认班干部存在权力滥用行为，需从"调查阶段"转入"处理阶段"
- 需要设计"温和而坚定"的批评策略，既要严肃指出"公器私用"危害，又避免公开羞辱导致寒蝉效应
- 需安排班干部私下向受害者道歉，修复同伴关系并保全双方尊严，防止次生对立
- 班级出现信任危机后，需通过不点名的主题班会重塑规则意识，将个案转化为集体教育契机
- 需制定"观察期"评估标准，持续追踪行为矫正效果，必要时调整职务或完善监督机制
- **适用范围**：单一事件、偶发违规、初次违规、个体行为（非系统性问题）

## When not to use
- 仅是怀疑阶段，尚无确凿证据（如仅凭举报、直觉）→ 应使用 **covert-evidence-collection** 进行隐秘调查取证
- 涉及身体暴力、校园霸凌或严重心理危机（如自杀威胁、重度抑郁发作）→ 立即启动校园安全预案并转介心理危机干预
- 家长已知晓并威胁法律诉讼，或要求公开处理 → 需先经学校管理层/法务部门介入，暂停教育性干预
- 涉及多名干部集体滥用权力形成派系、群体性事件 → 需升级为学校层面冲突调解，超出单一班级干预范畴
- **系统性违规/重度违规**（必须升级处理）：
  - 班干部**屡教不改**（观察期内再次违规或同一行为多次发生）
  - **多人串通**形成权力垄断、利益输送网络（如多名干部共同包庇、分赃）
  - **系统性舞弊**（长期、有组织的考勤造假、评分操纵）
  - **权力垄断**（班干部形成"小圈子"长期把持关键岗位并排斥异己）
  - → **以上情况需立即上报学校德育处/管理层**，启动校级调查与组织架构调整，本技能仅作为配合支持，不可单独依赖
- 纯粹询问班干部选拔、日常培训等预防性制度建设（未发生违规）→ 使用 **preventive-system-design** 技能

## Instructions
1. **教育优先于惩戒**：体现"教育性干预"理念，目标是纠正权力认知偏差而非惩罚个人，严禁公开羞辱或行政性惩戒。
2. **程序正义与情感保护并重**：出示确凿证据确保程序正当，同时明确告知"不公开是为了保护你的尊严"，维护青少年自尊。
3. **闭环修复**：必须包含"认错-道歉-改正-评估"完整闭环，确保班级信任基础得到实质性修复。
4. **借力打力**：通过设计不点名的主题班会（如"守时与规则"），将个别事件转化为全班教育契机，既警示涉事者又教育集体。
5. **重度违规识别与升级**：一旦发现违规具有"系统性、组织性、重复性"特征（如多人串通、长期舞弊、屡教不改），必须立即停止单一干预，建议班主任上报学校管理层。

## Workflow
1. **违规程度判定**：首先评估是"个体偶发"还是"系统性违规"。若涉及多人串通、长期舞弊、屡教不改，立即建议升级至校级处理。
2. **闭门敲打与证据展示**：私下约谈班干部，出示确凿证据，严厉指出"公器私用"危害，使用"如果老师利用权力针对你"等换位思考话术打破认知偏差。
3. **权力认知重构**：明确告知"权力是服务责任而非私愤武器"，强调授权基于信任，当前行为正在透支信任资本，必须立即停止。
4. **私下道歉与关系修复**：要求班干部在保护隐私前提下向受害者真诚道歉，教师提供话术指导但不出面代劳，确保双方在无压力环境下和解。
5. **主题班会与生态重塑**：设计不点名、不暗示的专题教育（如时间管理、规则意识），借个别案例警示集体，重建对程序正义的信心。
6. **观察评估与制度优化**：设立1-2个月观察期，通过考勤数据公正性、同学反馈等指标评估改正效果，必要时调整职务。
7. **再犯升级处理**：若观察期内再次违规，明确告知"教育性干预已无法奏效"，建议启动职务调整或上报学校管理层。

## Output Format
- **场景判断**：确认当前处于"已确认违规/需干预修复"阶段，明确涉事主体、权力滥用类型、班级信任受损程度，**并判定是否属于需升级的系统性违规**
- **处理方案**：分步骤操作指南（约谈→道歉→班会→观察），明确教师言行边界、保密原则与时间节点；**若判定为系统性违规，提供升级建议而非详细干预方案**
- **可用话术**：提供闭门敲打话术、换位思考引导语、私下道歉要求话术、主题班会开场白等关键场景表达
- **风险提示**：列出干预过程中可能出现的突发情况（情绪崩溃、否认狡辩、家长介入、受害者拒绝和解、**再犯迹象**）及应对预案
- **后续评估**：设定观察期评估标准（考勤公正性、同伴反馈、自我反思日志），明确行为矫正的验收指标与再犯升级处理措施

## Examples
- **Example 1（个体偶发）**：学习委员卡点记名报复同学 → 闭门敲打："权力是服务班级的责任，绝不是你用来宣泄私愤的武器" → 安排私下道歉并保留职务 → 召开"守时规划"主题班会（全程不点名），从源头规避迟到争议
- **Example 2（个体偶发）**：班长考勤舞弊被查实 → 证据展示后强调"老师不公开是保护你的羽毛，但做错了必须承担责任" → 要求撰写权力使用反思日志 → 设立"双人复核"观察期 → 重塑班级监督机制
- **Example 3（需升级案例）**：发现学习委员、班长、纪律委员三人长期串通，通过考勤造假为彼此及"关系好"的同学包庇，形成利益输送网络 → **判定为系统性违规** → 建议："此情况已超出单一班级教育干预范畴，涉及组织性舞弊，建议立即上报学校德育处，启动校级调查，必要时重新选举班委架构"

## Common Issues
- **学生否认狡辩**：提前准备多重证据链（监控+数据比对），保持"温和而坚定"态度，不陷入辩论而是明确陈述事实与期望，必要时使用"我已掌握确凿证据，现在需要你的诚实"打破防御。
- **家长质疑包庇**：准备干预记录（谈话时间、内容摘要、改正计划），向家长说明"教育性干预"理念与《未成年人学校保护规定》中关于保护学生尊严的法律依据。
- **受害者拒绝和解**：尊重受害者意愿，不强求立即原谅，转为教师持续关注与阶段性调解，必要时引入第三方调解，优先确保受害者安全感。
- **班级舆论发酵**：若事件已泄露，立即召开简短说明会强调"正在妥善处理中"以遏制谣言，但绝不公开细节或点名批评，防止二次伤害。
- **再犯与升级**：若班干部在观察期内再次违规，或使用"我已经道歉了为什么还要被监督"等话术对抗，表明教育干预失效，需立即启动职务调整或上报管理层，避免陷入"无限宽容"的陷阱。
- **群体性违规误判**：若初期误判为个体行为，后续发现涉及多人串通，应立即暂停当前干预，重新评估并建议升级处理。
