# 班级微权力治理顾问评估量表

## 方案概述
**包名**：班级微权力治理顾问  
**适用场景**：当班主任面临班干部权力滥用、公报私仇或班级管理不公问题时，提供基于授权与控制理论的完整治理方案，包括隐秘调查取证、教育性干预和班级生态修复策略。

---

## 维度一：选拔机制公平性（15%）
**对应技能模块**：`preventive-system-design`（制度性预防）

| 等级 | 评分标准 |
|------|----------|
| **优秀**（90-100） | 建立"自荐-推荐-竞选"多元选拔渠道，全过程公开透明；制定明确的品德能力双维评价标准；引入学生代表参与评审；建立选拔申诉机制；从源头构建权力监督意识，确保"程序正义"在起点实现 |
| **良好**（75-89） | 选拔程序基本规范，包含民主评议环节；评价标准较清晰，兼顾能力与人品；学生有一定参与度；但缺乏系统的申诉或复核机制 |
| **合格**（60-74） | 有基本选拔程序，但透明度不足或标准单一（如只看成绩）；学生参与有限，主要由教师主导；存在潜在的任人唯亲风险 |
| **不合格**（<60） | 教师指定任命或暗箱操作；缺乏明确选拔标准；完全排斥学生参与；形成"权力垄断"或"小圈子"文化，为后续滥权埋下隐患 |

## 维度二：职责权限明晰性（15%）
**对应技能模块**：`preventive-system-design`（制度性预防）

| 等级 | 评分标准 |
|------|----------|
| **优秀**（90-100） | 制定详细的《班干部权力清单》与《行为负面清单》；明确每项权力的边界、程序与禁止行为；建立"授权-行权-控权"闭环；提供可操作的履职指南（如考勤记录的"双人复核"制度）；定期根据实际修订完善 |
| **良好**（75-89） | 职责描述较清晰，有基本的行为规范；权力边界相对明确；但缺乏细化的操作流程或禁止性规定；对"自由裁量权"约束不足 |
| **合格**（60-74） | 有简单分工但权限模糊（如"负责纪律"但无具体说明）；缺乏明确的行为边界；未区分"管理权"与"处罚权"，易导致权力膨胀 |
| **不合格**（<60） | 职责不清，权限无限扩大；班干部可随意解释规则；缺乏任何规范约束；存在"以权谋私"的制度漏洞（如单人掌握关键资源分配权） |

## 维度三：权力教育导向性（20%）
**对应技能模块**：`preventive-system-design`（制度性预防）+ `restorative-intervention`（干预中的教育融入）

| 等级 | 评分标准 |
|------|----------|
| **优秀**（90-100） | 构建系统的权力观教育课程（如"权力与责任"主题系列）；深度融入"服务意识、责任担当、廉洁自律"教育；建立班干部定期反思机制（如"权力使用日志"）；体现"教育性干预"理念，将权力教育视为德育核心而非附加任务；干预过程中注重认知重构而非单纯惩戒 |
| **良好**（75-89） | 有权力教育内容，注重服务意识培养；定期开展相关教育活动（如班会谈权力观）；但缺乏系统性课程设计或深度反思环节；干预时能指出问题但教育深度不足 |
| **合格**（60-74） | 偶尔提及权力责任，但缺乏系统性教育；重"管人技巧"轻"服务精神"；教育内容碎片化，未形成价值引导；干预时仅关注行为纠正 |
| **不合格**（<60） | 只强调权力使用和管理权威，忽视责任教育；或完全缺乏相关引导；默许"官本位"思想；导致班干部产生"特权意识"；干预时采用羞辱或惩罚导向 |

## 维度四：监督反馈有效性（20%）
**对应技能模块**：`covert-evidence-collection`（隐秘调查）+ `preventive-system-design`（监督机制建设）

| 等级 | 评分标准 |
|------|----------|
| **优秀**（90-100） | 建立多元监督体系（教师监督、同伴互监、自我监督）；设置匿名意见箱/数字化反馈平台等畅通渠道；建立保护举报者机制（防止报复）；具备"隐秘调查取证"能力（如交叉验证考勤记录、侧面核实）；定期（如每月）开展权力运行评估；确保"权力运行在阳光下" |
| **良好**（75-89） | 有基本监督机制（如班主任抽查、意见箱）；设有反馈渠道并定期收集意见；但缺乏匿名保护机制或系统的调查验证手段 |
| **合格**（60-74） | 监督渠道单一（仅班主任监督）；反馈机制不畅（如意见箱形同虚设）；缺乏定期评估；学生因担心报复而不敢举报；缺乏调查技巧，仅凭主观判断 |
| **不合格**（<60） | 无监督机制或压制反馈意见；班干部排斥监督；存在打击报复举报者行为；权力运行完全封闭，形成"黑箱操作"；教师缺乏证据意识，仅凭感觉处理 |

---

## 维度五：干预惩戒公正性（20%）
**对应技能模块**：`restorative-intervention`（修复性干预）

| 等级 | 评分标准 |
|------|----------|
| **优秀**（90-100） | 建立分级干预机制（警示谈话→行为纠正→职务调整）；严格遵循"教育性干预"原则，避免公开羞辱；体现"修复性司法"理念（关注关系修复而非单纯惩罚）；程序公正透明（告知-申辩-决定-申诉）；充分保护当事人隐私与尊严；能准确识别系统性违规并及时升级处理；制定风险预案（应对情绪崩溃、家长介入等） |
| **良好**（75-89） | 有干预措施，注重教育引导；处理相对公正，能保护学生自尊；但程序规范性不足或修复性措施不够系统；偶有"敲打"过度风险；对升级处理判断不够敏锐 |
| **合格**（60-74） | 有简单惩戒但可能过重（公开批评）或过轻（简单警告）；程序不够规范；缺乏关系修复环节；可能造成一定心理伤害但未达危机程度；误判系统性违规为个体问题 |
| **不合格**（<60） | 公开羞辱（如当众撤职、讽刺挖苦）或完全放任不管；缺乏程序正义；造成"寒蝉效应"或二次伤害；激化师生/生生矛盾；违反《未成年人学校保护规定》；面对系统性违规仍坚持单一干预导致事态恶化 |

## 维度六：生态恢复与制度优化（10%）
**对应技能模块**：`restorative-intervention`（生态重塑）+ `preventive-system-design`（制度补漏）

| 等级 | 评分标准 |
|------|----------|
| **优秀**（90-100） | 危机后能有效修复班级信任关系（私下道歉、关系重建）；设计针对性主题班会重塑集体价值观；从个案中提炼制度漏洞并完善预防机制（如补上监督盲区）；建立"观察期"评估确保改正效果；形成可复用的危机处理经验 |
| **良好**（75-89） | 能处理个案并修复关系；开展相关教育活动；但缺乏系统的制度优化；观察期设置不够规范 |
| **合格**（60-74） | 仅处理当下事件，未关注关系修复；未从制度层面反思；班级氛围恢复依赖时间自然冲淡 |
| **不合格**（<60） | 事件处理后留下长期对立；班级信任体系持续恶化；同类事件反复发生而未修补制度；出现"寒蝉效应"或"报复循环" |

## 等级评定标准
- **卓越级**（90-100分）：形成可推广的班级微权力治理模式，兼具教育性与规范性，预防-调查-干预-修复全流程闭环
- **优良级**（75-89分）：基本建立完整治理体系，个别环节有待优化，能处理常规问题但应对复杂危机能力有限
- **发展级**（60-74分）：具备基础治理框架，但系统性、操作性不足，某些技能模块缺失或薄弱
- **待改进**（<60分）：存在严重治理缺陷，亟需重构班级权力运行机制，可能存在明显的能力盲区或原则性错误

## 技能短板诊断指南
根据各维度得分，可定位需要强化的技能模块：
- **选拔/职责/教育维度高分，监督维度低分**：需强化 `covert-evidence-collection` 技能（调查取证能力薄弱）
- **监督维度高分，干预维度低分**：需强化 `restorative-intervention` 技能（处理冲突和修复关系能力不足）
- **干预维度高分，选拔/职责维度低分**：需强化 `preventive-system-design` 技能（制度建设能力不足）
- **生态恢复维度低分**：需加强干预后的跟踪评估和制度建设联动能力
