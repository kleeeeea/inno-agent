# 家校共识沟通与风险控制

## When to use

- 家长坚决反对减少学科学习时间，需要结构化沟通化解"落后焦虑"与"失控恐惧"
- 需组织家校生三方对话，将争议焦点从"上不上学"转为"如何设计可逆路径"
- 制定并签署共识备忘录（含目标、时间盒、里程碑、底线、复学退路）
- 设计学业补学方案、成绩缺口解释策略及升学申请叙事（推荐信要点）
- 建立里程碑复盘机制（4–8周周期）与动态转轨预案

## When not to use

- 若仅需评估学生兴趣的**持续性、深度与自驱性**（三维评估），应使用 `interest-evaluation-pathway` 技能
- 若涉及**学籍办理具体行政流程**（如休学表格填写、审批时限），应提醒咨询当地教育行政部门
- 若学生存在**临床诊断的心理健康障碍**（抑郁、焦虑等），必须建议先寻求专业医疗评估与干预，暂停本流程
- 若涉及**违法违规创业指导、高风险商业活动**或需法律/财务/心理咨询，应明确拒绝并建议寻求专业帮助

## Instructions

- **价值重构优先**：使用"时间窗口""机会成本"框架，将家长焦虑从"升学抢跑"转向"职业抢跑"长期风险，避免空泛鼓励
- **风险前置**：每个方案必须同时包含**底线设定**（最低学业维持标准）、**可逆性预案**（回归路径）、**最坏情况提示**
- **证据锁定**：所有承诺必须基于可验证证据（作品集、专利、用户反馈），拒绝"一定成功"等虚假保证
- **合规底线**：严格遵守学籍管理规定，不承诺具体升学结果或创业成功率

## Workflow

1. **冲突诊断**：识别家长焦虑类型（落后恐惧/失控恐惧/社会评价恐惧），确认学生身心健康基线是否适合推进
2. **结构化对话**：组织三方会议，使用话术卡片将情绪对立转为方案设计（议题：路径选择A/B、时间盒、里程碑）
3. **共识备忘录**：签署非法律文件，明确目标、时间盒（半年/一年）、复盘节点（4–8周）、底线（学籍/健康/学科能力）、复学退路
4. **风险控制**：设定学籍合规边界、身心健康监测机制、学业关键能力维持方案（语文/数学底线）
5. **升学衔接**：构建项目履历（问题-解决-验证-影响结构）、推荐信要点（持续性/自驱性/技术贡献/社会价值）、成绩缺口解释方案
6. **循环复盘**：按里程碑评估进展，动态调整路径（A↔B转换）或触发复学预案

## Output Format

- **场景判断**：家长焦虑类型、冲突焦点、当前信任水平、身心健康基线
- **处理方案**：
  - 三方对话议程（含时间分配与话术锚点）
  - 共识备忘录模板（目标、时间盒、里程碑、底线、退路、签署方）
  - 学业补学计划框架（关键学科维持标准、补学资源、验收方式）
- **可用话术**：长期视角重构话术（区分升学/职业抢跑）、风险具体化回应话术、里程碑复盘话术
- **风险提示**：学籍合规风险、学业断裂风险、身心健康红线、最坏情况预案、转介触发条件
- **后续评估**：里程碑验收标准（产出物/持续性/自驱性）、转轨触发条件、升学材料完善清单（项目档案/推荐信要点）

## Examples

**家长**："休学一年就永远落后了，必须现在停止这些没用的项目！"

**回应结构**：
1. **共情与重构**："您担心的是时间窗口的不可逆性，这非常关键。但请区分两种抢跑：升学抢跑（早一年晚一年差别有限）与职业抢跑（错过行业窗口可能落后很多年）。"
2. **风险具体化**："我们不是放弃学业，而是设计一个有时间盒（一年）和 milestones（每两月验收）的可逆实验。如果第4个月里程碑未达成，立即触发预案A回归学业，并启动补学方案。"
3. **证据锚定**："基于学生过去12个月的项目证据（专利/用户反馈），这是低概率事件，但我们需要把控制机制写下来，这是备忘录的核心。"
4. **底线明确**："学籍保留、语文数学每周保底课时、每月心理评估，这三条红线一旦触碰，自动触发回归程序。"

## Common Issues

- **家长将"弹性学制"等同于"放弃学业"**：需用"时间盒+可逆性"框架明确这是可控实验而非永久退出
- **学校担心承担风险责任**：强调备忘录为"共识记录"非法律承诺，明确各方权责（家长监护/学校合规/学生执行）
- **缺少可验证证据导致承诺空洞**：拒绝仅凭"热情"推进，必须提供≥6–12个月的持续性记录与深度产出物
- **忽视身心健康红线直接推进路径设计**：若学生存在睡眠障碍、情绪持续低落或临床诊断，必须先暂停本流程并转介医疗评估
- **复学退路模糊导致双输**：未制定清晰的补学计划与成绩缺口解释方案，导致项目失败且学业无法衔接
