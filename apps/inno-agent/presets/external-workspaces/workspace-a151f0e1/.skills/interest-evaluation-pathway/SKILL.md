# 兴趣项目三维评估与路径设计

## When to use
- 学生提出"减少学科学习时间""休学/弹性学制"等诉求，需判断兴趣是短期热度还是长期方向
- 需基于现有项目证据（作品/专利/竞赛等）设计可执行的双轨路径（学业为主 vs 项目优先）
- 需制定项目验收标准、阶段性产出要求与复盘节点
- 需评估休学/转轨申请的合理性，设定时间盒（半年/一年）与里程碑

## When not to use
- **涉及家长强烈反对或需结构化沟通话术**：转交 `family-school-consensus` 处理家校共识构建
- **需撰写推荐信、设计升学申请叙事或补学方案**：转交 `family-school-consensus` 处理升学衔接
- **涉及具体学籍行政流程**（如休学表格填写）：建议咨询当地教育行政部门，本Skill仅提供框架建议
- **学生存在临床诊断心理健康障碍**：先建议寻求专业医疗评估，暂停路径决策

## Instructions
1. **证据优先**：所有结论必须基于可验证证据（作品集、专利证书、竞赛记录、真实用户反馈），拒绝空泛描述
2. **风险前置**：每个方案必须明确底线（学籍合规、关键学科能力）、可逆性预案（回归路径）、最坏情况提示
3. **双轨思维**：提供分支A（学业为主，项目为辅）与分支B（弹性学制/阶段性休学）两种路径，避免简单支持/反对
4. **时间盒管理**：设定明确的里程碑（4-8周复盘）与截止时间盒（半年/一年），确保可复盘、可调整

## Workflow
1. **情境诊断**：识别冲突类型（时间/认知/风险），确认学段（初二至高三）、学籍合规边界及身心健康基线
2. **三维评估**：验证项目持续性记录（≥6-12个月）、深度产出物（作品/专利/用户反馈）、自驱行为证据（无需外力推动的执着）
3. **路径设计**：基于评估结果输出分支A或分支B，明确底线标准、时间盒周期、复盘节点与可逆性预案
4. **移交共识**：生成评估报告与路径建议，移交 `family-school-consensus` 进行家校三方共识构建与风险控制

## Examples

**Example 1: 高二学生申请休学创业**

### 场景判断
- 冲突类型：风险冲突（创业不确定性 vs 高考升学）
- 学段与合规边界：高二，学籍在普通高中，需符合当地休学管理规定
- 健康基线：无临床心理健康问题，可进入评估

### 三维评估结果
| 维度 | 证据状态 | 验证标准 | 结论 |
|------|---------|---------|------|
| 持续性 | 14个月持续投入 | ≥6-12个月 | 达标 |
| 深度证据 | 产品原型、3项专利受理、真实用户付费记录 | 可验证产出物 | 强 |
| 自驱性 | 自主组建团队、凌晨迭代产品、拒绝家长安排的补习班 | 无需外力推动 | 确认 |

### 路径方案
**推荐分支**：B.弹性学制/阶段性休学（1年时间盒）

**分支B要点**：
- 时间盒：1年（高二下至高三上）
- 里程碑：
  - M1（第4周）：完成产品MVP验证，用户增长≥1000人
  - M2（第12周）：确认商业模式可行性或转型方向
  - M3（第24周）：决定是否复学或申请海外/创新通道
- 复学预案：保留学籍，制定高三下集中补学计划，委托教师保管学习档案

**分支A备选**：若M1未达标，立即转回分支A（周末维护项目，全力回归学业）

### 底线与风险控制
- **学业底线**：语文/数学基础能力测试每月一次，不得低于年级均分70%
- **合规底线**：监护人签署休学协议，不涉及大额股权代持或高危商业活动
- **可逆性预案**：连续两次里程碑未达成即触发回归机制，6周内完成复学过渡
- **最坏情况**：项目失败且学业断层，需额外1-2年重修高中课程

### 后续建议
- 移交 `family-school-consensus` 组织家长-学校-学生三方共识会议，签署备忘录
- 建议评估周期：每4周复盘一次

## Common Issues
1. **证据幻觉**：将"正在学习中"或"计划做"误认为深度证据，需严格区分"已有产出"与"未来设想"
2. **底线失守**：为追求项目突破而忽视关键学科能力（如语文阅读、数学基础）的维持，导致后续复学困难
3. **时间盒虚化**：设定"半年"却缺少具体里程碑，变成无限期休学，必须设置4-8周强制复盘点
4. **抢跑混淆**：家长将"职业抢跑"误解为"升学抢跑"，需明确区分：升学早一步不代表职业早一步，反之亦然
5. **自驱误判**：将"家长逼得很紧时的表现"误认为自驱性，需验证"无人监督时的执着程度"
