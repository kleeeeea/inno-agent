# 六条家长影响路径设计

## When to use

- 设计家长阅读计划（书单筛选逻辑、分享组织形式、学期产出物）
- 改造家长会结构（从"老师讲"转向"家长讲"的流程设计、控时方案、发言提纲）
- 制定家访共育方案（观察要点、谈话顺序、与"告状式通报"的明确区分）
- 撰写校长/班主任公开信（单主题聚焦结构、3类样例框架、发布节奏）
- 设计家长写作任务（7天微型模板、收集规则、 anonymized 编册流程）
- 构建"学生反馈影响家长"的安全机制（调查问卷设计、非羞辱性反馈方式、家长会呈现边界）

## When not to use

- 若核心诉求是"教师抵触家校工作、要求设计去KPI化激励机制" → 转交 **teacher-dignity-incentive** 处理
- 若涉及心理疾病治疗、危机干预、家暴处置 → 建议立即转介专业心理或社工机构，不提供操作方案
- 若要求商业化招生运营、家长变现策略或侵入隐私的强制家访（查看财产、进入卧室）→ 明确拒绝，超出合规边界
- 若请求替代课堂教学设计或学科提分方案 → 仅提供原则性建议，不深入设计

## Instructions

1. **阅读改变路径**：每学期精选2-3本家庭教育书目，强调"家长读书的形象本身就是对孩子的感染"，组织线上轻分享或读书会，产出"家长阅读札记"汇编
2. **写作表达路径**：设计7天微型写作任务（每天2句话），引导家长写给孩子，不求文采重真诚；建立"优秀来信匿名编册"机制，形成同伴感染
3. **家长互学路径**：改造家长会结构，减少教师单向宣讲，增加"家长主讲"环节（分享如何转变孩子/改善亲子沟通），提供发言提纲模板与3分钟控时规则
4. **公开信路径**：校长每学期2-3封公开信，一封信只谈一个教育理念（如看待成绩、习惯培养）；班主任围绕具体议题（如电子产品管理）写连续信件
5. **学生反馈路径**：设计匿名调查（"最不能接受父母的3个行为"），家长会上以"孩子希望父母改进的TOP5"形式呈现，避免点名羞辱，利用"面子机制"驱动改变
6. **家访共育路径**：明确区分"电话通报"与"家访共育"，家访核心任务是营造平等谈心氛围、观察家庭生态（房间布置、劳动习惯、代际互动），将积极观察点带回班级表扬

## Workflow

1. **诊断与选型（第1周）**：确认目标家长群体画像（沟通型/习惯型/学业型/亲子冲突型），选定2-3条路径组合，明确学期核心矛盾
2. **机制搭建（第2-3周）**：制定书单与分享节奏、写作模板与收集规则、家长会流程与发言人筛选、家访观察清单与谈话结构、公开信发布日历
3. **学期运行（第4-12周）**：按月推进主题（阅读月/写作月/沟通月/习惯月），同步收集过程性产出（信件、观察记录、家长分享实录）
4. **复盘迭代（第13-14周）**：评估家长参与率、文本反馈质量、学生行为变化，调整路径权重与主题，形成下学期行动包

## Output Format

每次响应必须包含以下固定字段：

- **场景判断**：目标家长群体特征与主要矛盾类型（沟通/习惯/学业/亲子）
- **六路径组合方案**：选定的具体路径及组合逻辑（如"阅读+写作+家访"轻量级组合，或六路径全激活）
- **学期节奏表**：第几周做什么、责任人（校长/德育主任/班主任）、具体产出物（书单、信、册子、观察记录）
- **可用话术**：关键节点的表达范例（如家访开场白、家长会过渡语、公开信开篇结构）
- **防形式主义措施**：如何确保真实参与（如家访不做数量要求、写作不重文采重真诚、家长会发言人提前访谈）
- **风险提示**：隐私保护要点（匿名规则）、避免羞辱的边界（不点名、不对比）、合规底线（家庭教育促进法）
- **后续评估**：可量化指标（参与率、信件收集量）与可质性指标（家长反馈文本分析、学生行为观察记录）

## Examples

**示例1：校长公开信结构**
- 主题：如何看待孩子的考试成绩
- 结构：①共情当前焦虑（"我理解看到分数时的忐忑"）②重新定义分数（"一次考试只是学习过程的切片"）③具体行动建议（"和孩子一起分析错题比质问更重要"）④结尾赋能（"相信孩子成长的可能性"）

**示例2：家长写作7天模板**
- Day1：孩子最近让你感动的1个瞬间
- Day2：你小时候面对困难时，希望父母如何对你说话
- Day3：看到孩子今天做的1件让你欣慰的小事
- Day4：你想对孩子说的1句抱歉或感谢
- Day5：你眼中孩子最独特的1个优点
- Day6：你对孩子的1个具体期待（非学业）
- Day7：写1句你希望孩子记住的话

**示例3：家长会"家长讲"流程**
- 前15分钟：班主任简要反馈班级整体情况（数据+感谢）
- 中间30分钟：2-3位家长分享（提前一周沟通内容、提供提纲模板、每人8分钟+2分钟互动）
- 最后15分钟：班主任总结提炼共性经验，布置下周轻量级家校任务

**示例4：家访观察点清单**
- 家庭氛围：进门时家庭成员互动语气、房间整洁度与个性化布置
- 亲子关系：孩子与家长的物理距离、眼神交流、谁主导谈话
- 代际互动：对老人的称呼与态度、是否有三代同堂的相处细节
- 劳动习惯：孩子是否有固定家务、物品归位情况
- 谈话顺序：先观察环境→再肯定家庭积极点→最后平等探讨共育策略

## Common Issues

- **路径单一依赖**：仅设计家访或仅改造家长会，缺少六路径组合导致的干预深度不足
- **形式化风险**：写作任务变成"征文比赛"导致家长应付，家访沦为"上门告状"引发防御对立
- **隐私侵犯**：学生反馈调查未做匿名处理变成"批斗大会"，家访要求查看卧室/家庭财产构成越界
- **周期断裂**：缺乏9-12周学期节奏设计，活动变成一次性运动难以持续
- **羞辱性反馈**：公开点名"问题家长"或对比"优秀 vs 落后"家庭，破坏信任基础
