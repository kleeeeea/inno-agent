# 教师尊严感激励机制

## When to use
- 教师明确表达对家校工作的抵触情绪（"又要考核打卡""增加负担"）
- 管理者担心强推方案导致弄虚作假（虚假家访记录、应付式打卡）
- 需设计"低考核、高激励"的学期推进机制
- 需构建教师职业尊严感与可见成就体系（非金钱激励）

## When not to use
- 涉及具体六条路径的内容设计（书单选择、家长会流程、家访观察点、写作模板等）→ 转 **six-path-engagement-design**
- 涉及学生心理危机、家暴或需专业治疗的情况 → 建议转介专业心理或社工机构
- 涉及商业招生变现或家长运营 → 明确拒绝
- 替代课堂教学或学科教学设计 → 拒绝

## Instructions
1. **去KPI化原则**：取消家访数量、打卡频次等硬性指标，改用"氛围驱动+正向激励"
2. **尊严感优先**：通过故事出版、学生颁奖、校本论坛等提升职业自豪，而非考核压力
3. **关系修复**：先改善师生关系与教师情绪，再推进家校协同任务
4. **防造假设计**：用质性成果（故事册、真实案例）替代量化表格，减少应付空间

## Workflow
1. **诊断抵触根因**：区分是"考核焦虑""尊严受损"还是"工作量过载"
2. **移除绩效绑定**：取消数量指标，改为"自愿参与+成果展示"机制
3. **搭建激励体系**：设计校本分享会、公众号推文、教育故事出版、学生表彰等具体载体
4. **建立防假机制**：设置过程性展示（如故事分享会现场讲述）替代纸质材料检查

## Output Format
- **场景判断**：教师抵触类型（考核反感/形式疲惫/尊严缺失）
- **根因分析**：具体触发点（KPI压力/过往挫败/时间冲突）
- **去KPI化方案**：取消的指标、替代的评价方式
- **尊严感建设措施**：分享平台、出版机会、学生认可机制
- **防造假机制**：真实性保障措施（如现场讲述、学生验证）
- **学期节奏表**：第几周启动、关键节点、责任人
- **风险提示**：可能的形式主义变异点与预防

## Examples

**示例：教师抵触家访的激励重构**
- 原方案：每学期必须家访10户，提交记录表
- 优化方案：取消数量要求，改为"教育故事分享会"——教师自愿分享家访中最触动的1个家庭故事，优秀者编入《每一个孩子都是故事》校本出版物，教师节由学生颁发"最懂我的老师"证书

**示例：防家长会造假**
- 风险：要求"家长主讲"后，教师可能找托演戏
- 措施：不检查"家长是否讲了"，而关注"家长反馈是否真实"——会后收集3位家长的语音反馈（真诚即可），教师整理成300字教育随笔，在校本论坛朗读而非提交表格

## Common Issues
- **管理者考核惯性**：校长难以放弃"可量化的控制感"，需强调"质性成果更具传播价值"
- **形式造假转移**：从"表格造假"转为"故事造假"，需通过细节追问（学生验证、现场互动）确保真实
- **激励不均抱怨**：部分教师认为"总是那几个人被表彰"，需设置多元展示平台（学科组、年级组、全校层）分散机会
