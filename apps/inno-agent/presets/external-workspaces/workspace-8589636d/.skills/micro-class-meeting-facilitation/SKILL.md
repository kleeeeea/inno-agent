# 微班会 Facilitation 设计

## When to use
- 班级突发师生冲突（如任课教师被气走、课堂冷场尴尬）需即时召开微班会重建对话场域
- 群体性课堂沉默或学业断崖危机（如测试大面积红灯）需在碎片化时间（10-15分钟）进行高张力干预
- 需要可直接照读的"自己人效应"情感破冰话术与"找病根-寻方法-重行动"三问驱动流程设计

## When not to use
- 涉及学生严重心理障碍、自伤自杀危机 → 立即转介学校心理中心，暂停微班会计划
- 涉及法律纠纷的师生冲突（如体罚指控、人身伤害） → 建议寻求专业法律支持，不启动微班会
- 需要设计4人小队每日5分钟微测互批机制、组内轮换规则或《家校双向契约》文本 → 使用 `decentralized-self-discipline-system` 技能
- 需要45分钟常规主题班会设计（非碎片化微班会场景）

## Instructions
1. **前置调查诊断**：提示用户必须分别单独约谈班长、课代表、受挫任课教师，获取"课堂效率/走神率/基础难度"的分层原初信息，识别教师盲区（如不同学业层次学生的心理防御机制）
2. **情感破冰设计**：基于"自己人效应"生成带语气停顿（...）与肢体动作（[环视全班]）标注的现场话术，建立"我们是一伙"的同盟感，坚决避免说教场、批评会
3. **三问驱动架构**：设计10分钟微班会的三层递进——"一问找病根"（好/中/差三梯度代表发言，将问题剥离为技术漏洞而非人格否定）、"二问寻方法"（头脑风暴引导词）、"三问重行动"（自律共识提炼）
4. **风险预警前置**：在方案开头明确提示该场景最易陷入的误区（如"切勿在此刻追责起哄源头""切勿翻旧账"），并标注反面案例警示

## Workflow
1. **危机诊断与前置约谈**：解析危机类型（冲突/沉默/断崖），输出《前置调查访谈提纲》，明确必须约谈的3类对象（班长、课代表、受挫任课教师）及关键探查问题（如"学生不回答是因为不会/怕嘲笑/还是对抗"）
2. **微班会现场架构设计**（10分钟）：
   - **情感破冰（2分钟）**：提供可直接照读的2-3句高情商台词脚本，含停顿与动作标注，建立尊重与信任同盟
   - **一问找病根（3分钟）**：设计"好、中、差"三梯度学生代表发言引导策略，将抽象问题精准剥离为具体技术漏洞（如"公式记不住/基础概念混淆"）
   - **二问寻方法（3分钟）**：预设学生头脑风暴的引导词与班主任补充的"科学时间安排法"切入点，避免班主任单向灌输
   - **三问重行动（2分钟）**：明确"自律是决定性保障"的共识提炼话术，完成自我教育转化
3. **行动衔接指引**：简要说明后续需依托4人小队机制落地（具体小队规则设计移交 `decentralized-self-discipline-system`），并提供家校协同备忘录框架（具体契约文本移交 sibling skill）
4. **成效检验指标**：提供"后三问"评估指标（学到了什么/是否有助成长/能否自主解决），指导案例结构化入库

## Output Format
- **场景判断**：危机类型（师生冲突/群体性沉默/学业断崖）与风险等级，附"反面案例"警示
- **前置调查要点**：必须完成的3类约谈对象及关键探查问题清单（附心理学依据）
- **情感破冰话术**：含语气停顿、肢体动作标注的2-3句现场台词脚本，附"自己人效应"原理说明
- **三问驱动设计**：
  - 一问找病根：三梯度代表发言引导策略（学霸聚焦技术细节/中等生聚焦执行难点/后进生聚焦情绪卡点）
  - 二问寻方法：头脑风暴引导词与班主任补充切入点（如"每日5分钟碎片利用法"）
  - 三问重行动：自律共识提炼话术与现场互动设计
- **风险提示**：该场景最易陷入的3个误区（如避免追责源头、避免泛化否定、避免对比其他班级）
- **后续评估**："后三问"成效检验指标与案例入库分类标签建议

## Examples

**示例1：化学课冷场与成绩断崖（群体性沉默）**
- **场景判断**：学业断崖（测试平均分30分）+ 课堂沉默，风险等级：高（师生对立+家长群焦虑）
- **前置调查要点**：约谈课代表了解"不回答问题"真实原因（基础差/怕嘲笑/听不懂）；约谈化学教师了解情绪触发点（是否觉得"付出喂了狗"）；约谈班长了解群体心理（是否出现习得性无助）
- **情感破冰话术**："这次成绩让大家抬不起头，老师也知道你们不是不想学...（停顿3秒，[走到教室中间，双手撑讲台]）没学好，咱们可以现在赶上去，老师和你们是同一战线的。"
- **三问驱动设计**：
  - 一问：请成绩好的张同学说说"你觉得大家卡在哪一步"，请中等生李同学补充"是不是跟不上节奏"，请基础薄弱的王同学确认"是不是公式记不住而不是脑子笨"
  - 二问："如果每天只攻克一个小漏洞，你们觉得从哪开始？"引导学生提出分类复习，班主任补充"午间5分钟碎片利用法"
  - 三问："最后问大家，成绩差是智商不够还是暂时没管好自己？（停顿，环视）对，是自律。自律才是决定性因素，从今天起我们..."
- **风险提示**：切勿在此时追责"谁带头不回答问题"；切勿对比其他班级"人家班怎么没问题"
- **后续评估**：使用"后三问"检验学生是否将归因从"我笨"转为"我暂时没做好技术点"

**示例2：突发课堂冲突（英语课起哄教师气走）**
- **场景判断**：突发师生冲突（教师被气走），风险等级：极高（需先单独安抚教师再开班会）
- **前置调查要点**：约谈起哄带头者了解动机（看小说被收后的从众/发泄）；约谈课代表了解英语教师情绪状态（是否已哭/是否拒绝再进班）
- **情感破冰话术**："今天的事情，老师知道你们不是针对英语老师这个人，（停顿，[严肃环视]）但结果是老师伤心了，我们也损失了学习时间。现在咱们不是来批斗的，是来修补水坝的，（语气转温和）老师相信你们心里也不好受。"
- **三问驱动设计**：
  - 一问：邀请起哄参与者、旁观同学、课代表分别发言，剥离为"缺乏边界意识/情绪管理失效"而非"品德败坏/班风腐烂"
  - 二问：头脑风暴"下次遇到不满，如何既表达情绪又不伤害课堂"，引导学生提出"课后沟通机制"
  - 三问：共识"自律是尊重他人的底线，也是保护自己的铠甲"
- **风险提示**：切勿让起哄学生在全班面前"当众检讨"（易激发二次对抗）；切勿全盘否定"你们这个班太差了"
- **后续评估**：观察次日英语课抬头率与互动率，检验"学到了什么/是否有助成长/能否自主解决"

## Common Issues
1. **话术沦为说教**：避免使用"你们要深刻反思""你们让我很失望"等评判性语言，必须转换为"我们遇到了什么困难""我们可以一起做什么"的同盟句式
2. **忽略分层差异**：切勿只让学霸发言或只点名批评差生，必须设计"好-中-差"三梯度代表发言，防止
