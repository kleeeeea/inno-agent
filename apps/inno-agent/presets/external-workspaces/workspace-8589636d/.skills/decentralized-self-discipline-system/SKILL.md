# 去中心化自律机制与家校协同

## When to use
- 微班会已结束，需要将"自律共识"转化为具体落地机制，且班主任无法长期亲力盯防时
- 现有小组互批机制面临流产（如组长压力过大、落后生家长投诉标签化）需要紧急救场时
- 家庭端出现焦虑传导、过度干预或放任不管，需要建立情绪阻断与家校协同闭环时
- 需要将宏大复习计划拆解为"每日5分钟"极低门槛行动，依托生生互助完成技术漏洞修补时

## When not to use
- 若尚未召开微班会或需要设计"三问驱动"现场话术、情感破冰策略，应交由 **micro-class-meeting-facilitation** 技能处理
- 若涉及学生严重心理障碍、自伤自杀危机，立即转介学校心理中心，停止任何机制设计
- 若涉及法律纠纷、教师严重情绪失控或家长威胁极端行动，优先启动学校危机干预预案

## Instructions
- **去中心化铁律**：班主任仅制定规则与提供工具，不直接参与每日批改与订正，监督权下放至小组长
- **5分钟时间红线**：微测题量严格控制在3-5题，防止任务膨胀导致机制崩溃
- **组长轮换防标签**：首轮组长由中等生担任，每2周强制轮换，确保落后生也有机会担任记录员等非评判角色
- **焦虑阻断机制**：家校契约核心为"每日只确认掌握的1个点，禁止翻旧账"，用结构化沟通替代自由发挥

## Workflow
1. **异质小队组建**：按"1好2中1差"或"1好1中2差"组建4人小队，明确首轮组长（中等生优先），制定2周轮换规则与组长职责清单
2. **微测极简设计**：联动任课教师提炼"每日1个核心知识点"，设计3-5题微测卷，固定于午休12:40-12:50实施，红笔批改当场返还
3. **互批订正闭环**：组长批改→组员当场订正→全组过关签字→未过关者课后5分钟组内互助→班主任每日仅收齐签字表抽查
4. **家校双向契约**：发送《家校协同情绪护航双向契约》，明确家长"三做三不做"（做每日1点确认、做正向暗示、做后勤；不翻旧账、不泛化否定、不越级施压），要求家长签字回执
5. **动态迭代机制**：每周统计过关率，每月轮换组长，根据执行阻力调整题目难度；建立"组长互助联盟"定期减负谈话

## Output Format
- **场景判断**：当前处于机制设计的哪个阶段（初建/流产危机/迭代优化）
- **处理方案**：4人小队组建规则、每日微测流程、组长轮换机制、家校契约关键条款
- **可用话术**：小组长职责说明（"你是技术漏洞扫描仪，不是小老师"）、家长沟通话术（"我们只盯每日1个绿灯，不数历史红灯"）
- **工具模板**：《每日5分钟微测通关清单》（含知识点/3-5题/订正栏/签字栏）、《家校双向契约书》、《小组互批记录表》
- **风险提示**：机制易崩溃点（组长压力过载、家长投诉标签化、任课教师不配合出题）及应急补丁
- **后续评估指标**：一周后小测合格率、学生睡眠改善率、家长群焦虑发帖回落度、任课教师课堂满意度

## Examples

**《家校协同情绪护航双向契约》核心条款：**
> "家长承诺：每日仅询问孩子'今天掌握了哪一个具体知识点'，并予以确认；不做大面积试卷复盘，不与其他孩子比较，不向校级越级投诉。教师承诺：每日仅布置5分钟可完成的基础过关任务，不额外增加作业量，及时通报孩子每日掌握的那1个亮点。"

**小组长职责话术（防止权力滥用）：**
> "你的任务不是当小老师，而是当技术漏洞扫描仪。用红笔打勾或圈出错误，但不对同学说'你怎么这么笨'，而是说'这个公式咱们再对一遍'。每天批改时间不得超过5分钟，遇到争议直接找老师，不自己硬扛。"

**落后生家长投诉应对话术：**
> "这个机制不是为了给孩子贴标签，而是让每个孩子都体验学习责任。下周组长轮换，您的孩子将担任小组记录员，负责统计全组进步点数。我们只记录每日掌握的1个绿灯，不累计历史红灯，请您配合在家也只问那1个绿灯。"

## Common Issues
- **小组长压力过载罢工**：建立"组长互助联盟"，每周五中午班主任请组长喝饮料减负谈话；设置"双组长制"（两人搭档批改）
- **家长继续翻旧账焦虑传导**：强制使用"每日1点掌握确认"模板，要求家长签字回执，用结构化工具替代自由沟通；对持续越级投诉者单独电话隔离沟通
- **任课教师不配合提炼微测题**：提供"知识点清单提取话术"（"不用您编新题，只需在课本上圈画3个今日最基础公式，我来改成填空题"），降低教师负担
- **机制流于形式学生敷衍**：引入"随机抽测+全组连坐"机制（每日随机抽1人代表全组接受教师抽检，未过关则全组次日重测），提升组内互助真实性
