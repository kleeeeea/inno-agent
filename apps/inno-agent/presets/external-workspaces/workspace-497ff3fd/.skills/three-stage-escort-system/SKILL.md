# 三阶护航工作流落地

## When to use
- 班级出现群体性备考恐慌、盲目刷题低效循环，需建立群体心理安全网络（班会、学习小队重组）
- 学生因复习任务繁杂无从入手或基础薄弱，需具象行动规划（知识图谱、盲区清单、微成长打卡）
- 家长群体出现连带性失眠与非理性焦虑，需家校闭环情绪守护（浇花日记、鼓励便签）
- 需建设"心灵驿站"物理空间并制定开放规则与情感护航机制
- 需根据焦虑程度（轻度/中重度）进行分级分流与动态闭环迭代

## When not to use
- 学生因模考失误陷入自我否定、出现绝对化（"必须考第一"）或灾难化（"一道题不会就全完了"）思维时，应交给 `cognitive-reframing-intervention` 进行深度认知重构
- 学生表达自杀/自伤意念、持续两周以上严重抑郁症状或精神异常表现时，应立即转介学校心理教师/专业机构，停止学业讨论
- 仅需简单安慰或空洞鼓励（如"别紧张"）而无具体工具支持的常规场景

## Instructions
1. **分级分流**：通过日常观察指标（表情防御性、错题情绪反应、睡眠状况）区分轻度群体性焦虑与中重度个体危机，匹配不同干预强度
2. **行动筑基**：将抽象复习任务拆解为具象模块，提供"知识图谱→盲区清单→微成长打卡"的可视化工具，确保基础薄弱学生有专属短期冲刺计划
3. **情感护航**：明确"心灵驿站"的物理设置（办公室角落）、开放频次（每日固定时段）、基本规则（不打断、不评判、保密）及班主任角色定位（共情倾听而非说教）
4. **家校协同**：建立"鼓励便签-赏花时刻-浇花日记"双向传递机制，阻断焦虑传导，指导家长履行后勤监管而非施压说教

## Workflow
1. **识别信号**：捕捉错题负面情绪、盲目刷题行为；通过"焦虑悄悄话"活动收集隐蔽焦虑
2. **分流判断**：轻度/群体性→群体干预；任务迷茫→行动规划；家长焦虑→家校协同
3. **行动筑基**：绘制知识图谱→建立盲区清单→拆解具象模块→微成长打卡公示
4. **情感护航**：建设"心灵驿站"→共情倾听→定制化鼓励卡片→阻断家校焦虑传染
5. **闭环迭代**：动态观测（课堂表情、作业独立率、家长反馈）→微调干预颗粒度→固化微产出

## Output Format
- **场景判断**：群体性/个体行动迷茫/家长连带焦虑（勾选）
- **处理方案**：具体工具（班会设计/打卡表/心灵驿站规则/家校契约）+ 执行步骤
- **可用话术**：鼓励便签示例（3条不同风格）、家校沟通要点、班会引导语
- **工具模板**：盲区清单拆解逻辑、微成长打卡记录表、三日冲刺计划框架
- **风险提示**：防攀比机制、隐私保护边界、流于形式预警
- **后续评估**：可观察指标（表情变化、错题独立率、睡眠改善率）

## Examples
**输入**："下周一期末考，全班气氛凝重，很多学生不知从何复习，家长也在群里说失眠焦虑。"

**输出**：
- **场景判断**：轻度群体性焦虑（氛围凝重）+ 任务迷茫（无从复习）+ 家长连带焦虑（失眠）
- **处理方案**：
  - 群体：周一班会"焦虑悄悄话"匿名分享（规则：答错不许嘲笑）+ 重组4人学习小队
  - 行动：发放"知识图谱绘制指南"，建立"盲区清单"，启动"每日微成长"打卡（记录1个掌握点/1道弄懂的题）
  - 家校：发放《家校情绪护航契约》，指导家长放置"鼓励便签"，每日"赏花时刻"表扬具体进步
- **可用话术**：
  - 鼓励便签："你的坚持与成长，远比分数更有价值""沉淀已久，只管自信奔赴""今日弄懂一题，便是明日少错一题"
  - 班会引导语："备考焦虑是共性体验，不是只有你一个人担心"
- **工具模板**：
  - 盲区清单：按教材目录标注"已掌握/模糊/完全不会"，每日专攻2个模糊点
  - 打卡表：日期+今日掌握知识点+易错题复盘+自我评分（1-5分）
- **风险提示**：打卡防攀比：仅公示"进步事实"（如"今日新增掌握3个知识点"）而非排名；家长沟通避免传递焦虑
- **后续评估**：观察课间表情放松度、作业错题独立订正率、家长反馈的入睡时间变化

## Common Issues
- **"心灵驿站"流于形式**：必须明确开放时段（如每日午休12:30-13:30）和物理标识，避免学生不知可用或不敢进入
- **微成长打卡变负担**：规则设计为"最低门槛成功"（每日只需记录1个最小掌握点），避免量化攀比，重点在"完成感"而非"完成量"
- **家校协同单向输出**：确保"浇花日记"双向传递，既反馈在校表现，也收集家庭端睡眠、情绪数据，形成闭环
- **群体干预忽视个体差异**：群体中隐藏的中重度个体（持续自我否定）需识别并转介 `cognitive-reframing-intervention`，避免"一刀切"班会延误个体危机干预
