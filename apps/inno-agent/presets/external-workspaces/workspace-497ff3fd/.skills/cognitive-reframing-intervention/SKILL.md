# 认知破局与合理情绪干预

## When to use
- 学生因微小失误（如错一道题）陷入情绪崩溃或强烈自我否定，出现"肯定考不上了"等言语
- 识别到绝对化信念："必须考年级第一"、"绝不能出错"、"我应该完美无缺"
- 识别到灾难化思维："这次考不好人生就完了"、"全完了"、"彻底失败"
- 需要设计一对一深度谈心的话术流程，将"非理性失败恐惧"转化为心理韧性契机

## When not to use
- 涉及全班性班会组织、学习小队重组、"心灵驿站"物理空间建设、家校沟通策略 → 转交 **three-stage-escort-system**
- 涉及复习计划制定、打卡机制设计、"盲区清单"拆解 → 转交 **three-stage-escort-system**
- 学生表达自杀/自伤意念、持续两周以上严重情绪低落、出现幻觉妄想 → **立即安全转介**，停止学业讨论，24小时内联系学校心理教师/专业机构
- 仅需一般性鼓励或常规班级管理 → 使用普通问答

## Instructions
- **ABC理论应用**：识别诱发事件(A)→挑战非理性信念(B)→改善情绪后果(C)，重点干预B环节
- **核心重构框架**：将"错题是判决书"重构为"错题是信号灯"；将"必须/绝对"转化为"尽己所能/允许失误"；将"全完了"转化为"及时补漏"
- **苏格拉底式提问**：用"如果错一道题就考不上，那平时练习的意义是什么？"引导学生自我质疑非理性逻辑
- **第一人称共情**：使用"我理解这种感受""很多人在这种时刻都会这样想"建立心理安全，避免"你应该放松"等否定式说教

## Workflow
1. **信念识别**：捕捉学生言语中的"必须""肯定""全完了""绝对"等关键词，判定为绝对化或灾难化思维
2. **共情破冰**：先接纳情绪："看到你这么难过，我知道这次失误对你打击很大"，建立信任后再进入认知层面
3. **认知重构**：挑战非理性信念："考不上重点高中＝人生失败，这个等式真的成立吗？"引导学生发现逻辑漏洞
4. **意义重塑**：赋予错题新定义："这道题现在错了，是帮你发现了复习盲区，是考前的信号灯，不是判决"
5. **理性锚定**：与学生共同确立一句可记忆的理性宣言，如"尽己所能便是最优解"，作为未来焦虑时的自我提醒

## Output Format
- **场景判断**：学生当前认知扭曲类型（绝对化/灾难化/过度概括）及严重程度（轻度/中度/重度）
- **非理性信念摘录**：学生原话中的关键非理性表述（如"我肯定完了"）
- **处理方案**：分步骤谈话流程（破冰→质疑→重构→锚定），含每阶段目标与时长建议
- **可用话术**：具体对话示例（教师话术+预期学生反应），含RET标准话术模板
- **风险提示**：是否需启动安全转介（自杀/自伤意念）或转交群体干预技能
- **后续评估**：观察指标（情绪平复度、后续面对失误的反应速度、自我对话内容变化）

## Examples
**场景**：学生小浩因错一道数学题哭着说"肯定考不上高中了，人生全完了"。

**处理方案**：
1. **共情破冰**："看到你现在这么难过，老师理解这种失望感。如果是我准备了这么久还做错，也会很难受。"
2. **质疑灾难化**："你刚才说'肯定考不上'，是基于这道题的分值推算的，还是一种感觉？如果这道题占5分，意味着剩下145分都没机会了吗？"
3. **认知重构**："这道题现在错了，其实是好事——它在考前帮你发现了这个知识漏洞，就像信号灯提醒前方有坑，你是愿意考前看到信号灯，还是考场上才掉进去？"
4. **锚定宣言**："我们一起定个新信念：'错题是信号灯，不是判决书。及时补漏，就是进步。'"

## Common Issues
- **避免感受否定**：不说"你想太多了""别矫情"，而是"这种感受很正常，我们来看看想法是否需要调整"
- **警惕说教陷阱**：RET是对话技术而非单向灌输，避免直接告诉学生"你应该这样想"，而是引导其自我发现逻辑矛盾
- **区分焦虑与抑郁**：RET适用于焦虑相关的认知扭曲；若学生持续情绪低落、丧失兴趣，需转介临床评估
- **单次干预局限**：长期形成的非理性信念需多次强化，单次谈话后需配合行动筑基（转交三阶护航系统）巩固新认知
