# 慢性作业抗拒诊断分析

## When to use
- 学生长期呈现"空本子"状态，对常规批评呈现"情绪绝缘与规则脱敏"
- 需要区分认知超载（能力不足）、习得性无助（归因失败）、权力试探（博弈行为）或混合类型
- 评估风险等级（绿/黄/红）及家庭支持系统塌陷程度
- 判断当前教育优先目标：底线重塑（完成作业）vs 心理修复（重建自信）

## When not to use
- **已确诊需生成干预方案**：如需设计任务降级策略、时间置换规则、强制留堂流程或具体话术脚本 → **转交 `helplessness-intervention`**
- **危机状态**：学生出现自伤自杀言语/行为、严重情绪危机（持续崩溃、躯体化反应）、疑似家庭暴力或虐待 → **立即转介心理危机干预热线/校医/德育处，停止诊断流程**
- **法律对抗**：家长已采取法律手段或表现出激烈敌意威胁 → **转介学校管理层/法务，暂缓教育干预**

## Instructions
- **严格锚定四维**：学段（小学中高年级/初中）、风险等级、问题归类、教育优先目标权重
- **先筛查后诊断**：优先执行安全红线检查（自伤、家暴、重病、法律对抗），任一命中立即暂停并转介
- **成因三分法**：区分认知超载（作业难度远超最近发展区）、习得性无助（反复失败后固化归因）、权力试探（摸清教师底线后的零成本博弈）
- **材料保真**：基于用户提供事实分析，不虚构家庭结构、既往病史或具体事件经过

## Workflow
1. **前置适配**：明确年级段认知发展水平、风险等级（绿/黄/红）、问题类型（认知/心理/博弈/混合）、教育目标权重
2. **安全筛查**：检查自伤自杀、家暴虐待、严重躯体疾病、家长法律对抗等红线指标，命中即停
3. **行为图谱**：分析空本子频率、抗拒方式（沉默/对抗/逃避）、免疫机制形成史
4. **成因矩阵**：
   - 认知超载：评估作业难度与学生现有能力落差
   - 家庭支架：识别监管真空点（留守儿童/隐性失职）
   - 权力试探：判断是否存在"交空本子-无实质后果"的博弈循环
5. **组件预配置**：根据条件分支预建议组件权重（如情绪危机→调高共情安抚、微小行动、发展支持）

## Output Format
- **场景判断**：学段锚定/风险等级（绿黄红）/问题归类（认知超载/习得性无助/权力试探/混合）/教育优先目标
- **安全筛查**：红线指标命中情况（如无则标注"通过"）
- **成因分析**：
  - 认知维度：超载程度、最近发展区匹配建议
  - 家庭维度：支架断裂点、监管真空评估
  - 博弈维度：权力试探模式、免疫机制形成史
- **适配建议**：是否适合进入干预流程/前置缓冲需求（如先心理修复后底线重塑）/强制留堂可行性评估
- **转介指引**：如需要，明确转介路径（心理干预/helplessness-intervention/管理层）

## Examples

**Input**：班上小明（六年级）一学期没交语文作业，问他就低头不说话，偶尔说"反正写不好"。父母离异，跟奶奶住。其他老师反映他上课发呆，但没有情绪爆发。

**Output**：
- **场景判断**：小学高年级/黄色风险（家庭支持塌陷）/习得性无助为主，伴轻度认知超载/教育优先目标：心理修复权重60%，底线重塑40%
- **安全筛查**：通过（无自伤、家暴、法律对抗迹象）
- **成因分析**：
  - 认知维度：作文类开放作业远超其断裂的支架支持能力，需提供句子级任务降级
  - 家庭维度：隔代抚养监管真空，课后作业环境支架完全断裂
  - 博弈维度：呈现"习得性无助"特征（归因失败），暂无"光脚不怕穿鞋"的权力试探迹象
- **适配建议**：适合进入干预流程，但**暂缓强制留堂**（先重建安全依恋），需先实施2周认知支架重建，再引入时间置换
- **转介指引**：建议班主任先启动 `helplessness-intervention` 设计任务降级方案，同步关注心理状态

## Common Issues
- **诊断越界**：在本阶段生成具体"强制留堂话术"或"三步干预流程"（应留待 `helplessness-intervention`）
- **成因混淆**：将"权力试探"（故意对抗）误判为"习得性无助"（能力归因），导致后续干预方向错误
- **安全遗漏**：未筛查自伤/家暴迹象即建议"强制留堂"或"课后单独谈话"
- **虚构填充**：编造"家长已同意配合"或"学生曾有某病史"等未提及细节支撑诊断结论
