# 习得性无助干预方案生成

## When to use

- 学生已确诊为认知超载、习得性无助或权力试探类型，且风险等级明确（绿/黄/红级）
- 需要设计"跳一跳够得着"的任务降级策略与具体作业替换方案（如作文→单词抄写）
- 需要生成课间/午休时间置换规则或强制留堂流程的具体话术与操作细节
- 需要配置共情安抚、认知支架、微小行动等干预组件的具体实施方案
- 需要制定24小时内可落地的微行动与家校沟通策略

## When not to use

- **未诊断转介**：尚未完成前置诊断（学段锚定、风险等级、问题归类）时，应转介至 `chronic-homework-diagnosis`
- **危机转介**：学生出现自伤自杀言语/行为、严重情绪危机（持续崩溃、躯体化）、疑似家暴或虐待时，立即暂停并转介心理危机干预热线/校医/德育处
- **法律对抗**：家长已采取法律手段、表现出强烈敌意或威胁投诉时，转介学校管理层处理，暂停强制留堂
- **身体限制**：学生身体状况不允许留校（如疾病、饥饿、严重疲劳）时，调整策略

## Instructions

- **三步递进**：严格遵循"任务降级→时间置换→强制留堂"顺序，不可跳过前置步骤直接实施留堂
- **组件必含**：每套方案必须融合至少3类组件（共情安抚、事实澄清、价值边界、认知支架、微小行动、发展支持），且**必须包含1个发展支持类组件**（认知支架、微小行动或发展支持）
- **条件分支**：IF 情绪危机 THEN 调高共情安抚+微小行动；IF 规则违纪 THEN 调高价值边界+认知支架；IF 高压冲突/家长激烈 THEN 先调高共情安抚与事实澄清，暂缓权责界定
- **合规底线**：强制留堂必须满足"不加量、不清账不放行"、教师陪伴而非监视、单次留校不超过18:30（小学）或合理时限，严禁体罚、变相体罚及教育冷暴力
- **适龄转译**：话术需匹配小学中高年级或初中认知水平，避免司法/行政威慑或管理黑话，使用"清账""底线"等可理解概念

## Workflow

1. **方案适配**：确认诊断结论（问题类型、风险等级、教育优先目标权重：底线重塑 vs 心理修复）
2. **组件配置**：根据条件分支选择组件组合，确保含发展支持类组件
3. **三步设计**：
   - **任务降级**：按最近发展区拆解作业（如作文→段落→句子→5个单词；数学卷→3道基础题→例题模仿）
   - **时间置换**：应用普雷马克原理，规划课间/午休/副课时段，明确"先完成底线任务，再享受闲暇权利"
   - **强制留堂**：（如适用）制定清账标准、陪伴方式、时限与安全出口，避免punitive氛围
4. **话术生成**：为每一步配备具体话术脚本，包含教师示范语与学生抗拒回应策略
5. **微行动**：制定24小时内可落地的最小行动单元（如"明天课间完成3道基础题"）与检查节点
6. **合规审查**：对照《中小学教育惩戒规则（试行）》检查无体罚、无冷暴力、时限合规

## Output Format

- **前置确认**：诊断结论摘要（问题类型、风险等级、教育优先目标）
- **组件配置**：选用的3+类组件及条件分支依据（如"因情绪危机，选用共情安抚+微小行动+认知支架"）
- **三步干预方案**：
  - **任务降级**：具体降级路径、底线任务量、完成标准（如"抄写5个核心词汇，非10个"）
  - **时间置换**：占用时段、普雷马克规则、监督方式（如"午休12:40-13:00讲台旁，完成即自由阅读"）
  - **强制留堂**：（如适用）清账标准、陪伴方式、时限、安全出口（如"放学后至17:30，教师陪同，做完即放行"）
- **话术脚本**：各步骤教师示范语、学生可能的抗拒（沉默/哭闹/对抗）及回应策略
- **24小时微行动**：具体可执行任务、完成标准、检查节点（如"明日完成3词抄写，午休前检查"）
- **风险提示**：实施中的高风险信号（情绪崩溃、攻击性行为、家长激烈对抗）及暂停条件
- **后续评估**：3天后/一周后复盘节点、规则修补动作、调整触发条件

## Examples

**输入**：五年级学生，长期不交作文，确诊习得性无助（黄级），家庭支架断裂，教育优先目标为底线重塑。

**输出**：
- **组件配置**：共情安抚（高）+ 认知支架（中）+ 微小行动（必含）
- **任务降级**：今日作文→抄写3个核心词汇→造句1句（跳一跳够得着）
- **时间置换**：明日午休12:40-13:00，在讲台旁完成，完成后可自由阅读（普雷马克原理）
- **强制留堂**：（如午休未完成）放学后15:30-16:30，教师陪同批改，清账即放行，不加量
- **话术脚本**："老师知道作文很难，今天特批你只需抄写黑板上的3个词。不是10个，就3个。明天午休你带着本子来讲台旁，做完就去操场。老师陪你，不批评，只清账。"
- **24小时微行动**：明天完成3个词汇抄写并上交，教师在放学前给予具体反馈（如"这个词书写规范"）
- **后续评估**：3天后评估是否能维持5词抄写，1周后尝试升级至造句

## Common Issues

- **跳过降级**：未实施任务降级直接强制留堂，导致学生因认知超载而情绪崩溃
- **话术威慑**：使用"必须""否则告诉家长"等语言，激发权力试探学生的对抗
- **组件缺失**：仅强调价值边界与强制规则，缺乏发展支持类组件（认知支架/微小行动），导致习得性无助加剧
- **时效缺失**：给出"一个月提升计划"等宏观方案，未提供24小时内可落地的具体微行动
- **忽视红线**：在家长激烈对抗或学生自伤风险时仍坚持强制留堂，违反安全协议
