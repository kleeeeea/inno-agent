# 课堂高效诊断与数据采集

## Instructions
- **数据优先原则**：所有诊断必须基于秒级可观测数据（抬头率%、动笔率%、答题正确率%），彻底摒弃"气氛活跃"、"听讲认真"等主观玄学描述。
- **轻量化部署**：1分钟扫描法观察项≤5个，全部通过打勾或填数字完成，禁止听课时记录大段文字。
- **即时可见性**：利用信息化工具（答题器、平板语音转文字）在3分钟内生成"谁没参与"的客观证据，杜绝事后凭印象补录。

## Workflow
1. **部署1分钟扫描法**：在讲授环节观测抬头率（眼睛看向教师/屏幕的比例），在活动环节观测动笔/思考率（手持笔操作或键盘录入的比例），使用《课堂学生大脑运转度极简观察表》记录。
2. **配置信息化采集**：根据设备条件选择ABCD答题器（统计全员正确率与缺席率）或平板语音转文字（生成词云展示观点分布），设计轻量化SOP确保3分钟内完成数据采集。
3. **生成数据看板**：将扫描数据与信息化统计结果整合为秒级可见的客观看板，标记假性参与区域（如仅2人发言占比<10%、C/D角色搭便车）。

## Output Format
- **极简观察表**：≤5个核心观察项，全为勾选框或数字填空栏，零开放式文字描述。
- **量化指标清单**：抬头率%（讲授时）、动笔率%（活动时）、答题正确率%、缺席率%、互动覆盖率%（发言人数/总人数）。
- **数据看板**：包含时间戳的行为数据截图，明确标注"假性参与"热点区域（眼神空洞、未动笔、未答题）。

## Examples
- **抬头率采集**：讲授到第15分钟时，快速扫视全班，记录抬头看向黑板的学生数：32人/40人=80%，并在观察表勾选"抬头率>70%"；同时记录持续低头学生座位号。
- **动笔率采集**：小组讨论环节，巡视记录手持笔在任务单书写或操作平板输入的学生比例：12人/40人=30%，判定为"假性参与高风险"，立即触发干预机制。
- **答题器部署**：使用ABCD答题器推送概念检测题，10秒后统计：正确率35%，缺席率25%（10人未选）。数据自动同步至看板，标记为"理解断层区域"。

## Common Issues
- **主观描述回潮**：避免记录"学生精神状态良好"等模糊评价，必须转化为"抬头率85%""动笔率60%"等具体数值。
- **工具过度复杂**：拒绝需要10分钟以上部署的复杂系统，坚持"3分钟呈现结果"的轻量化原则，无信息化设备时优先使用纸质 tally mark（正字计数法）。
- **数据滞后采集**：禁止课后凭记忆补填数据，所有观测必须在课堂进行同时完成，确保客观性。
