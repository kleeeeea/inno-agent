# rubric.md — 中职教研机制优化顾问评分标准

## 评分维度总览

本评分标准用于评估"中职教研机制优化顾问"的输出质量，共包含 **5 个维度**，每个维度满分 **4 分**，总分 **20 分**。

---

## 维度一：问题诊断的准确性与因果链完整性

> 评估顾问是否能基于用户提供的具体事实，准确识别问题类型并构建有逻辑支撑的因果链，而非泛泛归因。

| 分值 | 评分标准 |
|------|----------|
| **4分** | 准确识别全部相关问题类型（结构性挤占 / 机制性失灵 / 机制副作用），每类问题均锚定用户提供的具体事实（如被占次数、退化路径），因果链清晰完整，各类问题独立分析，无混淆 |
| **3分** | 识别主要问题类型，因果链基本完整，但存在个别事实未被引用，或两类问题分析略有交叉混淆 |
| **2分** | 能识别部分问题类型，但因果链不完整或存在简单归因（如"教师不重视"），对用户提供事实的引用较少 |
| **1分** | 问题识别模糊，缺乏因果链分析，或将多类问题笼统归为同一原因，诊断结论缺乏依据 |

---

## 维度二：机制方案的可治理性与制度可操作性

> 评估输出的机制方案是否具备明确的责任主体、审批流程、规则边界，能够在学校管理层面真正落地执行。

| 分值 | 评分标准 |
|------|----------|
| **4分** | 五个模块均有具体制度设计，明确规定责任主体（谁执行、谁审批、谁公示）、操作规则（如会议占用审批流程、补研替代路径）和执行边界；班主任双线并行等现实阻力有针对性缓解策略 |
| **3分** | 大部分模块有制度设计，责任主体基本明确，但个别模块操作规则不够具体，或对关键阻力的缓解策略较为笼统 |
| **2分** | 方案以建议性语言为主，缺乏明确的责任主体或审批流程，制度约束力弱，难以在学校层面直接推行 |
| **1分** | 输出以口号式或原则性表述为主，无具体制度设计，无法指导实际治理行动 |

---

## 维度三：产出闭环设计的完整性与可追踪性

> 评估方案是否建立了"产出—反馈—复盘"的完整闭环，包括最低产出标准、证据类型定义和跨次追踪机制。

| 分值 | 评分标准 |
|------|----------|
| **4分** | 明确定义每次教研的最低可交付物类型（课例、任务单、学情要点等），清晰界定证据类型（课堂片段、作业数据、问卷摘要等），复盘机制与上次任务证据直接挂钩，有效做法有沉淀路径，无效原因进入再设计流程 |
| **3分** | 产出标准和证据类型基本明确，复盘机制存在但与上次任务的关联不够紧密，沉淀机制有提及但细节不足 |
| **2分** | 有产出要求但标准模糊，证据类型未明确界定，复盘机制流于形式或仅为"下次汇报"，缺乏真正的闭环设计 |
| **1分** | 无明确产出标准，无证据要求，无复盘机制，教研活动结果无法追踪 |

---

## 维度四：共性与专业差异化的双层结构适配

> 评估方案是否兼顾了跨专业教研的共性底座与专业适配层，避免"共性压过个性"的机制副作用。

| 分值 | 评分标准 |
|------|----------|
| **4分** | 明确区分共性底座（统一目标/核心任务/评价标准）与专业适配层（情境、案例、任务设计），提供"专业需求—知识点—课堂任务—评价证据"映射模板或等效工具，跨专业参与机制（如专业负责人共研、需求清单）有具体设计 |
| **3分** | 双层结构有所体现，共性与差异化的边界基本清晰，但映射模板或跨专业参与机制细节不足，实操指导性有限 |
| **2分** | 方案以共性设计为主，专业差异化仅作原则性提及，无具体适配工具或跨专业协同机制 |
| **1分** | 未区分共性与专业差异化，方案对所有专业一刀切，存在明显的机制副作用风险 |

---

## 维度五：成效指标的可量化性与系统整合度

> 评估方案提供的成效指标是否可量化、可观测，并与校历、教务、绩效评优等现有系统有效对接，避免成为额外负担。

| 分值 | 评分标准 |
|------|----------|
| **4分** | 过程指标（固定时段保留率、专业议题占比、任务完成率等）与结果指标（课例质量、学生作品达标率等）均有明确定义，指标与校历/教务/绩效评优机制有具体对接方式，数据采集路径清晰，不额外增加教师负担 |
| **3分** | 过程指标与结果指标基本完整，与现有系统的对接有所说明，但部分指标定义不够精确或数据采集方式不明确 |
| **2分** | 指标以定性描述为主，量化标准模糊，与现有系统的整合方式未说明，存在成为额外负担的风险 |
| **1分** | 无可量化指标，或仅列举笼统目标（如"提升教研质量"），无法支撑实际监测与评估 |
