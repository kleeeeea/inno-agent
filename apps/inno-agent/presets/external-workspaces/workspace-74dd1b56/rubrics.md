# 小学科学过程性评价设计师 评分维度说明（rubric.md）

## 维度一：评价标准的可操作性

> 考察输出的评价条目是否真正做到"可观察、可判断、可打勾"，教师无需额外解读即可直接使用。

| 分值 | 描述 |
|------|------|
| 4 | 所有条目均为具体可观察行为（如"测量时重复操作了3次"），3—5条，无抽象词汇，教师可直接打勾判断 |
| 3 | 大多数条目可观察，个别条目略显抽象，稍作调整即可使用 |
| 2 | 条目数量或表述存在明显问题（过多、过于笼统、含"积极参与"等无法判断的词语），需较大修改 |
| 1 | 条目以抽象目标或能力描述为主，教师无法在课堂中直接判断，实用性极低 |

---

## 维度二：全流程覆盖与环节匹配度

> 考察方案是否根据教师实际需求，合理选取并覆盖探究全程的关键评价环节（标准共定→自评→互评→反馈→改进→展示），而非仅聚焦结果评价。

| 分值 | 描述 |
|------|------|
| 4 | 根据课型和需求精准选取相关环节，评价嵌入探究每一关键节点，无明显缺漏，过程与结果评价比例合理 |
| 3 | 覆盖主要环节，但个别节点（如"实时改进"或"展示激励"）处理较简略，整体仍以过程为主 |
| 2 | 环节选取与教师需求匹配度一般，部分环节流于形式或仅停留在结果评价层面 |
| 1 | 方案以终结性评价或单一打分为主，缺乏过程性评价的核心设计 |

---

## 维度三：无技术条件下的落地可行性

> 考察方案是否严格遵循"无数字化平台、无专业工具、无额外经费"的约束，所有工具和流程能否在普通课堂（含乡村学校）中立即实施。

| 分值 | 描述 |
|------|------|
| 4 | 所有工具仅限纸质记录单、黑板、口头话术等，表格结构清晰可直接抄写或打印，无任何设备依赖，乡村教师可零成本实施 |
| 3 | 绝大多数环节无需设备，个别建议（如"拍照记录"）可轻松替换为纸质方案，整体可行 |
| 2 | 部分环节隐含对设备、平台或额外材料的依赖，需教师自行调整才能落地 |
| 1 | 方案明显依赖数字工具、投票软件或专项经费，在资源薄弱学校无法实施 |

---

## 维度四：差异化支持与面向全体学生

> 考察方案是否为薄弱生、内向生等特殊群体提供具体可行的参与路径，确保无边缘学生，同时具象反馈话术是否替代了空泛表扬。

| 分值 | 描述 |
|------|------|
| 4 | 明确提供针对薄弱生/内向生的分层策略（如"最小成功单元"、降低门槛的具体做法），具象反馈话术格式规范（复述→描述行为→追问），可直接说出口 |
| 3 | 有差异化建议，但部分策略较笼统；反馈话术基本具象，偶有"你做得不错"等模糊表达 |
| 2 | 差异化内容较少，仅泛泛提及"关注薄弱生"；反馈话术以空泛表扬为主，缺乏具体行为描述 |
| 1 | 方案未考虑学生差异，无差异化策略；反馈停留在"你真棒""很好"等无效表扬层面 |

---

## 维度五：教师负担与方案简洁度

> 考察方案是否在设计上主动控制教师工作量，评价流程能否在课堂内完成，输出内容是否简洁、结构清晰、易于理解和执行。

| 分值 | 描述 |
|------|------|
| 4 | 方案结构清晰、模块分明，评价流程可在课堂内完成，无需课后大量批改；输出内容精炼，教师阅读后可立即行动 |
| 3 | 整体简洁，但个别环节（如记录单设计）略显繁琐，教师需少量课后整理 |
| 2 | 方案内容较多，教师需花费较多时间理解或准备，课堂内难以完整实施 |
| 1 | 方案复杂冗长，记录项目过多，实际执行会显著增加教师负担，可操作性差 |
