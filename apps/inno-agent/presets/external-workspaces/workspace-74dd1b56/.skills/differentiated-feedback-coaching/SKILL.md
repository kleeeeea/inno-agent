# 差异化反馈与学生引导

## Instructions

- 先用1—2句诊断问题根源（如：标准不具体导致自评失效；互评缺乏聚焦点导致走过场）
- 所有话术必须可直接说出口，格式固定为：**复述观点 → 描述具体行为 → 提出一个追问**
- 禁止出现"你真棒""做得好""很积极"等无法指向改进的空泛表扬
- 针对薄弱生/内向生，必须设计"最小成功单元"——找到一个他们一定能完成的最小任务，建立成就感
- 所有策略只用纸质记录单、口头话术、黑板板书实施，不依赖任何设备

## Workflow

1. **识别问题类型**：判断是自评失效、互评走过场、薄弱生边缘化，还是教师反馈空泛（可同时存在多种）
2. **诊断根源**：一句话说明为什么会出现该问题（如"自评打满分=标准太抽象，学生无法判断"）
3. **输出具象话术**：提供3—5条可直接使用的反馈示例，覆盖不同场景
4. **给出分层策略**：针对薄弱生/内向生，提供降低门槛、设置最小成功单元的具体做法
5. **提供互评改进方法**：给出让互评聚焦行为而非能力的操作步骤

## Output Format

**【问题诊断】**
一句话说明根源。

**【具象反馈话术】**
- 不说："[空泛表扬]"
  要说："[复述观点]+[描述行为]+[追问]"
（列3—5条，覆盖不同情境）

**【自评/互评改进策略】**
- 针对自评打满分：……
- 针对互评走过场：……
（每条给出具体操作步骤，不超过3步）

**【薄弱生/内向生专项支持】**
- 最小成功单元设计：……
- 角色安排建议：……
- 降低门槛的具体做法：……

## Examples

**场景：学生自评全打满分**
- 问题诊断：评价标准过于抽象（如"认真参与"），学生无法判断自己是否达标，默认全打勾。
- 改进策略：将标准改为可数、可看、可验证的行为，如"□ 测量重复了3次""□ 记录单每格都填写了数据"；自评前让学生先找出一处"还可以更好的地方"再打勾。
- 话术示例：
  - 不说："你自评很认真。"
    要说："你给自己的记录打了满分，我看到第3格数据还是空的，你觉得这里达标了吗？"

**场景：互评变成互夸**
- 问题诊断：互评没有聚焦点，学生不知道该看什么，只好说"你真棒"。
- 改进策略：互评只评三件事——①他参与了吗？②他完成了自己的分工吗？③记录单填完整了吗？用打勾代替打分，结果用于改进不用于排名。

**场景：薄弱生每次评价垫底**
- 最小成功单元：为该生单独设定本节课只需完成"□ 完整填写了实验步骤第1步"即为达标，公开肯定这一进步。
- 话术示例：
  - 不说："你比上次好一点点。"
    要说："你今天把第一步的观察结果写下来了，上次这里是空的。你是怎么想到要写的？"

## Common Issues

- **话术生硬不自然**：先复述学生说的话或做的动作，再追问，语气保持好奇而非评判，避免追问变成质问。
- **薄弱生最小成功单元设置过高**：门槛必须是"该生今天一定能做到"的事，而非"努力一下能做到"，否则仍会失败。
- **互评结果被用来打分排名**：互评结论只写在小组记录单上，教师收集后用于了解学情，不公开比较、不计入成绩。
- **教师担心具象反馈太耗时**：每节课只需对2—3名学生使用完整话术，其余学生用打勾确认即可，不必每人都做长反馈。
