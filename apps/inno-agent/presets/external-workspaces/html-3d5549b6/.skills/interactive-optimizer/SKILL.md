# 交互优化与交付指导

## When to use
- 教师要求调整字体大小、配色方案、背景主题或按钮位置（如"字太小了后排看不见"、"换成护眼色"）
- 需要增加/修改动画效果、音效反馈、视觉特效（如"要有撒花效果"、"点击后没声音"）
- 询问技术操作问题：如何保存为.html文件、如何通过U盘传输、如何在教室一体机打开运行
- 适配特定硬件需求：大屏触摸优化（按钮≥48px）、低视力保护色、投影仪对比度调整
- 修复交互体验问题：触摸响应延迟、动画卡顿、布局错位

## When not to use
- **首次创建新工具**：教师描述全新教学需求（如"想要一个随机分组工具"）→ 转交 `html-tool-generator` 处理
- **功能逻辑变更**：需要添加复杂数据存储、多人在线同步、后端数据库 → 拒绝并解释单文件HTML限制
- **隐私安全风险**：要求预置学生照片、身份证号等敏感信息 → 拒绝并提示数据保护原则
- **纯教学咨询**：仅需教学方法建议无需技术实现 → 直接对话回答

## Instructions
- **保持单文件约束**：所有优化必须集成在单个HTML文件内，不引入需配置的构建工具
- **大屏优先原则**：确保触摸目标≥48px，关键文字≥24px，高对比度配色适配投影仪
- **零门槛可调整**：使用CSS变量定义颜色/字号，便于教师后续自行修改
- **离线可用**：不依赖外部API，仅使用CDN引入的公开库（如Tailwind、Vue3）
- **中文注释**：在代码关键修改处添加中文说明，方便教师理解

## Workflow
1. **需求确认**：明确优化维度（视觉/交互/部署），复述理解避免偏差
2. **代码定位**：分析现有HTML结构，定位CSS样式或JS逻辑需调整部分
3. **迭代生成**：输出优化后的完整代码块（非补丁形式，确保直接可用）
4. **部署指导**：提供"复制→保存→传输→打开"四步傻瓜式教程
5. **预防性提示**：提醒常见操作误区（如勿用IE浏览器、需解除文件锁定）

## Output Format
每次响应必须包含以下结构化字段：

**【修改摘要】**
- 本次调整的具体项（如：字体32px→48px、背景色#fff→#C7EDCC）
- 适配场景说明（如：针对后排 visibility、低视力保护）

**【优化代码】**
- 完整单文件HTML代码块（可直接复制保存使用）
- 关键修改处带中文注释

**【部署指南】**
- 保存步骤：如何复制粘贴到记事本→另存为→编码选UTF-8→.html扩展名
- 传输方法：U盘拷贝、微信文件传输助手、邮件附件
- 打开方式：必须使用Chrome/Edge浏览器（强调勿双击、勿用IE）
- 全屏操作：F11全屏、ESC退出、触摸缩放设置

**【自定义说明】**
- 颜色修改：定位CSS变量 `--bg-color` 位置
- 字体调整：修改 `body { font-size: XXpx }`
- 名单更新：查找 `const students = [...]` 数组位置

**【风险提示】**
- 浏览器兼容性警告（避免Safari旧版本）
- 数据未持久化提醒（刷新页面名单重置）

## Examples

**示例1：视觉优化请求**
输入："字太小了，学生后排看不见，改成护眼色，按钮要大一点"
输出：
- 修改摘要：正文24px→36px、标题倍增、背景#C7EDCC（豆沙绿）、按钮padding 20px、触摸区域60px
- 提供含CSS媒体查询的完整代码，支持自动适配屏幕尺寸

**示例2：部署指导请求**
输入："我已经复制了代码，接下来怎么在教室电脑上打开？"
输出：
- 详细步骤：1. 桌面右键新建文本文档 2. 粘贴代码 3. 另存为→文件名`tool.html`→编码UTF-8 4. 发送到U盘 5. 教室电脑插入U盘→右键文件→打开方式→Microsoft Edge 6. 按F11全屏

**示例3：动画效果添加**
输入："点名的时候要有抽奖机的滚动效果，最后停下时有个 ding 的提示音"
输出：
- 修改摘要：添加CSS@keyframes滚动动画、JavaScript控制停止逻辑、嵌入base64编码的提示音或Web Audio API生成音效
- 代码包含动画速度参数注释，便于教师调整快慢

## Common Issues
- **文件打不开**：教师双击HTML文件被记事本或WPS错误打开 → 指导"右键→打开方式→选择Chrome/Edge"
- **中文乱码**：保存时未选择UTF-8编码导致姓名显示乱码 → 强调"另存为"对话框中编码选项
- **触摸不灵敏**：按钮太小或缺少touch事件 → 确保padding≥12px，添加`touch-action: manipulation`
- **音效无法播放**：浏览器自动播放策略限制 → 添加"点击任意处开启声音"的引导层
- **名单丢失**：刷新页面后学生名单重置 → 说明单文件无存储特性，提供Excel快速生成数组的公式方法
- **投影显示不清**：背景与文字对比度不足 → 提供高对比度模式切换按钮（黑底黄字/白底黑字）
