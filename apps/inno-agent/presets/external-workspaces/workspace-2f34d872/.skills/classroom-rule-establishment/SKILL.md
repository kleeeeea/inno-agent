# 课堂规则建立与开学控场

## When to use
- **开学第一课**：需要宣告课堂纪律底线、建立教师权威、制定合规后果清单
- **课始冷启动**：课前3分钟学生喧闹，"别说话了"反复无效，需要即时控场脚本
- **规则重置**：前期规则执行松散导致班级纪律滑坡，需要重新立规矩并进入"严格执行期"
- **后果设计**：需要替代体罚/羞辱的合规教育惩戒方案（课堂内/课后任务、座位调整等）

## When not to use
- **课中实时纪律问题**（学生讲话/插话/顶嘴的即时处置）→ 转交 **discipline-intervention-strategies**
- **需要分层干预话术**（Level 1-4升级策略、压迫感话术强化）→ 转交 **discipline-intervention-strategies**
- **严重安全事件/心理危机**（校园暴力、自伤倾向、性骚扰等）→ 强制转介学校德育处/心理中心，停止策略建议

## Instructions
1. **规则精简原则**：仅设3条底线（如"未经允许不讲话""发言先举手""课前备齐物品"），配套3条可立即执行的合规后果，避免规则过繁
2. **气场建立技术**：强调"沉默凝视+慢语速+眼神扫射"而非提高音量；课始控场必须包含30-60秒面无表情的沉默等待
3. **前两周严格执行期**：规则宣告后，前两周必须"说到做到"，每次违规立即按预设后果处理，形成稳定预期
4. **合规后果设计**：依据《中小学教育惩戒规则（试行）》，提供课堂内（暂离座位、课后谈话）与课后（增值任务、家校沟通）选项，严禁体罚/羞辱
5. **重置话术模板**：当规则已失效时，使用"之前老师执行不严，今天重新明确"的坦诚宣告+前两周强化执行路线图

## Workflow
1. **场景诊断**：确认学段（小学高年级/初中/高中）、班级规模、当前时间节点（开学首课/前两周中/规则失效后）
2. **规则生成**：输出"三条底线+三条奖励+三条后果"清单，确保后果具体、即时、可落实
3. **冷启动脚本**：提供"进门→上讲台→放书→站直→沉默→开口"的微观动作指导（含时长、站姿、眼神、语速）
4. **合规审查**：自检后果清单，确保无体罚、无羞辱、无超时限罚站，标注《中小学教育惩戒规则（试行）》依据
5. **执行路线图**：制定前两周"零妥协"执行计划，包括高频违规点记录表与每周复盘节点

## Output Format

**场景判断**：学段/班型/当前纪律阶段

**规则清单**：
- 三条底线（明确可观察行为）
- 三条奖励（即时正向反馈）
- 三条后果（合规、即时、匹配违规程度）

**冷启动脚本**：
- 动作清单（站位/手势/眼神/沉默时长）
- 压迫感话术（含正负案例对比）
- 禁忌提醒（避免反复劝告/音量升级）

**合规标注**：后果对应《中小学教育惩戒规则（试行）》条款，明确体罚/变相体罚红线

**前两周执行要点**：何时处理/如何记录/何时微调

**重置方案**（如适用）：承认松懈→重新宣告→严格执行期承诺

## Examples

**正面案例（专家做法）**：
- 课始控场：入室→上讲台放书→手背后站直→面无表情扫视全班45秒→慢语速："说完了吧？说完了没有？"→停顿→进入正课
- 规则宣告："这三条底线，做到就奖，做不到就罚，没有第三次机会。前两周我会很严格，之后你们会感谢这个习惯。"

**反面案例（常见误区）**：
- 反复劝告式："别说话了""安静点""不要讲了"循环多次，学生麻木
- 只立不执：第一课宣告了后果，第二课却不处理，学生迅速试探成功
- 规则过繁：列出10条纪律要求，学生记不住，教师无法坚持检查

## Common Issues
- **规则过多**：超过5条则难以执行，精简至3条核心底线
- **后果模糊**：避免"再这样就严肃处理"等虚词，改为"第一次提醒→第二次站后听课5分钟→第三次课后谈话+增值任务"
- **前两周松懈**：规则建立的关键窗口是开学前两周，此期间"宁严勿松"，避免学生形成"规则可试探"预期
- **重置失败**：重新立规矩时未坦诚承认前期执行问题，也未承诺后续严格，导致学生不信服
