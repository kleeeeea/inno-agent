# 课堂分层干预与话术处置

## When to use

- 讲课过程中学生持续讲话、插话、传纸条等一般性违纪，需要即时处置避免全班失焦
- 眼神提示无效，需判断升级到"靠近/停讲凝视/话术强化"哪一层级
- 遭遇顶嘴、接话等对抗行为，需要"给台阶但形成压力"的合规话术
- 需要立即压制纪律问题并快速回归教学流程，避免课堂节奏崩塌

## When not to use

- **开学第一课规则宣告、课始3分钟冷启动控场、合规后果清单制定** → 转 **classroom-rule-establishment**
- **严重安全事件、校园暴力、学生心理危机（自伤/自杀倾向）、性骚扰** → 立即停止建议，强制提醒转介学校德育处/心理中心
- **家校正式处分沟通、学校德育处分流程替代** → 转普通问答（本功能仅限课堂即时管理技术）
- **课后深度复盘、长期行为矫正计划制定** → 转普通问答（本功能仅提供即时干预与简要记录建议）

## Instructions

1. **四级升级逻辑**：严格遵循Level 1（眼神提示）→Level 2（靠近压制）→Level 3（停讲凝视）→Level 4（话术强化）的递进，明确标注升级触发条件（如"看一眼5秒无效即升级"）。
2. **气场控制**：强调"沉默、眼神扫射、慢语速、低音量"的压迫感建立，禁止建议"提高音量吼叫"或"反复重复劝告"。
3. **话术设计**：提供"质问-给台阶-留白"结构（如"说完了吧？上讲台来说"），确保教师不失控、学生不被羞辱，且能立即回归教学。
4. **合规边界**：所有后果建议必须符合《中小学教育惩戒规则（试行）》，严禁体罚、变相体罚、羞辱性语言或公开隐私。

## Workflow

1. **场景定级**：判断违纪行为等级（一般讲话/插话/顶嘴对抗）与当前所处层级（已尝试眼神/靠近是否无效）。
2. **策略匹配**：根据层级选择对应干预动作（靠近继续讲/停讲凝视/话术强化）。
3. **脚本生成**：输出具体动作（站位、眼神、沉默时长）与可背诵话术（含语气标注）。
4. **执行与回归**：干预后必须包含"立即回归教学"的过渡语句，避免长时间拉扯。
5. **风险提示**：自检输出，确保无体罚、无羞辱、无心理伤害风险。

## Output Format

```markdown
**场景判断**：{违纪类型} | 当前层级：Level {1-4}

**处理方案**：
- 动作清单：{具体站位、眼神、沉默时长、语速要求}
- 升级条件：{何时进入下一层级的明确标准}

**可用话术**：
- 压迫感开场："{话术}"（{语气：慢/低/停顿}）
- 给台阶收尾："{话术}"（{目的：保全面子/立即止损}）

**风险提示**：{合规边界提醒，如"禁止罚站超一节"等}

**回归教学指令**：{干预后如何无缝切回教学内容的过渡语}
```

## Examples

**正面案例（专家做法）**：
- 学生A讲话不停：教师边讲边走到其桌旁（Level 2）→仍不改，停讲凝视3秒（Level 3）→"上讲台来，老师陪你说。给你时间、给你空间，这节课你说个够。"（Level 4）→停顿2秒→"好，回去。我们继续看第三段。"（立即回归）

**反面案例（常见误区）**：
- 反复劝告式：连续说"别说话了""安静点"超过3次，学生麻木，教师权威损耗。
- 过度拉扯式：与顶嘴学生当众争辩"你为什么讲话""你什么态度"超过30秒，导致全班注意力崩塌。

## Common Issues

1. **升级过快**：未尝试Level 1-2直接怒吼或点名批评，易引发对抗。
2. **升级过慢**：Level 1眼神无效后仍继续讲课不靠近，或Level 3凝视后不敢开口强化，导致纪律标准模糊。
3. **过度拉扯**：使用Level 4话术后与学生陷入"来啊/不来/敢不敢"的长时间对峙，忘记立即回归教学。
4. **话术羞辱**：使用"你脸皮真厚""全班就你最吵"等侮辱性语言，违反合规边界。
