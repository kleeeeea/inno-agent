# 心理安全重建与正向激励设计

## When to use
- 学生陷入"跟不上→畏惧→走神→拖延→被批评→自信丧失"恶性循环，需阻断心理防御机制时
- 需建立课堂心理安全规则（如"答错不许嘲笑"底线、防嘲笑机制）或重组学习小组归属时
- 需设计即时正向激励（每日鼓励便签、每周"赏花时刻"表扬、六维雷达图多元评价）替代分数排名时
- 需通过"浇花日记"建立家校正向反馈闭环，阻断家庭负向施压时
- 需为自卑沉默学生设计"低风险、高成就"微型角色任务（如计时员、传话筒）时

## When not to use
- 需进行学业能力诊断、前测分析或设计"跳一跳够得着"的分层教学目标卡时 → **转交 diagnosis-and-layered-support**
- 涉及拼音薄弱、识字量不足、口算分层等具体学科补救教学策略时 → **转交 diagnosis-and-layered-support**
- 疑似ADHD、阅读障碍、自闭症谱系等神经发育问题 → **建议转介学校心理中心或医疗机构评估**
- 涉及临床抑郁、焦虑障碍等重度心理健康障碍 → **建议转介专业心理干预**
- 针对小学4-6年级或中学学段的学业问题 → **超出本包服务范围**

## Instructions
- 坚持"生态柔性孕育"理念，严禁建议补课、加作业、频繁请家长、留堂惩罚等粗放施压手段
- 所有干预必须明确执行频率（如"每日放置便签""每周一次赏花时刻""每月雷达图评估"）
- 使用"松土、守护、花开"等生态隐喻，避免军事化、工业化管理术语
- 表扬聚焦过程、努力、策略（"你反复尝试了三种方法"），禁止"你真聪明"等固定型思维话术
- 严格区分公开表扬与私下表扬：涉及隐私、基础漏洞或情感脆弱点必须使用私下鼓励

## Workflow
1. **松土蓄力（安全建构）**：建立"答错不许嘲笑"底线规则，重组4人异质小组赋予具体角色（计时员、材料员、传话筒），构建无错课堂生态
2. **暖心守护（情感陪伴）**：每日放置定制鼓励便签（具体描述微行为），捕捉闪光点设立"赏花时刻"小范围表扬，设计"我同意…还想补充…"等低门槛发言支架
3. **多元评价（雷达跟进）**：实施六维雷达图（课堂参与、作业、交往、习惯、情绪、自我管理）月度评估，替代单一分数评价，每维度记录具体行为证据
4. **家校协同（浇花日记）**：启动双向传递日记本，教师与家长各写一句具体正面观察（如"今天他看了作业本一眼"），每周往返2次，禁止负面评价
5. **静待花开（成果展示）**：学期末举办"我也在开花"成长展示会，呈现个人过程性证据（如便签集、雷达图变化），禁止横向比较

## Output Format
- **场景判断**：当前学生处于恶性循环的哪个环节（畏惧/防御/麻木/习得性无助）
- **心理安全方案**：具体班级规则文本、小组重组方案、防嘲笑机制、角色任务设计
- **激励工具包**：鼓励便签模板、赏花时刻话术脚本（公开/私下区分原则）、六维雷达图量表、浇花日记样表
- **执行频率**：明确每日/每周/每月的具体操作频次与单次时长
- **风险提示**：识别需转介专业心理支持的信号（如长期躯体化症状、自伤倾向、持续两周以上的情绪低落）
- **效果评估**：2周后心理安全感指数变化（课堂举手频次、眼神接触时长、主动发问次数）

## Examples
**例1（自信重建）**：为自卑沉默学生设计"图书角管理员"低风险角色，每日私下放置"我注意到你整理了歪斜的书架"便签，两周后举办微型"赏花时刻"仅邀请2名同伴见证其贡献，使用话术"因为小宇的细心，大家找书更方便了"。

**例2（无错课堂）**：制定"提问勋章"规则，每提出一个问题即得章无论对错；设置"观点传声筒"角色让不敢发声学生复述同伴观点，使用支架"我同意小明的想法，因为…"。

**例3（家校协同）**：针对长期不交作业学生，启动浇花日记，教师写"今天他主动翻开了作业本"，家长写"回家路上他说了学校趣事"，每周二、五传递，明确禁止提及"作业未完成"等负面评价。

## Common Issues
- **混淆"降低要求"与"心理安全"**：教师误将无错课堂理解为不纠正错误 → 明确区分"容错氛围"（不嘲笑）与"学习反馈"（温和纠正错误），建立"安全但挑战"的边界
- **浇花日记变批评清单**：家长记录负面行为 → 提供家长话术模板，限定格式为"我观察到+具体正面行为+我的感受"，禁止负面评价
- **公开表扬引发焦虑**：受表扬者感到尴尬或同伴嫉妒 → 严格遵循"微小进步+小范围见证（3-5人）"原则，避免夸大其词，强调"比昨天的自己进步"
- **六维雷达图操作复杂**：教师难以坚持每月六维度全记录 → 简化为每周聚焦一个维度记录，每月汇总，使用勾选或贴纸替代文字描述
