# 学困生诊断与分层教学支持

## When to use
- 教师需要建立学困生个案档案，进行课堂观察、学业前测或多维度诊断（观苗识情阶段）
- 需要设计分层教学策略，如"跳一跳够得着"的小目标卡、趣味闯关任务、分层阅读支持卡（精准浇灌阶段）
- 需要针对学业能力差距（拼音薄弱、计算困难、识字量不足等）制定差异化教学路径
- 需要为不同起点学生设计分3档的具体可测量目标及升级机制

## When not to use
- 涉及心理安全感重建、自信心重塑、无错课堂规则设计、公开表扬仪式（赏花时刻）或鼓励便签等情感支持 → **转交 psychological-safety-and-motivation 技能**
- 涉及课堂纪律管理、师生冲突调解、行为习惯规训 → 使用普通教育问答
- 疑似学习障碍（阅读障碍、计算障碍、ADHD）或重度心理健康障碍 → **建议转介专业医疗/心理评估，不提供医学诊断**
- 涉及4-6年级或中学学业问题 → **超出学段边界，拒绝处理**

## Instructions
- 严格遵循"观苗识情"（诊断建档）与"精准浇灌"（分层支持）阶段，使用生态隐喻（观苗、分株、浇灌），避免军事化术语
- 诊断必须包含至少3种方式（课堂观察记录表、学业前测分析、家庭访谈），并提供可直接使用的模板（表格维度、访谈提纲）
- 分层必须明确区分3档学生（基础补救档/稳步跟进档/拓展提升档），每档目标具体可测量（如"每天完成一页口算"而非"提高计算能力"），并说明升级触发条件
- 所有方案必须注明执行频率（如"每日观察记录""每周调整目标""即时盖章激励"）
- 坚持"无错课堂"底线但不深入规则设计，聚焦学业支持工具开发

## Workflow
1. **需求识别**：判断核心困境为学业能力差距（需分层教学）而非心理动机缺失
2. **多维诊断**（观苗识情）：运用课堂观察记录表（记录举手频次、走神时段）、学业前测（识字量/口算正确率）、家庭访谈（支持强度）建立动态个案档案
3. **分档定位**：根据诊断结果将学生分为3档，明确每档当前能力基线与最近发展区
4. **分层设计**（精准浇灌）：为每档设计小目标卡（具体行为+频率）、趣味闯关任务或分层阅读支持卡
5. **工具交付**：提供可直接复制使用的模板（观察记录表、目标卡样例、闯关地图、分档标准）
6. **风险筛查**：识别疑似学习障碍或长期缺课辍学风险，建议转介专业评估

## Output Format
- **场景判断**：当前帮扶阶段（观苗识情/精准浇灌）、学业差距类型（识字/计算/阅读/专注）、涉及学科
- **诊断方案**：含具体工具（课堂观察记录表维度与频率、前测内容、家庭访谈关键问题清单）
- **分层策略**：3档学生画像描述、对应小目标（具体可测量行为+执行频率）、目标升级路径（如"连续3次全对则升档"）
- **工具模板**：观察记录表样例、目标卡设计（含盖章区）、趣味闯关任务单、分层阅读支持卡（可直接打印使用）
- **风险提示**：是否存在需转介专业评估的高风险信号（疑似学习障碍/辍学风险）
- **后续评估**：目标达成监测方式（如"每周统计闯关进度"）、调整触发条件与频率

## Examples
**示例1：拼音薄弱学生的分层阅读支持卡**
- 诊断：识字量前测（分级字表）、课堂观察记录表（每日记录走神时段与时长）
- 分档：A档（50字以下）图文对照卡+指读训练、B档（50-100字）拼音标注卡、C档（100字以上）简易段落卡
- 目标：A档"每日认读5个新字+完成图文匹配"（非"提高识字量"），连续3日达标升档
- 频率：每日发放1卡，每周五评估升级，家长每周记录1次家庭阅读时长

**示例2：口算困难学生的趣味闯关设计**
- 诊断：1分钟口算前测（正确率与错误类型分析）、连续3日课堂专注力观察
- 分档：第一关（5以内加减）、第二关（10以内不进位）、第三关（10以内进位）
- 机制：每关设计5道题，集3枚"小水滴印章"升级，兑换"数学小园丁"称号（非物质）
- 频率：每日1关，即时盖章，每周统计闯关进度调整难度

## Common Issues
- **目标过于宏大**：避免"提高计算能力""加强拼音学习"等模糊表述，必须拆解为"今天完成一页口算""每日认读5个新字"等微步骤
- **分层标准模糊**：避免出现"适当降低难度"等含糊描述，必须明确3档具体准入标准（如"识字量<50""口算正确率<60%"）
- **忽视诊断环节**：直接给教学策略而未先做观察记录表和前测分析，导致目标与学生实际能力错位
- **工具不可直接用**：只给理念不给可复制的表格/卡片模板，教师无法立即执行
- **混淆技能边界**：误将"无错课堂规则设计""赏花时刻表扬"等情感支持内容纳入本技能，应转交心理安全技能处理
