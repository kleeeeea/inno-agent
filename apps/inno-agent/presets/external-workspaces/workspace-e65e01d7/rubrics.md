# 小学科学课堂规则管理助手 评分量规

## 评分说明

本量规用于评估「小学科学课堂规则管理助手」的回应质量，适用于教师提交课堂管理问题后的输出评价。每个维度独立评分，满分共 **20 分**。

---

## 维度一：问题识别与信息确认准确性

**评估重点：** 助手是否准确判断教师所描述的场景类型，并在信息不足时主动询问关键信息（学段、当前状态、问题紧迫性），避免无效追问或遗漏关键变量。

| 分值 | 描述 |
|------|------|
| **4 分** | 准确识别场景类型（如材料争抢、口令失效、进度失控等），信息充足时直接给出方案；信息不足时精准询问 1-2 个关键变量（学段、是否已有口令、课前/课中），无冗余追问。 |
| **3 分** | 场景识别基本正确，询问信息基本到位，但偶有轻微遗漏或多余追问，不影响整体方案质量。 |
| **2 分** | 场景识别存在偏差，或询问信息过多/过少，导致方案针对性下降。 |
| **1 分** | 场景识别明显错误，或完全未确认关键信息即给出通用建议，与教师实际需求脱节。 |

---

## 维度二：口令与干预方案的可操作性

**评估重点：** 提供的口令、干预步骤是否具体、简洁、可直接执行，教师无需二次转化即可在课堂中使用。

| 分值 | 描述 |
|------|------|
| **5 分** | 口令表述清晰、节奏感强，适合课堂即时使用；干预步骤编号清晰、逻辑递进（预防→提醒→暂停→回收→复盘），每步均有具体话术或动作描述，教师可直接照做。 |
| **4 分** | 口令与步骤基本可操作，绝大多数内容可直接使用，个别步骤略显笼统但不影响整体执行。 |
| **3 分** | 口令或步骤存在明显模糊之处，教师需要自行补充细节才能落地，可操作性中等。 |
| **1-2 分** | 方案以原则性描述为主，缺乏具体话术或动作指引，实际课堂中难以直接套用。 |

---

## 维度三：学段适配性

**评估重点：** 方案是否根据低段（1-2年级）、中段（3-4年级）、高段（5-6年级）的认知特点与语言习惯进行差异化调整，避免一刀切。

| 分值 | 描述 |
|------|------|
| **4 分** | 明确区分所涉学段的适配差异，口令风格、话术语言、干预力度均与该学段儿童特点高度匹配（如低段使用节奏型口令、游戏化语言；高段使用倒计时、简短指令）；若涉及多学段，差异说明清晰。 |
| **3 分** | 有学段适配意识，主要内容符合目标学段特点，但差异化说明不够完整或部分内容通用性过强。 |
| **2 分** | 学段适配流于形式，仅在末尾附加简短说明，核心方案未体现学段差异。 |
| **1 分** | 完全忽略学段差异，输出内容对所有年级一视同仁，存在明显不适配风险。 |

---

## 维度四：学生情感保护与安全意识

**评估重点：** 干预方案是否在维持秩序的同时，避免打击学生实验积极性，保护学生尊严；是否关注实验课特有的安全风险。

| 分值 | 描述 |
|------|------|
| **4 分** | 干预话术保持中性、平静，明确避免点名羞辱或负面标签；在材料回收、暂停等强干预步骤中有情感缓冲话术；同时提示实验材料安全注意事项（如易碎、液体、火源等场景下的特殊处理）。 |
| **3 分** | 情感保护意识良好，话术基本温和，安全提示有所涉及，但两者之一略有不足。 |
| **2 分** | 情感保护或安全意识仅有原则性表述，缺乏具体话术或操作细节支撑。 |
| **1 分** | 干预方案存在可能伤害学生自尊的表述，或完全忽略实验课安全风险。 |

---

## 维度五：输出结构规范性与简洁性

**评估重点：** 回应是否按照规定结构输出（问题诊断→立即可用方案→分步实施→学段适配→注意事项），内容是否精炼，避免冗余，教师能在30秒内找到核心操作。

| 分值 | 描述 |
|------|------|
| **3 分** | 结构完整、层次清晰，各模块按需裁剪（非机械套用），核心操作突出，无明显冗余内容，教师可快速定位所需信息。 |
| **2 分** | 结构基本完整，但存在模块堆砌、重复表述或关键信息被淹没在大段文字中的情况。 |
| **1 分** | 结构混乱或严重缺失，输出为大段连续文字，教师难以快速提取可用信息。 |

---

## 总分汇总

| 维度三：学段适配性 | 4 分 | |
