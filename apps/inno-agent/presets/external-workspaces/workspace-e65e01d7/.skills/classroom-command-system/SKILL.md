# 统一口令与规则固化

## Instructions

- 优先与班主任已有静息口令保持一致，避免学生接收冲突指令
- 口令必须简短、有节奏感、可重复，低段用节奏型/游戏化语言，中高段用倒计时型简短指令
- 规则讲解在发材料**之前**完成，先讲规则再给材料，杜绝边发边讲
- 低年级规则必须"演出来"：示范站位、坐姿、递材料、轮流操作，让学生看得见、学得会
- 每节课坚持使用同一套口令，形成肌肉记忆，不随意更换

## Workflow

1. **确认关键信息**：学段（低/中/高段）、是否已有班级口令、问题属于课前预防还是课中强化
2. **匹配口令方案**：从标准口令库中选取适合学段的口令，或结合班主任现有口令调整
3. **输出规则讲解话术**：按学段语言风格给出可直接朗读/使用的话术
4. **提供示范步骤**：给出现场示范的具体动作序列（适用于低年级或规则首次建立）
5. **给出强化建议**：说明如何在后续课堂中持续巩固条件反射

## Output Format

【口令推荐】
- 列出 2-3 条可直接使用的口令，标注适用学段

【规则讲解话术】
- 给出教师可直接说的 2-4 句话，语言符合目标学段认知水平

【现场示范步骤】（低年级或首次建立规则时）
- 编号步骤，每步说明示范动作与配套语言

【强化建议】
- 1-2 条持续巩固口令效果的操作建议

【学段适配提示】（如涉及多学段）
- 低/中/高段差异说明

## Examples

**场景：一年级，学生一拿到材料就乱，尚无统一口令**

【口令推荐】
- "1-2-3-3-4，5-6-7-7-8"（节奏型，低段首选，配拍手）
- "手放膝盖上——眼睛看老师"（动作锚定型，视觉可验证）

【规则讲解话术】
"材料放在桌子中间，不是你的手里。老师说'开始'，你们再动手。听到'停'，手马上放膝盖上，嘴巴关上。"

【现场示范步骤】
1. 教师拿起材料，示范放在桌面中央指定区域，说"材料放这里"
2. 示范双手放膝盖，说"手放好，等口令"
3. 说"开始"，示范拿起材料操作
4. 说"停"，立刻放下材料，双手归位，请学生跟做两遍

【强化建议】
- 前三节课每次发材料前重复示范一遍，表扬第一个做到的小组

---

**场景：五年级，已有"54321暂停"口令但响应率低**

【口令推荐】
- 保留"5、4、3、2、1——暂停"，加入**无声倒计时手势**（手指依次折叠），增强视觉信号

【规则讲解话术】
"听到倒计时开始，不管做到哪一步，立刻停手、停声、看老师。倒计时结束还没停的小组，材料先收回来。"

【强化建议】
- 连续三次口令后全班响应，给予集体正向反馈（如"这次全班3秒内停下，很专业"）

## Common Issues

- **口令太长或太复杂**：超过6个字的口令低年级难以记忆，应精简或改为节奏型
- **口令与班主任不一致**：课后与班主任沟通，统一或明确区分使用场景，避免学生混淆
- **规则讲完就发材料，学生立刻忘**：发材料前让学生复述1条核心规则，再发材料
- **口令练了几次就放弃**：条件反射需要至少10-15次重复，前两周必须每节课坚持使用同一套口令
