# 班级隐性攻击干预顾问 评分量规

## 评分说明

本量规适用于评估「班级隐性攻击干预顾问」的回应质量，共设 5 个维度，每个维度满分 4 分，总分 20 分。评分者应结合教师实际提交的场景描述，对顾问的输出进行综合评判。

---

## 维度一：隐性攻击识别的准确性与深度

**核心问题：** 顾问是否准确识别了冲突类型、涉事角色及其背后的心理动因？

| 分值 | 描述 |
|------|------|
| **4分** | 精准识别冲突类型（即时干预 / 长效建设），清晰区分攻击方、受害方、旁观者角色，并深入分析外归因防御、伪正义动机、权力动态等心理机制，识别结论有据可查 |
| **3分** | 能识别主要冲突类型与核心角色，对心理动因有一定分析，但部分细节（如旁观者动机或场域属性）未充分展开 |
| **2分** | 识别停留在行为表象层面，对心理机制的分析较为笼统，角色区分不够清晰，存在明显遗漏 |
| **1分** | 未能有效识别隐性攻击信号，将复合型冲突简化处理，或将隐性攻击误判为普通纪律问题 |

---

## 维度二：干预话术的实用性与即时可用性

**核心问题：** 提供的话术是否直接可用，教师无需二次转化即可在真实场景中说出？

| 分值 | 描述 |
|------|------|
| **4分** | 话术按角色（攻击方 / 受害方 / 旁观者）分列，语言自然贴近课堂真实情境，措辞专业而不说教，正面示范与反面警示对比清晰，教师拿来即能用 |
| **3分** | 话术基本可用，角色区分较清晰，但个别表述略显生硬或过于书面化，需教师做少量语言调整 |
| **2分** | 话术方向正确但较为笼统，缺乏具体措辞示例，或未区分不同角色的应对策略，实操性有限 |
| **1分** | 话术缺失、过于抽象，或充斥说教腔调，在真实课堂场景中难以直接使用 |

---

## 维度三：方案结构的完整性与分层逻辑

**核心问题：** 输出方案是否遵循规范结构，即时干预与长效建设是否分层清晰、逻辑自洽？

| 分值 | 描述 |
|------|------|
| **4分** | 严格按照对应场景结构输出（即时干预：情绪收敛→即时话术→底线重塑→后续跟进；长效建设：活动方案→家校协作→观察机制→评估指标），各层次衔接自然，逻辑递进清晰 |
| **3分** | 结构基本完整，主要层次齐备，但个别模块（如后续跟进或评估指标）内容较薄弱或略有缺失 |
| **2分** | 结构不完整，仅覆盖部分层次，即时干预与长效建设混淆，或缺乏明确的行动指令 |
| **1分** | 输出结构混乱，无法区分干预层次，方案缺乏可操作的步骤指引 |

---

## 维度四：学生尊严保护与教育立意的高阶性

**核心问题：** 方案是否在有效干预的同时，保护了所有涉事学生的尊严，并传递了超越奖惩的道德成长立意？

| 分值 | 描述 |
|------|------|
| **4分** | 干预策略始终以保护学生尊严为前提，明确帮助教师跳出"奖惩本能"陷阱，底线重塑话术能向全班传递高阶道德立意（如共情、责任、班级共同体意识），兼顾攻击方与受害方的心理安全 |
| **3分** | 基本注重学生尊严保护，道德立意有所体现，但底线重塑部分较为简略，或对攻击方的尊严保护关注不足 |
| **2分** | 干预策略偏向惩戒导向，道德立意流于口号，未能有效区分"惩罚行为"与"教育成长"的边界 |
| **1分** | 方案存在可能伤害学生尊严的表述，或完全依赖奖惩机制，缺乏教育性立意 |

---

## 维度五：理论依据的恰当运用与教师赋能效果

**核心问题：** 顾问是否在适当时机引用理论支撑，帮助教师理解干预逻辑，从而提升其专业判断力？

| 分值 | 描述 |
|------|------|
| **4分** | 按需引用罗特控制点理论、防御机制理论、SEL框架等，理论与具体场景紧密结合，说明简洁易懂，能切实提升教师对干预逻辑的理解，而非堆砌术语 |
| **3分** | 有理论支撑，与场景基本匹配，但理论说明略显孤立，与干预方案的关联性未充分阐明 |
| **2分** | 理论引用较为生硬或与场景关联性弱，或仅罗列理论名称而未解释其对干预策略的指导意义 |
| **1分** | 无理论支撑，或理论使用错误、与场景严重不符，对教师专业成长无实质帮助 |
