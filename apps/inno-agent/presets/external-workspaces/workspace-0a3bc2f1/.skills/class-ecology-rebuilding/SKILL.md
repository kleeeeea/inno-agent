# 班级氛围与长效道德培育

## Instructions

- 本 skill 专注于**长效建设**场景，区别于即时干预；适用于班主任希望预防或系统修复班级生态的情况
- 输出须覆盖四个层次：活动方案 → 家校协作 → 观察机制 → 评估指标，缺一不可
- 活动设计须体验式、可操作，避免说教式讲授；话术示例须贴近真实家校沟通语境
- 始终在保护每位学生尊严的前提下推动班级道德成长，不以惩戒为核心手段
- 可按需附注理论依据（SEL框架、控制点理论等），帮助教师理解干预逻辑

## Workflow

1. **需求定性**：判断教师诉求属于哪个子场景——活动设计、家校合作、长期观察，还是综合生态重建
2. **角色梳理**：明确班级中攻击方、受害方、旁观者的现状分布，评估生态受损程度
3. **分层输出方案**：
   - 【活动方案】设计具体可落地的体验式班级活动，含目标、流程、引导语
   - 【家校协作】提供与家长沟通的策略与直接可用话术
   - 【观察机制】给出长期捕捉教育时机的方法与观察维度
   - 【评估指标】列出班级生态改善的可观察信号
4. **教育立意收尾**：每份方案以高阶道德立意作结，帮助教师传递班级文化价值观

## Output Format

```
【活动方案】
活动名称：……
目标：……
流程（分步骤）：……
关键引导语："……"

【家校协作】
沟通时机：……
话术示例："……"
注意事项：……

【观察机制】
观察维度：言语信号 / 肢体信号 / 关系信号 / 心理防御信号
记录方式：……
教育时机捕捉要点：……

【评估指标】
可观察的改善信号：
- ……
- ……
```

## Examples

**输入示例：** 我想通过班级活动让孩子们学会辨别和应对隐形攻击，比如别人说难听的话时怎么保护自己。

**输出示例（节选）：**

【活动方案】
活动名称：《这句话，伤到我了》情境体验课
目标：帮助学生识别隐性言语攻击，练习边界表达与自我保护
流程：
1. 展示3组"玩笑话"情境卡，学生投票：这是玩笑还是攻击？
2. 小组讨论：被这句话说中，你心里是什么感受？
3. 角色扮演：练习用"我信息"句式回应（"当你这样说，我感到……"）
4. 全班共创"班级语言公约"上墙
关键引导语："玩笑的边界，是对方笑了才算。"

【家校协作】
沟通时机：学期初家长会 + 发现苗头后的个别沟通
话术示例："孩子在班里有时会用'开玩笑'的方式说一些让同学不舒服的话，这不是坏孩子，但需要我们一起帮他建立边界感。您在家里可以这样和他聊……"

**输入示例：** 我该怎么跟家长配合，共同引导孩子建立健康社交观念？

**输出示例（节选）：**

【家校协作】
- 建立"班级社交观察卡"：每月向家长同步一次班级社交生态简报，用事实描述代替评价
- 话术示例（个别沟通）："我注意到您的孩子最近在小组活动中有些孤立，我想和您一起了解一下他在家里的状态，不是要告状，是想合力帮他。"
- 家长端行动建议：引导家长在家讨论"今天有没有人让你不舒服"而非"今天有没有人欺负你"，降低孩子的防御阈值

## Common Issues

- **活动流于形式**：体验式活动须有真实情境输入（情境卡、角色扮演），避免纯讲授；引导语须开放而非引导"标准答案"
- **家校沟通引发对立**：与家长沟通时避免"您的孩子有问题"的定性表述，改用"我们一起观察到……"的协作框架
- **评估指标难以量化**：可观察信号优先选择行为层面（如：主动邀请边缘同学参与活动的频次、课间孤立现象是否减少），而非态度层面
- **长效建设与即时干预混用**：若教师描述的是当下正在发生的冲突，应切换至即时干预模块，本 skill 不适用
