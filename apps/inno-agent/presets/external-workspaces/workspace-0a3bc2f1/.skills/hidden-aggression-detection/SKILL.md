# 隐性攻击识别与即时干预

## Instructions

- 收到教师描述后，首先判断冲突类型（即时干预 / 长效建设）、涉事角色（攻击方 / 受害方 / 旁观者）、场域属性（公共场合 / 班级内部 / 个别谈话）
- 精准识别隐性攻击信号：言语讥讽伪装成玩笑、集体性回避与哄笑、防御性外归因攀咬、旁观者借机告状等
- 识别深层心理机制：投射与合理化防御（攻击方）、伪正义动机与权力倾轧（旁观者）、自尊受挫与道德焦虑（受害方）
- 所有话术须直接可用，教师拿来即能说，区分正面示范与反面警示
- 在保护每位学生尊严的前提下输出干预方案，避免当众羞辱或合法化"互相揭发"的丛林法则

## Workflow

1. **场景定性**：判断是即时干预需求还是长效建设需求，明确涉事角色与场域
2. **信号识别**：列出本场景中存在的隐性攻击信号及其背后的心理动因
3. **分层干预输出**：
   - 【情绪收敛策略】教师自身情绪管理与场域选择（如将冲突延宕至私密场域处理）
   - 【即时话术】按角色分列：`【攻击方】` `【受害方】` `【旁观者】`
   - 【底线重塑】向全班传递的高阶道德立意
   - 【后续跟进】个别谈话时机与内容建议
4. **理论支撑**（按需附注）：罗特控制点理论、弗洛伊德防御机制理论、SEL框架

## Output Format

```
【场景定性】
冲突类型 / 涉事角色 / 场域属性 / 核心心理机制

【情绪收敛策略】
（教师行动建议，1-2条）

【即时话术】
【攻击方】"……"
【受害方】"……"
【旁观者】"……"

【底线重塑】
（面向全班的高阶道德表达，1段）

【后续跟进】
（个别谈话时机与要点，2-3条）
```

## Examples

**场景：** 课间操讲话，回班后被批评，学生反驳"李某某也在讲"，旁观者跟着告状踩一脚。

**输出示例：**

【场景定性】
即时干预需求；攻击方（外归因防御）+ 旁观者（伪正义告状）+ 受害方（被攀咬的李某某）；场域：班级内部；机制：投射合理化 + 权力倾轧

【情绪收敛策略】
在操场仅用眼神锁定违纪者，不当众爆发；将处理延宕至回班关门后进行，保留学生公众颜面。

【即时话术】
【攻击方】"我此刻的视线只聚焦于你的责任。不要试图拉别人来分担你的犯错成本，那不会让你的错误变少，只会让我看到你回避责任的方式。"
【受害方（被攀咬者）】"你的名字被提到了，我记下了，会单独了解情况。现在不是你的场合，你不需要为自己辩解。"
【旁观者】"放下你们的手。当同伴正在为自己的错误承担责任时，借机踩踏比违规讲话更令我痛心。我们要的是互相扶持的团队，不是互相算计的看客。"

【底线重塑】
"一个班级真正的安全感，来自于每个人知道：当我处于弱势时，没有人会趁机落井下石。这是我对这个班最基本的期待。"

【后续跟进】
- 课后单独找攻击方谈话，聚焦"为什么第一反应是拉别人垫背"，引导内控归因
- 私下关注被攀咬的李某某，确认其心理状态未受波及
- 一周内观察旁观者群体是否有类似告状行为，适时进行小组道德讨论

## Common Issues

- **误区：当众连续追问攻击方**——会激化防御，导致学生更强烈地外归因；应延宕至私密场域处理
- **误区：顺水推舟接受旁观者告状**——会在班级内合法化"互相揭发"文化，破坏安全感底线
- **误区：话术过于书面化说教**——教师应将话术语气调整为平静而坚定，避免高声或情绪化表达
- **遗漏点：忽视受害方（被攀咬者）的心理状态**——需在事后单独确认其感受，防止次生伤害
