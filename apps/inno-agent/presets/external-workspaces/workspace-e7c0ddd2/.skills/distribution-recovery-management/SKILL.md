# 材料分发回收与课堂秩序管理

## Instructions

- 优先诊断混乱根本原因（公平性诉求 / 规则缺失 / 分工不明 / 回收无序），再对症输出方案
- 低年级（1-2年级）强制采用实验桌集中摆放，禁止每人单独分发；中高年级可在规则建立后适度自主
- 为每组明确"材料员（拿取/归还）、操作员、记录员"三角色，职责边界清晰，防止单人霸占
- 分发材料前必须先讲规则、做示范，包括站立位置、凳子摆放、触碰材料时机
- 争抢/玩闹采用三步干预：提醒（口令/手势）→ 收回材料暂停实验 → 复盘讨论正确做法；初期宁可牺牲进度也要建立习惯

## Workflow

1. **接收问题** — 识别年级、班级人数、具体混乱场景（争抢/霸占/回收混乱/平行班差异）
2. **问题诊断** — 判断根本原因，区分是规则未建立还是执行失控
3. **输出摆放方案** — 说明桌面布置方式，低年级集中摆放细节
4. **输出分工模板** — 三角色职责说明，可直接贴在小组卡上
5. **输出干预话术** — 提醒话术 / 收回话术 / 复盘引导语，均为完整句子
6. **输出回收流程** — 提前30秒预告 → 统一口令 → 材料员归还 → 清点确认
7. **学段适配说明** — 明确低/中/高年级差异化建议

【集中摆放方案】

## Examples

**场景：一年级积木实验，学生因颜色大小不一争抢了一节课**

```
【问题诊断】根本原因是每人单独分发，触发低年级公平性心理，学生注意力转移到材料比较而非实验任务。

【集中摆放方案】
- 将全组积木统一放在实验桌中央托盘内，任何人不得私自拿走
- 实验桌四周各站一名学生，凳子推入桌下，围桌站立操作

【小组分工模板】
- 材料员：每次实验开始前从托盘取出所需积木，结束后放回
- 操作员：按步骤完成测量/观察动作
- 记录员：在记录单上填写结果

【干预话术】
- 提醒话术："停，手放背后，眼睛看我。"
- 收回话术："材料先回到老师这里，我们来说说刚才哪里出了问题。"
- 复盘引导语："积木是我们小组共用的，谁来说说怎么用才公平？"

【回收流程】
"还有30秒，材料员准备" → "停！材料员把所有积木放回托盘" → 教师逐组确认托盘数量 → 全部确认后方可离座

【学段适配说明】
- 低年级：全程集中摆放，材料员由教师指定，轮流担任
- 中年级：可由小组自选材料员，但仍需集中摆放
- 高年级：规则稳定后可允许小组自主分配材料，教师巡视抽查
```

## Common Issues

- **信息不足时**：若教师未说明年级或实验名称，先给出通用方案，同时主动询问"请告诉我年级和实验内容，我可以给出更精准的建议"
- **平行班差异**：重点排查两班规则讲解是否一致、示范是否到位，而非调整材料方案；混乱班需重新从"规则先行"环节做起
- **回收超时**：根本原因通常是角色不明确，优先检查材料员是否清楚自己的归还职责，而非增加口令频次
- **学生藏匿材料**：回收前增加"请检查抽屉和书包"口令，并在清点时公开报数，形成集体监督
