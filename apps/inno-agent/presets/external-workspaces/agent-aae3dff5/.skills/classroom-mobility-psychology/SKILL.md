# 课堂动线规划与心理安全构建

## When to use
- 教师询问"应该什么时候走下讲台"、"如何设计巡航路线"或"看到学生玩手机第一个动作是什么"等物理空间管理问题
- 学生抢答答不上来遭遇哄笑、尴尬冷场，需要化解话术与心理保护机制
- 新教师"恐慌快讲"（90分钟课60分钟讲完），需要设计课堂节奏减速锚点
- 学生反复试探底线（刚收手机又拿出来），需要非语言干预与博弈策略
- 需要构建课堂心理安全区，设计"答错也表扬意愿"的情感防跌落机制

## When not to use
- 涉及超星等平台技术参数、积分规则设置、防作弊功能开启 → 交给 `mobile-platform-configuration`
- 询问"如何在平台发测验"、"抢答积分具体参数怎么设"等技术操作 → 交给 `mobile-platform- configuration`
- 学生出现严重情绪失控、自伤倾向或暴力冲突 → 立即建议联系学校心理中心或班主任，不适用课堂管理技巧
- 要求设计监控学生屏幕、远程控制手机等侵犯隐私功能 → 拒绝并建议使用任务驱动替代

## Instructions
1. **动线设计优先**：强制要求教师脱离讲台，提供具体的"何时走、停哪里、看什么"的巡航路线
2. **非语言先行**：强调肢体干预先于口头批评，如走近、轻敲桌面、停留注视等低对抗动作
3. **心理保护兜底**：任何互动设计必须包含"答错如何表扬"的话术模板，防止学生彻底摆烂
4. **节奏强制切分**：为长时间课堂硬性插入3个减速锚点，打断教师的"恐慌快讲"惯性

## Workflow
1. **痛点识别**：判断用户描述属于"讲台依赖"、"恐慌快讲"、"尴尬冷场"还是"底线博弈"
2. **动线规划**：设计"U型/S型巡航路线"，明确触发走动的时机（讲完一个知识点/发现低头率超过30%）
3. **干预设计**：针对具体场景选择非语言动作（走近→停顿→轻敲）或话术（"手速最快说明很想参与"）
4. **心理加固**：为可能的失败场景预设保护机制（答错给参与分、全班哄笑时的两句化解语）
5. **节奏植入**：在90分钟课中硬性插入3个手机互动锚点（第20/50/75分钟），强制减速

## Output Format

```markdown
### 场景判断
[识别属于讲台依赖/恐慌快讲/尴尬冷场/底线博弈中的哪一类]

### 物理动线方案
- **触发时机**：[何时开始走动]
- **巡航路线**：[具体走位描述，如"讲台→左侧过道→后排→右侧过道" ]
- **停留点**：[在哪些位置必须停下观察/指导]

### 非语言干预策略
- **第一步动作**：[如"走近至课桌旁停留3秒"]
- **第二步动作**：[如"食指轻敲桌面，目光注视"]
- **底线动作**：[如"站定不动，等待收手机"]

### 心理保护话术
- **预防性话术**：[互动前的安全感建立]
- **补救性话术**：[答错/尴尬时的即时化解，必须包含两句具体 wording]
- **激励性话术**：[保护自尊心的正向反馈]

### 节奏控制锚点
- **锚点1**：[第X分钟，什么触发条件，什么互动形式]
- **锚点2**：[第X分钟，什么触发条件，什么互动形式]
- **锚点3**：[第X分钟，什么触发条件，什么互动形式]

### 风险提示
- [底线博弈中的心态警告：做好反复督促的心理准备]
- [安全转介提示：如涉及心理危机需转介]

### 后续评估
- **抬头率观察点**：[课堂上什么现象说明有效]
- **行为改变指标**：[学生从X行为变为Y行为视为成功]
```

## Examples

**Example 1: 抢答尴尬冷场**

用户：后排男生抢答答不出，全班哄笑，他憋红了脸，我该说什么？

输出：
- **场景判断**：挫败性互动/尴尬冷场，需立即心理保护
- **物理动线**：已走近学生（触发走动在先），此刻应半步侧向学生遮挡部分同学视线，形成物理保护
- **非语言**：手掌向下轻压示意安静，目光扫视制止哄笑
- **话术**："没关系，小明手速最快，说明他很想参与，给你加1分参与分，请坐。"（停顿）"这个问题确实有难度，谁来帮帮他？"
- **节奏**：立即转入"求助同伴"环节，避免僵持超过5秒

**Example 2: 反复试探底线**

用户：我敲了桌子他收起来，我回讲台他又拿出来，怎么办？

输出：
- **场景判断**：底线博弈，学生测试教师决心
- **核心心态**：做好"这节课需要反复下去督促3-5次"的心理准备，不生气不妥协
- **动线调整**：第二次不走回讲台，而是在教室后方/侧方停留讲授，保持物理接近
- **非语言升级**：第二次走近→直接伸手示意"给我"，不说话，眼神坚定，等待交出
- **话术**：收走后不说教，继续讲课，课后单独约谈："我知道手机很吸引你，但这节课我们需要你抬头，明天还你，我们约定好？"

**Example 3: 恐慌快讲减速**

用户：90分钟的课我总是60分钟就讲完，然后学生玩手机，怎么切分？

输出：
- **场景判断**：恐慌快讲，缺乏节奏锚点
- **强制切分**：
  - 锚点1（第20分钟）：讲完知识点A→超星发2分钟抢答→强制等待数据反馈
  - 锚点2（第50分钟）：讲完知识点B→平台讨论题（未提交隐藏答案）→强制等待独立思考
  - 锚点3（第75分钟）：实操演示前→微课视频分发→强制观看3分钟再动手
- **动线配合**：每个锚点触发时，必须从讲台走到后排，用走动强制自己慢下来

## Common Issues
1. **"我走下去了，但不知道停下来干什么"** → 明确停留点任务：看学生笔记/看手机屏幕内容/指具体错题，避免无目的乱逛
2. **"敲桌子后学生瞪眼/摔手机"** → 立即升级为底线博弈，停止言语冲突，保持安全距离，课后处理，必要时转介班主任
3. **"话术说出来很假，学生不信"** → 配合非语言：蹲下平视（降低压迫感）、轻拍肩膀（建立连接）、微笑（释放善意）
4. **"减速锚点插不进去，怕讲不完"** → 强制砍掉30%内容，优先保证学生听懂而非教师讲完，用当堂测验数据倒逼内容精简
