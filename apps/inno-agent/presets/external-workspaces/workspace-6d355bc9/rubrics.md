# 小学低年级科学课小组合作培养助手 评分量规

## 评分维度说明

本量规适用于评估"小学低年级科学课小组合作培养助手"的回答质量，共设 **5 个维度**，每个维度满分 **4 分**，总分 **20 分**。

---

## 维度一：问题诊断准确性

> 助手是否准确识别教师描述的课堂问题类型，并给出符合低年级学生认知特点的根源分析。

| 分值 | 描述 |
|------|------|
| 4 | 准确识别问题类型（合作行为缺失 / 分工协商失败 / 综合流程类），诊断语言简洁、直指根源，明确体现对1-2年级学生具象思维特点的理解 |
| 3 | 问题类型识别基本正确，诊断有一定针对性，但对低龄认知特点的关联分析略显不足 |
| 2 | 问题识别存在偏差或过于笼统，诊断停留在表面描述，未能指向可操作的干预方向 |
| 1 | 未能有效识别问题类型，诊断内容与教师实际情境明显脱节，或以抽象理论替代具体分析 |

---

## 维度二：方案可操作性

> 助手给出的解决方案是否具体到动作步骤，能否在普通小学教室或实验室中立即落地执行。

| 分值 | 描述 |
|------|------|
| 4 | 方案分步骤清晰呈现，每步均落实到可见的具体动作（如教师站位、材料摆放、学生肢体动作等），无需额外解读即可执行；不包含超出普通课堂条件的要求 |
| 3 | 方案步骤较清晰，大部分建议可直接执行，但个别步骤仍较抽象或依赖额外资源 |
| 2 | 方案有一定方向性，但步骤模糊，教师需自行大量补充才能落地；或存在不适合普通课堂的建议 |
| 1 | 方案以原则性表述为主（如"要引导学生合作"），缺乏具体操作步骤，实际可用性极低 |

---

## 维度三：话术即用性

> 助手提供的教师话术是否自然、准确，可在课堂中直接说出，且符合低年级学生的语言理解水平。

| 分值 | 描述 |
|------|------|
| 4 | 话术用引号清晰标注，语言口语化、简短易说，贴合1-2年级学生理解水平；覆盖关键干预节点（示范、纠偏、鼓励、复盘等），教师无需改写即可使用 |
| 3 | 话术基本可用，语言较自然，但部分话术偏书面化或略长，需教师稍作调整；覆盖主要节点 |
| 2 | 话术提供不完整，或语言超出低年级理解范围（如使用抽象词汇"协作意识""责任感"等），实际使用时需大幅修改 |
| 1 | 未提供具体话术，或话术与课堂情境严重不符，无法直接使用 |

---

## 维度四：全员参与保障

> 助手的方案是否兼顾小组内每位学生的参与机会，有效防止学生被边缘化，并体现公平分工意识。

| 分值 | 描述 |
|------|------|
| 4 | 方案明确为每位学生设定角色或参与路径，主动识别并应对边缘化风险；分工建议体现公平性，弱势学生（如内向、能力较弱者）有专属参与设计；安全规则在材料发放前优先说明 |
| 3 | 方案基本覆盖全员参与，有角色分配建议，但对边缘化学生的专项关注不够充分；安全规则有所提及 |
| 2 | 方案以"快速完成任务"为导向，部分学生可能被忽视；角色分配不均衡，或安全规则缺失 |
| 1 | 方案未考虑全员参与，存在明显边缘化风险，或完全忽略实验安全规则 |

---

## 维度五：闭环流程完整性

> 助手的回答是否覆盖"示范→分工→实验→复盘→强化"的完整教学闭环，并提供后续跟进建议。

| 分值 | 描述 |
|------|------|
| 4 | 回答结构完整，清晰涵盖当前干预措施与后续复盘/强化建议；明确指出下次实验前需强化的方向；整体形成可持续改进的闭环，而非一次性解决方案 |
| 3 | 回答包含主要干预步骤和基本复盘建议，但后续强化方向较笼统，闭环意识有所体现但不够完整 |
| 2 | 回答聚焦于当前问题的即时处理，缺乏复盘环节或后续跟进建议，流程呈碎片化 |
| 1 | 回答仅给出单一建议，无流程意识，未涉及复盘与强化，无法支持教师持续改进 |

---

## 总分参考

| 总分区间 | 综合评价 |
|----------|----------|
| 18–20 分 | 优秀：方案高度可落地，话术即用，全员参与有保障，闭环完整，可直接用于课堂 |
| 14–17 分 | 良好：方案基本可用，个别维度有提升空间，教师稍作调整后可执行 |
| 10–13 分 | 一般：方案方向正确但操作性不足，话术或流程存在明显缺失，需较多补充 |
| 6–9 分  | 较差：方案以抽象建议为主，落地困难，存在边缘化风险或安全隐患 |
| 5 分及以下 | 不合格：回答与课堂实际严重脱节，无法为教师提供有效支持 |
