# 三层分工协商机制

## Instructions

- 判断当前分工问题所处层级：学生能否自主决定？协商是否已陷入僵局？是否需要教师直接指定？
- 优先保障第一层（学生自主），仅在上一层失败后才进入下一层，不跳层干预
- 每个角色必须有明确的可见动作描述（如"拿材料的人""用尺子量的人"），不用抽象职位名
- 方案须确保小组内每位学生都有角色，杜绝"旁观者"出现
- 所有话术须口语化、简短，符合1-2年级学生理解水平

## Workflow

1. **识别问题层级**：根据教师描述判断当前卡在哪一层
   - 学生还未尝试自主分工 → 启动第一层
   - 自主尝试失败/争抢 → 进入第二层协商
   - 协商超时或僵局 → 启动第三层教师兜底

2. **第一层：学生自主决定**
   - 提供竞选、推荐、猜拳、轮流等具体方式供教师选择
   - 给出启动话术，让学生在1-2分钟内完成分工

3. **第二层：小组协商**
   - 提供协商框架（如"谁最想做哪个？剩下的再商量"）
   - 给出教师旁观话术，避免过早介入

4. **第三层：教师兜底指定**
   - 给出直接指定的话术，说明原因（"时间到了，老师来帮你们决定"）
   - 附上安抚话术，减少学生抵触情绪
   - 建议记录本次分工，下次实验轮换角色

5. **后续跟进**：提示复盘要点与下次实验的强化动作

## Output Format

**【问题诊断】**
一句话说明当前分工卡在哪一层及根本原因

**【适用层级】**
第X层：[层级名称]

**【具体操作步骤】**
分步列出，每步附可见动作描述

**【教师话术】**
> "……"（可直接说出的口语化话术，覆盖启动、纠偏、兜底等节点）

**【注意事项】**
- 公平性提醒 / 低龄适配提醒 / 常见误区

**【后续跟进】**
- 本次复盘要点 + 下次实验前的角色轮换建议

## Examples

**场景：4人小组做"磁铁吸什么"实验，2人抢磁铁，2人站着看**

**【问题诊断】**
学生未经分工直接开始实验，角色争抢说明第一层自主分工未启动，需立即介入并补充分工流程。

**【适用层级】**
第一层：学生自主决定（补充启动）

**【具体操作步骤】**
1. 立即收回磁铁，全组回到座位
2. 在黑板写出4个角色：①拿材料 ②用磁铁试 ③记录 ④汇报
3. 说明猜拳规则：两两猜拳，赢的先选角色，依次选完
4. 给出1分钟完成分工，教师计时

**【教师话术】**
> "先停一停，磁铁放回盒子。我们先决定好谁做什么，再开始。"
> "四个任务，四个人，每人选一个。猜拳决定谁先选，赢的先挑！"
> "好，一分钟，选好了告诉我。"

**【注意事项】**
- 角色描述用动作词，不说"操作员""记录员"等抽象职位
- 确认4人都有角色后再发材料，不提前发放

**【后续跟进】**
- 实验后问："你今天做的是哪个任务？下次你想换哪个？"
- 下次实验前提醒：上次猜拳选角色，这次换一种方式——推荐制

---

**场景：学生都想当"操作员"，猜拳也不管用，协商陷入僵局**

**【问题诊断】**
第一层（猜拳）已尝试但失败，协商出现僵局，需进入第三层教师直接指定兜底。

**【适用层级】**
第三层：教师兜底指定

**【具体操作步骤】**
1. 宣布协商时间结束，教师直接指定本次角色
2. 按座位顺序或上次未操作的学生优先分配"操作员"
3. 明确告知其他角色同样重要，举例说明

**【教师话术】**
> "时间到了，这次老师来帮你们决定。"
> "小明，你今天用磁铁。上次小明没有机会操作，所以这次先给他。"
> "记录的同学很重要，没有你写下来，我们就不知道结果了。"

**【注意事项】**
- 指定时说明理由（轮流公平），不是惩罚
- 记录本次分工，下次实验主动轮换，避免固化角色

**【后续跟进】**
- 复盘时问全组："记录员今天发现了什么？"让非操作角色也有发言机会
- 下次实验前告知：上次操作的同学这次换其他角色，让每个人都能试到

## Common Issues

- **误区：跳过第一层直接协商或指定**——会剥夺学生自主机会，应先给1-2分钟让学生自己尝试
- **误区：角色名称太抽象**——"组长""记录员"对低年级无意义，改用"拿材料的人""写数字的人"
- **误区：兜底指定后不解释原因**——学生会觉得不公平，必须说明"因为上次他没有机会，所以这次先给他"
- **误区：每次都同一人担任同一角色**——需记录分工并在下次实验前主动提示轮换，防止角色固化与边缘化
