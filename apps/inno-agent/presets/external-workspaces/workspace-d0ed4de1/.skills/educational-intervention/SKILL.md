# 教育干预与沟通话术

## When to use
- **课堂即时干预**：发现学生偷玩手机需非侵入式提醒或即时处置，避免课堂冲突升级
- **深度共情谈话**：学生因手机被收情绪低落、抗拒沟通，需探查成瘾背后的心理动机（逃避学业、社交焦虑、亲情缺失）
- **主题教育实施**：设计手机管理主题班会、辩论会或系列课程，需具体议程、互动环节与教案
- **家校协同沟通**：与抵触型家长沟通手机监管问题，需非告状式话术与家庭屏幕契约指导
- **长期转化帮扶**：针对深度成瘾学生设计多阶段心理重塑、习惯跟踪与微目标趋近方案

## When not to use
- **制度流程设计**：涉及班级/学校手机管理制度、保管柜SOP、自治委员会架构、分级惩戒机制设计 → **转 campus-phone-governance**
- **全校性治理框架**：涉及跨班级协调、全校性例外审批表格、统一保管流程 → **转 campus-phone-governance**
- **心理危机干预**：学生出现自伤自杀倾向、重度抑郁或病理性成瘾 → **建议立即转介学校心理教师或专业医疗机构，不单独处理**
- **技术监控手段**：涉及监控软件安装、信号屏蔽、私自搜身或查阅私人信息 → **拒绝并提供合规替代方案**

## Instructions
- **柔性表达**：使用"约定""暂停使用""数字斋戒"等建设性词汇，避免"禁止""处罚"等对抗性语言
- **话术标注**：关键对话脚本用【】明确标注，便于教师直接使用
- **学段适配**：小学侧重"习惯养成+亲情替代"，初中侧重"契约自治+边界试探"，高中侧重"自律训练+未来准备"
- **隐私保护**：所有干预手段须符合《未成年人保护法》，禁止侵犯学生人格尊严的识别或惩戒方式

## Workflow
1. **场景诊断**：判断当前属于"即时干预""深度谈话"还是"长期转化"，识别是否存在心理危机信号
2. **策略匹配**：根据问题性质选择非侵入式干预技巧、共情谈话框架或家校协同方案
3. **方案生成**：输出包含具体时间分配（如45分钟班会拆解）、角色分工、话术脚本的完整操作指南
4. **安全校验**：检查方案是否包含当众羞辱、过度惩戒或心理伤害风险，必要时插入转介建议
5. **迭代反馈**：提供7天/30天评估维度，支持PDCA循环改进

## Output Format
- **场景判断**：当前情境性质与风险等级（低/中/高）
- **即时行动清单**：1-2-3步具体动作，含时间控制与空间选择
- **核心话术脚本**：【教师开场】【共情回应】【需求探查】【约定达成】等模块化话术
- **风险提示**：可能引发的师生对立、亲子冲突及规避方法
- **后续评估**：3天反应观察、7天行为跟踪、30天习惯巩固的评估指标

## Examples
- **课堂非侵入式干预**：眼神停留3秒→走近轻拍桌面→课后约谈，避免当众没收引发对抗
- **深度共情谈话**：【"我注意到你最近有些疲惫，愿意和我聊聊手机对你意味着什么吗？"】→【"听起来手机是你放松的港湾，如果暂时靠岸，我们可以一起找其他码头"】
- **家长会沟通脚本**：【"这次联系是想和您一起想办法，而非单纯告状。我们一起制定家庭屏幕契约如何？"】

## Common Issues
- **避免当众羞辱**：不当众点名没收，改用"课后到我这里来一趟"保全学生自尊
- **防止家校二次伤害**：指导家长避免简单粗暴体罚，提供"固定视频时间""屏幕契约"等替代方案
- **识别备用机线索**：通过充电习惯、充电宝异常等观察，**绝不建议搜身**，转为私下询问与契约重申
- **区分正常社交与成瘾**：承认留守儿童亲情联系需求，区分"必要通讯"与"无节制娱乐"，避免一刀切否定
