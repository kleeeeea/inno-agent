# 校园手机治理方案设计

## When to use

- 设计班级/学校手机管理制度、公约制定流程或全校治理框架
- 规划手机统一保管的登记、存放、领还标准作业程序（SOP）
- 构建分级处理制度（首次→二次→三次违规的梯度惩戒机制）
- 设计运动会、研学等特殊场景的例外审批流程与防错机制
- 搭建学生自治委员会+教师督导的双线监管组织架构

## When not to use

- **涉及具体谈话话术与破冰技巧**（如"如何与情绪激动的学生沟通"）→ 转 educational-intervention
- **单次班会的详细教案与互动游戏设计**（如"45分钟主题班会逐字稿"）→ 转 educational-intervention  
- **个别学生心理干预或家校沟通话术**（如"如何给家长打电话不说成告状"）→ 转 educational-intervention
- **重度心理危机或病理性成瘾**（自伤倾向、重度抑郁）→ 建议立即转介学校心理教师或专业医疗机构
- **技术监控手段咨询**（信号屏蔽破解、监控软件、私自搜身）→ 明确拒绝并告知法律风险

## Instructions

1. **学段差异化设计**：小学侧重"完全代管+习惯养成"，初中侧重"契约自治+边界试探"，高中侧重"自律训练+未来准备"
2. **柔性德育语言**：避免"禁止""处罚"等对抗性词汇，改用"约定""暂停使用""数字斋戒"等建设性表达
3. **刚性制度框架内的最大人性化弹性**：兼容留守儿童亲情需求等刚性通讯需求，设置通讯平替（磁卡电话/老人机借用）与紧急绿色通道
4. **法律合规底线**：所有方案须符合教育部《关于加强中小学生手机管理工作的通知》（2021）及未成年人保护法，禁止体罚、变相体罚或侵犯隐私的监控手段

## Workflow

1. **场景诊断**：识别用户处于"制度初创""流程优化"还是"特殊场景补充"阶段，确认学段特征与寄宿制/走读制差异
2. **需求分层**：区分常规管理（课堂保管）、刚性需求（家校联系）、例外场景（活动记录）、风险防控（备用机识别）
3. **制度设计**：生成包含"认知对齐→民主定规→日常规训→协同托底"的线性流，嵌入"通讯平替"与"人性化例外"分支流
4. **安全校验**：检查是否包含侵犯隐私（搜身、查看私人信息）、过度惩戒（公开砸手机、长期扣留至毕业）或心理伤害风险
5. **迭代机制**：设计PDCA循环评估节点（试行→反馈→修订），配套自治委员会与家委会双向研讨流程

## Output Format

```markdown
### 场景判断
- 适用学段：[小学/初中/高中]
- 管理场景：[寄宿制全封闭/走读制/混合]
- 核心痛点：[课堂分心/家校真空/备用机泛滥/搜题依赖]

### 制度框架
**1. 认知对齐机制**
- 开学典礼/家长会统一路径
- 政策依据引用（教育部2021《通知》）

**2. 民主公约制定流程**
- 讨论→草案→表决→试行→修订（含时间节点）
- 自治委员会组织架构与轮值表模板

**3. 日常保管SOP**
- 收机时点：[如早晨入校/每节课前]
- 登记流程：双人核对（班干部+教师）
- 存放安全：专用保管柜+钥匙分离管理
- 领还确认：签字领取+损坏核查机制

**4. 分级处理制度**
- 首次：【温和提醒+公约重申】
- 二次：【家长告知+暂停使用申请】  
- 三次：【集中保管至周末+家校面谈】

**5. 例外审批体系**
- 适用场景：[运动会/研学/医疗监测]
- 申请流程：提前书面申请→班主任签字→活动结束立即收回
- 防错机制：统一保管袋+现场监督员

**6. 通讯平替方案**
- 宿舍磁卡电话/班级老人机借用制度
- 班主任手机紧急联系绿色通道

### 风险预警
- [ ] 是否存在侵犯隐私条款（搜身、查看内容）
- [ ] 是否设置过度惩戒（经济罚款、公开羞辱）
- [ ] 是否忽视留守儿童刚性需求
- [ ] 备用机识别手段是否合规（观察充电习惯、学生互查而非搜身）

### 评估维度
- 制度颗粒度：[具体时段/责任人/工具]
- 学生参与度：[民主程序完整性]
- 家校协同度：[家长告知书/ screen time契约模板]
- 弹性空间：[例外条款覆盖率]
```

## Examples

**用户输入**："设计一套班级手机公约的制定流程"

**输出**：
```markdown
### 场景判断
- 适用学段：初中（契约自治关键期）
- 管理场景：走读制
- 核心痛点：学生暗地抵触、公约流于形式

### 制度框架
**2. 民主公约制定流程**
- **第一周：议题讨论**（班会40分钟）
  - 匿名便签：学生写下"我最讨厌的手机使用场景"与"我需要的通讯时刻"
  - 分组辩论：A组"手机是学习工具" vs B组"手机是注意力黑洞"
- **第二周：草案拟定**
  - 由自治委员会（3名轮值生+1名科代表）整理意见，形成《班级手机管理草案》
  - 关键条款示例：【带机入校需提前一日在"数字自律角"登记】【课堂时段手机入"睡眠舱"保管袋】
- **第三周：民主表决**
  - 三分之二多数通过，教师保留"安全底线否决权"（如不得约定"教师无权保管"）
- **第四周：试行观察**
  - 试行两周，自治委员会每日记录执行情况
- **第六周：修订定稿**
  - 根据试行数据修订条款（如调整收机时间点），正式发布《班级数字自律公约》

**6. 通讯平替方案**
- 班级设置"亲情联络角"，配备仅具通话功能的老人机2台，午休/放学后免费开放
- 紧急联系：班主任手机"绿色通道"，学生可随时借用并登记通话时长

### 风险预警
- [x] 已通过匿名方式保护学生隐私
- [x] 教师保留底线否决权防止条款过松
- [ ] 需补充：不得因公约执行影响留守儿童必要的亲子通话
```

## Common Issues

1. **"一刀切"刚性陷阱**：忽视寄宿生亲情需求导致备用机泛滥，应嵌入通讯平替与例外审批的弹性设计
2. **制度流于形式**：缺乏"试行-修订"闭环，公约成墙上摆设，需配套自治委员会轮值与PDCA评估节点
3. **隐私侵犯风险**：建议采用"突击搜身""查看手机内容"等手段识别备用机，应替换为观察充电习惯、学生互查等合规方式
4. **学段错配**：对小学生采用高中自律模式或对高中生采用完全代管，需严格区分"完全代管/契约自治/自律训练"三阶段特征
