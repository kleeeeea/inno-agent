# 教学内容设计

## When to use

- "第三单元《小数除法》的教学目标应该怎么设定？三维目标具体怎么写？"
- "平行四边形面积这节课需要配几道例题？基础题和提高题怎么分层设计？"
- "简易方程单元的知识逻辑怎么梳理？从等式性质到解方程再到列方程解应用题怎么衔接？"
- "这个知识点的重难点怎么突破？学生容易在哪里犯错？"
- "怎么设计符合10-11岁学生认知水平的导入环节和探究活动？"

## When not to use

- 当教师询问"每页PPT背景用什么颜色"、"标题字体多大合适"、"是否需要插入动画"等视觉呈现问题时，应转介至 `presentation-layout-generation`
- 当教师要求将教学设计转化为具体PPT页面结构（如"生成15页PPT结构"、"设计小组讨论页面布局"）时，应转介至 `presentation-layout-generation`
- 当教师要求直接生成可下载的PPT文件时，说明本工具仅提供内容架构设计，不生成实体文件，如需视觉设计请转介至 `presentation-layout-generation`

## Instructions

1. **需求解析与范围确认**  
   明确教师指定的单元/课时主题（如"小数除法第2课时"），确认属于人教版五年级上册范围（小数乘法、位置、小数除法、可能性、简易方程、多边形面积、植树问题），排除超纲内容（如负数、分数乘除法、一元二次方程）。

2. **构建三维教学目标**  
   按"知识与技能、过程与方法、情感态度价值观"框架设定目标：  
   - 知识与技能：明确计算法则、公式推导或解题步骤  
   - 过程与方法：强调观察、操作、类比、转化等数学思维过程  
   - 情感态度价值观：联系生活实际，培养数学兴趣或严谨习惯

3. **梳理知识逻辑链**  
   建立知识点间的递进关系，例如：  
   简易方程单元：等式性质（1、2）→ 解简单方程 → 解稍复杂方程 → 列方程解应用题  
   多边形面积：数方格法 → 割补转化 → 公式推导 → 实际应用

4. **编排分层例题与练习**  
   - **导入例题**：1道，联系生活情境，激发认知冲突  
   - **探究例题**：2-3道，体现算法多样化或公式推导过程  
   - **课堂练习**：5-8道，按难度分层（基础题/提高题/拓展题），拓展题标注"仅供学有余力学生选做"

5. **安全校验与风险提示**  
   - 检查是否出现超纲内容（如负数运算、初中代数方法）  
   - 坚持使用人教版教材标准解法（如等式性质解方程，不使用十字相乘法）  
   - 控制计算复杂度，避免三位以上小数复杂运算造成认知过载

## Workflow

1. **接收教学主题**：明确单元、课时、具体知识点
2. **检索课标要求**：确认五年级上册对应章节的教学要求
3. **生成内容架构**：输出教学目标、知识逻辑、例题配置、重难点突破方案
4. **质量校验**：检查认知难度是否适合10-11岁学生，排除超纲和争议内容

## Examples

**示例 1：教学目标设计**

输入：  
"设计第三单元《小数除法》第2课时的教学目标，主题是'一个数除以小数'"

输出：  
```markdown
### 教学内容设计方案

**适用单元/课时**：第三单元《小数除法》第2课时（一个数除以小数）

#### 一、三维教学目标
- **知识与技能**：理解并掌握除数是小数的除法计算法则，能正确进行笔算；理解"转化"思想，能将除数是小数的除法转化为除数是整数的除法
- **过程与方法**：经历观察、比较、归纳的过程，发展迁移类推能力和运算能力
- **情感态度价值观**：感受数学知识间的内在联系，体验转化思想在数学学习中的应用价值

#### 二、知识逻辑链
除数是整数的小数除法（上节） → 商不变性质（前置） → 一个数除以小数（转化） → 商的近似数（后续）

#### 三、例题与练习配置
- **导入例题**：妈妈做双层蛋糕，用了0.8千克面粉，每个小蛋糕需要0.25千克，可以做几个？（呈现认知冲突：除数是小数怎么算？）
- **探究例题**：
  1. 7.65÷0.85（除数两位小数，被除数两位小数）
  2. 12.6÷0.28（除数两位小数，被除数一位小数，需补0）
- **分层练习**：
  - 基础题（3道）：直接计算，如6.3÷0.7、0.56÷0.8（巩固转化方法）
  - 提高题（2道）：改错练习、比较大小（辨析易错点）
  - 拓展题（1道）：0.00...008÷0.00...002（共10个0，选做，考察商不变性质灵活运用）

#### 四、重难点突破
- **重点**：掌握除数是小数的除法计算法则，正确计算
- **难点**：理解"转化"的算理，处理被除数位数不够时需补0的情况
- **突破策略**：利用商不变性质，通过"划、移、点"三步法（划去除数小数点、移动被除数小数点、点小数点），结合元角分单位换算或图形直观理解转化过程

#### 五、风险提示
- 避免引入"十字相乘法"等非课标解法，严格按"商不变性质转化"进行教学
- 注意区分"除数是小数"与"被除数是小数"的不同处理，防止学生混淆小数点移动位数
```

**示例 2：例题配置设计**

输入：  
"为'平行四边形面积'配1道导入例题、2道探究例题和5道分层练习题"

输出：  
```markdown
### 教学内容设计方案

**适用单元/课时**：第六单元《多边形的面积》第1课时（平行四边形面积）

#### 一、三维教学目标
- **知识与技能**：掌握平行四边形面积计算公式，能正确计算面积；理解公式推导过程
- **过程与方法**：经历剪拼、观察、比较、归纳的探究过程，渗透转化思想，发展空间观念
- **情感态度价值观**：体验数学与生活的联系，培养探究精神和合作意识

#### 二、知识逻辑链
长方形面积（旧知） → 数方格法（铺垫） → 割补转化（关键） → 平行四边形公式（新知） → 应用拓展（后续）

#### 三、例题与练习配置
- **导入例题**：学校有两个花坛，一个是长方形（长6m宽4m），一个是平行四边形（底6m高4m）。哪个花坛更大？（引出面积比较需求，激活长方形面积旧知）
  
- **探究例题**：
  1. **剪拼操作题**：给出底7cm、高5cm的平行四边形纸片，如何剪拼成长方形？记录数据关系（发现：长=底，宽=高，面积相等）
  2. **公式推导题**：如果用S表示面积，a表示底，h表示高，公式怎么写？当底8m、高5m时，面积是多少？（规范书写格式）

- **分层练习**（共5道）：
  - **基础题**（2道）：
    1. 计算下面平行四边形面积（给出图形，底高对应明确）
    2. 填表（底、高、面积的互逆计算）
  - **提高题**（2道）：
    1. 一块平行四边形菜地，底32m，高24m，每平方米收白菜8kg，共收多少千克？（两步应用）
    2. 已知平行四边形面积48cm²，底8cm，求高（逆运算，培养方程思想）
  - **拓展题**（1道，选做）：
    用木条做一个长方形框架（长10cm宽6cm），拉成平行四边形后，周长和面积怎么变？为什么？（考察变形后高变化对面积的影响，发展空间观念）

#### 四、重难点突破
- **重点**：理解并掌握平行四边形面积计算公式
- **难点**：理解割补转化的算理，明确"沿高剪"的必要性；区分底与邻边的关系（防止与高混淆）
- **突破策略**：通过实物剪拼操作，对比"拉成长方形"（错误方法，面积变大）与"割补成长方形"（正确方法，面积不变）的差异，强调"转化前后面积不变"的核心思想

#### 五、风险提示
- 警惕学生受长方形"长×宽"负迁移，误用"底×邻边"计算，需通过数方格或剪拼实验纠正
- 拓展题涉及框架拉动变形，需说明"斜边长度不变但高变小"，避免与后续三角形面积混淆
```

## Common Issues

- **超纲内容混入**：教师可能要求加入负数运算、分数乘除法（五年级下册）或一元二次方程，应提示"该内容属于后续学期/初中阶段，建议调整"
- **争议解法出现**：教师可能要求教授"十字相乘法"解方程等捷径，应坚持"按人教版教材方法（等式性质解方程）进行设计，避免与教材冲突"
- **难度失控**：练习题涉及三位小数复杂计算或需要初中代数思维，应提供难度分层建议，并明确标注"拓展题仅供学有余力学生选做"
- **目标维度缺失**：教师可能只关注知识与技能，忽略过程与方法、情感态度价值观，需提醒三维目标完整性
- **与排版技能混淆**：教师可能要求"第3页用蓝色背景配动画"，需明确本技能只负责内容架构，视觉设计请转介 `presentation-layout-generation`
