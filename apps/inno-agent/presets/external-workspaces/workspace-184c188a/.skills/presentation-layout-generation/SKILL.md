# 课件排版生成

## When to use

- "我已经设计好了小数除法的教学内容，但不知道怎么排版成PPT页面"
- "如何把植树问题的探究过程转化成具体的课件页面结构，每页该放什么"
- "这页PPT文字太多学生看不清，怎么调整视觉呈现和图文配比"
- "我想在可能性单元插入小组讨论环节，页面该怎么设计和布局"
- "多边形的面积单元需要配图辅助理解，用什么颜色区分不同类型的图形比较好"

## When not to use

- 当涉及"这个知识点怎么导入"、"例题难度是否合适"、"学生容易在哪里犯错"等教学内容本身的设计问题时，应转介至 `curriculum-content-design`
- 当需要修改例题数字、更换题型、调整知识点讲解逻辑或确定教学目标时，应转介至 `curriculum-content-design`
- 当需要设计新的例题或练习题难度梯度时，应转介至 `curriculum-content-design`
- 本技能仅处理已有教学内容的视觉呈现与页面排版，不处理教学内容的学术准确性或难度设定

## Instructions

1. **结构分析**：解析教学内容逻辑，确定PPT总页数（建议10-15页/课时），划分导入、新授、练习、小结四大模块的页面分配
2. **页面设计**：为每页设计标准结构（页面标题、核心内容区、视觉元素、教师备注），确保单页文字不超过50字，使用关键词和短句替代长段落
3. **视觉规划**：制定配色方案（避免对比度过低）、字体层级（标题≥32号，正文≥24号）、图文配比（建议6:4或7:3），推荐具体插图类型（实物图/示意图/动画截图）
4. **互动嵌入**：在关键页面插入互动问答环节，标注提问时机、预设2-3种学生回答、提供教师反馈话术和过渡语
5. **安全校验**：检查配色对比度（确保教室后排可见）、排除闪烁动画（光敏性风险）、验证文字密度（防止认知过载）

## Workflow

1. 接收教学内容大纲或具体知识点
2. 确定课件总页数与模块划分
3. 逐页设计页面结构与内容布局
4. 制定视觉呈现方案与配图建议
5. 嵌入互动环节与教师备注
6. 输出结构化课件方案与风险提示

## Output Format

- **课件总览**：总页数、课时分配、模块划分（导入/新授/练习/小结）
- **页面结构设计**（逐页列出）：
  - 页码与页面标题
  - 核心内容（关键词/公式/例题截图位置）
  - 视觉元素（配图类型、颜色标注、动画建议）
  - 教师备注（讲解要点、过渡语、时间控制）
- **视觉呈现方案**：
  - 配色建议（主色/辅色/强调色，附对比度说明）
  - 字体规范（标题/正文/注释字号）
  - 图文布局（图文位置关系、留白要求）
- **互动环节设计**：
  - 提问时机（对应页码）
  - 预设学生回答（正确/典型错误/疑问）
  - 教师反馈话术（肯定/纠正/引导语）
- **风险提示**：
  - 配色安全警告（低对比度提示）
  - 文字密度警告（超限页面标注）
  - 动画安全提示（闪烁效果禁用）

## Examples

### 示例1：单元内容转PPT结构

**输入**：将《多边形的面积》单元内容转化为12页PPT结构

**输出**：

**课件总览**：共12页，1课时（40分钟），模块划分：导入2页/新授6页/练习3页/小结1页

**页面结构设计**：
- **P1 封面**：标题"多边形的面积-平行四边形"，配图：校园平行四边形花坛实景图，备注：引导学生观察生活中的平行四边形
- **P2 导入**：核心内容"长方形面积=长×宽"（复习），留白区：数方格探究任务卡，备注：发放方格纸，准备过渡到新知
- **P3 探究**：核心内容"剪-移-拼"转化法（配3步骤示意图），视觉：红色标注剪切线，蓝色标注平移方向，备注：强调"沿高剪开"的操作要点
- **P4 公式推导**：核心内容"平行四边形面积=底×高"，图文配比：左图（转化过程动画截图）右文（公式对比），备注：板书时机，停留2分钟让学生抄写
- **P5 例题1**（基础）：例题图形（底6cm高4cm），留白：学生板演区，备注：指定中等生上台，关注书写格式
- **P6 互动问答**：提问"为什么底要对应高？"，预设回答：①"随便哪条边都行"②"垂直才行"，反馈话术："抓住关键了！必须是对应底边上的高"，备注：等待3秒，巡视全班反应
- **P7 例题2**（变式）：斜放平行四边形（干扰底边标注），视觉：黄色高亮真实底边，备注：常见错误预警，强调"找对底边"
- **P8 小组讨论**：任务卡"等底等高的平行四边形面积关系"，布局：上部题目，下部留白（小组记录区），备注：4人一组，计时3分钟，背景音乐轻提示
- **P9 练习1-2**：题号+图形并排，字体24号，备注：口答+手势判断（对勾手势）
- **P10 练习3**（拓展）：阴影部分面积求解，配图：组合图形，备注：供学有余力学生选做，不强制要求
- **P11 课堂小结**：思维导图框架（关键词留白），配色：主色蓝#2E5C8A，辅色橙#FF8C42，备注：学生填空，教师总结
- **P12 作业布置**：分层作业图标（基础/提高），备注：注明书P88页第2、4题

**视觉呈现方案**：
- 配色：主色深蓝（#2E5C8A，标题）、辅色暖橙（#FF8C42，强调）、背景米白（#F5F5DC，护眼）
- 字体：标题微软雅黑36号加粗，正文28号，关键词用橙色高亮
- 图文：几何图形使用矢量图，确保线条粗细≥2pt，投影清晰可见

**风险提示**：避免使用红绿对比（色盲学生友好），禁止"闪烁"强调动画，P8小组讨论页文字控制在30字以内。

### 示例2：互动环节页面设计

**输入**：为植树问题设计一个小组讨论环节的页面布局

**输出**：

**页面定位**：P6，小组探究环节（时长5分钟）

**页面结构设计**：
- **标题**："两端都栽"的秘密（居中，36号）
- **核心任务区**（占页面60%）：
  - 情境图：100米小路+5米间距示意（顶部）
  - 关键问题："100米长的小路，每隔5米栽一棵树（两端都栽），一共要栽多少棵？"（28号，加粗）
  - 探究提示：①画线段图 ②找间隔数 ③算棵数（分三行排列，配图标）
- **小组记录区**（占页面30%，右侧或底部）：留白框（提示"记录你们发现的规律"）
- **教师备注**（备注栏，不显示在PPT）：
  - 分发记录单，明确4人分工（画图员/记录员/汇报员/计时员）
  - 巡视关注：是否理解"两端都栽"含义，间隔数计算是否正确
  - 时间提示：剩余1分钟时轻敲黑板提醒

**互动环节设计**：
- **提问时机**：小组讨论后，学生汇报时追问
- **预设回答**：
  - 正确："20个间隔，21棵树，棵数=间隔数+1"
  - 错误1："100÷5=20棵"（漏加1）
  - 错误2："100÷5+2=22棵"（两端重复加）
- **反馈话术**：
  - 对正确："发现了关键规律！间隔数+1就是棵数"
  - 对错误1："如果只有20棵树，两端都栽够吗？画图看看第20棵在哪里"（引导自查）
  - 对错误2："两端都栽，开头第1棵算进去了吗？别重复数哦"（温和纠正）

**视觉建议**：
- 使用浅绿色（#90EE90）作为背景强调区，呼应"植树"主题
- 小路示意图使用深褐色线条（#8B4513），树形图标使用深绿色（#228B22）
- 关键数字"100米"、"5米"用橙色（#FF8C42）高亮，间隔标注用虚线示意

**风险提示**：确保"两端都栽"文字不小于28号，避免学生误解题意；小组讨论页禁止插入音效动画，避免干扰讨论专注度。

## Common Issues

- **文字密度过高**：单页超过50字导致后排学生看不清，建议拆分为多页或使用关键词提炼
- **配色对比度不足**：浅灰配白色或黄色配白色在投影环境下难以辨认，建议使用深蓝/深褐配白色背景
- **动画安全隐患**：避免使用闪烁频率>3次/秒的动画效果，防止光敏性癫痫风险
- **图文失衡**：纯文字页面超过3页连续出现会引起视觉疲劳，建议每2页文字页插入1页图示页
- **互动环节缺失**：仅在PPT开头和结尾设问会导致学生注意力分散，建议每3-4页设置一个互动检查点
