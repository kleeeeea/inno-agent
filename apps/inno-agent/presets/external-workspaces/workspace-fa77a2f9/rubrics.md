# 小学科学课堂小组分工助手 评分量规

## 评分维度总览

本量规用于评估「小学科学课堂小组分工助手」在实际使用场景中的表现质量，共包含 5 个核心维度，每个维度满分 4 分，总分 20 分。

---

## 维度一：情景识别准确性

**评估重点**：助手能否准确判断教师描述中的年级段、矛盾类型与紧迫程度，并在回应中明确说明。

| 分值 | 表现描述 |
|------|----------|
| **4 分** | 准确识别年级段（低年级 / 中高年级）、矛盾类型（抢组长 / 推组长 / 强势霸位 / 沉默边缘化等）及紧迫程度，并在情景判断环节清晰呈现，无误判或遗漏 |
| **3 分** | 识别基本准确，偶有细节模糊（如年级段判断略有偏差），但不影响整体方案方向 |
| **2 分** | 识别部分正确，存在明显遗漏（如未区分年级段，或混淆矛盾类型），导致方案针对性下降 |
| **1 分** | 情景识别错误或完全缺失，直接给出与实际情景不符的通用建议 |

---

## 维度二：方案的可操作性与即时性

**评估重点**：所提供的解决步骤是否具体、分步清晰，教师能否在课中或课后立即执行，无需额外转化。

| 分值 | 表现描述 |
|------|----------|
| **4 分** | 方案分步骤呈现，每步均有明确动作指令（如"宣布规则""巡视时主动询问"），教师可直接照做；课中紧急情景与课后预防情景区分处理 |
| **3 分** | 步骤较清晰，大部分建议可直接执行，少数环节略显笼统但不影响整体落地 |
| **2 分** | 方案方向正确但步骤模糊，教师需自行补充大量细节才能执行；或仅给出原则性建议而无具体动作 |
| **1 分** | 方案过于抽象或理论化，缺乏可执行步骤，实际课堂中难以直接应用 |

---

## 维度三：年级差异化适配度

**评估重点**：方案是否根据低年级（1—2年级）与中高年级（3—6年级）的认知发展特点，在语言表达、角色设计、引导策略上做出有效区分。

| 分值 | 表现描述 |
|------|----------|
| **4 分** | 明确区分低年级与中高年级策略：低年级使用具象指令、固定角色、示范引导；中高年级鼓励自主协商、角色轮换、责任理解；语言风格与执行方式均有对应调整 |
| **3 分** | 有年级差异意识，主要策略有所区分，但部分细节（如语言指令具象化）未能充分体现低年级特点 |
| **2 分** | 年级差异化处理较弱，方案基本通用，仅在表述上略有提及年级差异，未落实到具体操作层面 |
| **1 分** | 完全忽视年级差异，对所有年级给出相同建议，不符合小学生认知发展规律 |

---

## 维度四：情绪安全与教育伦理保障

**评估重点**：方案是否始终保护学生的情绪安全，避免公开批评、贴标签、偏袒等行为；是否将教师定位为引导者而非裁判者。

| 分值 | 表现描述 |
|------|----------|
| **4 分** | 方案全程体现情绪安全原则：强势学生课后私聊而非当众批评，边缘化学生获得低压力切入角色，教师注意事项中明确列出"不该做什么"；无任何可能伤害学生自尊的建议 |
| **3 分** | 整体注重情绪安全，偶有建议略显强硬（如直接公开点名），但整体不构成明显伤害风险 |
| **2 分** | 部分建议存在当众批评、强制指定等做法，可能对学生造成负面情绪影响；教师角色定位偏向裁判者 |
| **1 分** | 方案忽视情绪安全，存在明显不当建议（如公开羞辱、强制惩罚），违背教育伦理底线 |

---

## 维度五：预防性建议的前瞻性与系统性

**评估重点**：在解决当前矛盾之外，助手是否提供有效的预防建议，帮助教师在后续课堂中主动规避同类问题，体现系统性思维。

| 分值 | 表现描述 |
|------|----------|
| **4 分** | 提供具体、可落地的预防建议（如提前公示轮次表、建立角色轮换制度、课前说明规则），建议与当前矛盾类型高度相关，具有长期适用性 |
| **3 分** | 有预防建议且基本合理，但部分建议较为通用，与具体矛盾类型的关联性有待加强 |
| **2 分** | 预防建议简短或流于形式，仅作简单提及，缺乏可操作的具体措施 |
| **1 分** | 无预防建议，或建议与当前情景完全无关，对教师后续课堂设计无实质帮助 |

---

## 总分说明

| 总分区间 | 综合评价 |
|----------|----------|
| **18—20 分** | 优秀：方案精准、可操作、有温度，充分体现分年级差异化与情绪安全原则，是高质量的课堂参谋支持 |
| **14—17 分** | 良好：方案整体有效，个别维度有提升空间，能满足教师大多数实际需求 |
| **10—13 分** | 一般：方案方向基本正确，但可操作性或年级适配性不足，教师需自行补充较多细节 |
| **6—9 分** | 较差：方案存在明显缺陷，情景识别或情绪安全方面有突出问题，实用性
