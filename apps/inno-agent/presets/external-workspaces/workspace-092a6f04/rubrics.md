# 本土化项目化学习设计助手 评分量规

## 说明

本量规用于评估"本土化项目化学习设计助手"的输出质量，共包含 5 个评分维度，每个维度满分 4 分，总分 20 分。评分者应结合乡村小学真实教学场景进行判断。

---

## 维度一：本土资源适切性

> 评估方案是否真正立足本土自然与文化资源，所有材料是否满足零成本或极低成本要求。

| 分值 | 描述 |
|------|------|
| **4 — 优秀** | 所有材料均为本土可获取的免费/废旧材料（泥土、树叶、石子、塑料瓶等），无任何需采购的专业器材；驱动问题直接来源于学生可感知的本地真实情境（家乡植物、水源、传统制作等），与学生日常生活高度相关。 |
| **3 — 良好** | 绝大多数材料免费可获取，偶有 1 处建议使用低价易得物品但未超出合理范围；驱动问题基本贴近本土情境，但与学生生活的关联稍显间接。 |
| **2 — 一般** | 部分材料需要购买或难以在乡村获取；驱动问题较为通用，缺乏明显的本土特色，可在任何学校套用。 |
| **1 — 不足** | 推荐了专业实验器材或需要采购的材料；驱动问题与本土资源无关，完全脱离乡村小学实际场景。 |

---

## 维度二：微任务可执行性

> 评估微任务拆解是否合理，每个微任务能否在单课时（40 分钟）内由无项目经验的小学生独立完成。

| 分值 | 描述 |
|------|------|
| **4 — 优秀** | 提供 4–6 个微任务，每个任务目标单一、操作步骤清晰具体；每个微任务均可在 40 分钟内完成，且为无项目经验的学生提供了足够细的操作指引；任务间逻辑递进，整体项目结构完整。 |
| **3 — 良好** | 微任务数量合理，大多数任务目标明确、步骤清楚；个别任务操作指引略显简略，或时间安排稍显紧张，但整体仍可执行。 |
| **2 — 一般** | 微任务数量不足（少于 4 个）或过多（超过 6 个）；部分任务目标模糊或步骤缺失，难以在单课时内完成；对无项目经验学生的支持不足。 |
| **1 — 不足** | 微任务拆解混乱，任务间缺乏逻辑关联；操作说明缺失或过于抽象，普通教室内无法实施。 |

---

## 维度三：结构化记录与分工方案完整性

> 评估记录单模板和小组分工方案是否实用、完整，能否确保每位学生都有明确职责。

| 分值 | 描述 |
|------|------|
| **4 — 优秀** | 提供至少 1 个可直接使用的结构化记录单模板，包含观察框、数据格、结论栏等必要要素；小组分工明确覆盖材料员、记录员、记录保管员、汇报员 4 类角色，每类角色职责说明具体，确保无旁观者；记录单与微任务内容高度匹配。 |
| **3 — 良好** | 记录单模板基本完整，包含主要要素，但个别栏目设计略显粗糙；4 类角色均有提及，职责说明基本清晰，偶有笼统之处。 |
| **2 — 一般** | 记录单模板要素不完整（缺少观察框或结论栏等关键部分）；角色分工未完整覆盖 4 类，或职责说明过于简略，存在学生无事可做的风险。 |
| **1 — 不足** | 未提供记录单模板，或模板无法直接使用；角色分工缺失或仅做形式性提及，无实质性职责说明。 |

---

## 维度四：过程性评价方案质量

> 评估评价方案是否包含完整的三级过程性评价，评价目的是否指向促进学习而非单纯打分。

| 分值 | 描述 |
|------|------|
| **4 — 优秀** | 完整包含自评、互评、小组验收三个层级，每级均有具体可操作的评价标准；评价标准聚焦学习过程与技能成长，而非仅评最终作品；设计了"达标后进入下一微任务"的学习闭环机制，评价与教学流程紧密衔接。 |
| **3 — 良好** | 包含三级评价结构，评价标准基本清晰；大部分标准关注过程性维度，但个别标准仍偏重结果评价；学习闭环机制有所体现但不够完整。 |
| **2 — 一般** | 三级评价结构不完整（缺少其中一级）；评价标准以最终作品为主，过程性维度薄弱；评价与教学流程关联不紧密。 |
| **1 — 不足** | 仅提供终结性评价或完全缺失评价方案；评价标准缺失或无法操作；未体现促进学习的评价理念。 |

---

## 维度五：信息收集与追问策略

> 评估助手在教师描述不完整时，能否精准识别需求、合理追问，并在信息充分时输出结构化方案。

| 分值 | 描述 |
|------|------|
| **4 — 优秀** | 准确识别教师需求类型（设计项目 / 课堂支持 / 两者兼有），并按正确流程响应；信息不足时每次追问不超过 2 个问题，且优先追问高优先级信息（年级学段、本地资源）；信息充分后主动输出完整结构化方案，并询问是否需要进一步调整。 |
| **3 — 良好** | 需求识别基本准确，流程选择正确；追问数量基本合理，偶有超过 2 个问题或追问优先级不
