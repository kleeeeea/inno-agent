# 中职家校冲突调解专家评估标准

## 评估维度与权重

### 1. 合规性（30%）
**核心要求**：严格遵守法律法规，保护隐私，程序合法

| 评分等级 | 标准描述 |
|---------|---------|
| 5分（优秀） | 全程无法律风险，隐私保护措施完善，引用法规准确（《未成年人保护法》《个人信息保护法》《职业学校学生实习管理规定》《中小学教育惩戒规则（试行）》等） |
| 4分（良好） | 基本合规，偶有轻微瑕疵，无重大法律风险 |
| 3分（合格） | 整体合规，但存在模糊地带，如未明确告知隐私保护边界 |
| 2分（较差） | 存在潜在违规风险，如建议调取敏感信息、越权承诺 |
| 1分（不合格） | 明显违反法律法规，如建议查看私人聊天记录、承诺撤销处分等越权行为 |

**扣分项**：
- 建议调取通话记录、私人聊天内容（扣10分）
- 承诺班主任无权决定的事项（改成绩、撤处分、换企业）（扣10分）
- 泄露学生隐私给无关家长（扣10分）
- 未告知隐私保护边界（扣5分）

### 2. 场景适配性（25%）
**核心要求**：精准识别中职家校冲突特点，细节到位

| 评分等级 | 标准描述 |
|---------|---------|
| 5分（优秀） | 准确识别实习管理/手机管理/违纪处分等场景的中职特殊性，提及三方协议、申述期、岗位对口性等关键细节 |
| 4分（良好） | 场景识别准确，细节较全，偶有遗漏 |
| 3分（合格） | 能区分场景类型，但细节泛泛，未体现中职特点（如与普通中小学混淆） |
| 2分（较差） | 场景识别错误，或细节严重不符（如将顶岗实习等同于就业） |
| 1分（不合格） | 完全偏离场景，建议不相关 |

**关键细节检查点**：
- 实习场景：是否区分顶岗/跟岗、提及保险、三方协议、工时规定
- 手机场景：是否提及教学时段界定、应急联系流程
- 违纪场景：是否提及申述程序、告知义务、班主任权限边界

### 3. 话术安全性（20%）
**核心要求**：教师自我保护，无越权承诺，可安全使用

| 评分等级 | 标准描述 |
|---------|---------|
| 5分（优秀） | 每句话都有法规/校规依据，明确权限边界，提供"上报阻断话术"保护教师 |
| 4分（良好） | 话术整体安全，个别表述可更严谨 |
| 3分（合格） | 无明显越权，但存在模糊承诺（如"尽量帮您争取"） |
| 2分（较差） | 存在越权暗示（如"我跟领导说说看"），或缺乏教师保护机制 |
| 1分（不合格） | 直接承诺越权事项，或让教师单独面对攻击 |

**必含要素**：
- 明确告知"24小时内答复"（避免立即承诺）
- 提供上报话术（超出权限时的脱身方案）
- 注明依据（校规第X条/法规X条）

### 4. 调解有效性（15%）
**核心要求**：情绪疏导到位，事实核查路径清晰，方案可落地

| 评分等级 | 标准描述 |
|---------|---------|
| 5分（优秀） | 共情精准（30秒内降低对抗），核查清单具体（查哪张表、联系谁），方案分情境（学校有责/家长误解/需第三方） |
| 4分（良好） | 共情到位，核查路径较清，有基本分支策略 |
| 3分（合格） | 有共情尝试，但核查路径模糊，方案单一 |
| 2分（较差） | 共情生硬（如"规定就是这样"），无有效核查方案 |
| 1分（不合格） | 无共情，直接对抗或妥协 |

**检查点**：
- 是否提供前3句即时应对脚本（共情+核实承诺）
- 是否提供具体核查清单（校内记录名称）
- 是否区分情境A/B/C给出不同策略

### 5. 执行可行性（10%）
**核心要求**：流程清晰，时限明确，资源可及

| 评分等级 | 标准描述 |
|---------|---------|
| 5分（优秀） | 步骤编号清晰（第一步/第二步），时限明确（24小时/3日），所需资源在校内可获取 |
| 4分（良好） | 流程较清，时限基本明确 |
| 3分（合格） | 有步骤但顺序不清，时限模糊（如"尽快"） |
| 2分（较差） | 步骤混乱，或要求不可获取的资源 |
| 1分（不合格） | 无法执行，或需要外部特殊资源 |

## 综合评级

| 总分 | 评级 | 说明 |
|-----|-----|-----|
| 90-100 | A+ | 可直接投入使用 |
| 80-89 | A | 优秀，轻微优化即可 |
| 70-79 | B | 良好，需针对性调整 |
| 60-69 | C | 合格，需较大改进 |
| 60以下 | D | 不合格，需重新设计 |

## 评估示例

**优秀案例特征**：
- 提供3句话以内的话术模板（先共情、再事实、后底线）
- 明确列出"查看校内考勤表、联系实习指导教师、依据《职业学校学生实习管理规定》第16条"
- 区分"学校有责（立即整改）/家长误解（出示记录）/需上报（24小时内专人回复）"
- 包含上报话术："此事超出我权限，已上报德育处，24小时内专人联系您"

**不合格案例特征**：
- 建议"查看学生微信聊天记录了解真相"
- 承诺"我帮您把处分撤销了"
- 回应"规定就是这样，我也没办法"
- 未提供任何核查路径或时限承诺
