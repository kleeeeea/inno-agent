# 家校沟通工具生成

## When to use

- 撰写校长/班主任家长公开信（单理念聚焦，800字以内，便于传播消化）
- 设计分层家访方案（入户/视频/电话的选择标准、开场话术、观察清单）
- 制作家长会组织脚本（含家长分享环节、分组讨论流程、时间分配）
- 创建家长阅读/写作活动模板（低门槛启动，如"每天两句话日记""五分钟阅读"）
- 设计"孩子影响家长"的安全调查与反馈工具（学生视角传递需求，避免家长反感）

## When not to use

- 需要设计学期整体路线图、制度体系、教师激励机制或评价复盘框架 → 应使用 **family-school-mechanism-design**
- 涉及学生心理危机、家庭暴力、虐待忽视 → 立即转介心理教师、德育部门或法律机构，不提供沟通工具
- 家访发现严重安全隐患或家长情绪失控/精神异常 → 建议立即上报，停止工具生成
- 涉及教育惩戒边界争议或法律诉讼风险 → 建议先确认合规性，不直接生成沟通话术

## Instructions

- **单理念聚焦**：每封信、每次活动只传递一个核心概念，避免信息过载
- **低门槛设计**：为抵触/忙碌家长提供"最小行动单元"（如两句话、五分钟、不打卡）
- **正向激励语言**：采用"关注-理解-期望-具体行为"结构，避免指责与告状口吻
- **隐私与尊严**：家访观察聚焦环境改善而非人格评价，避免羞辱性标签，遵循信息最小化原则
- **模板可复用**：提供含[此处填写]占位符的完整框架，标注使用场景与替代方案

## Workflow

1. **需求识别**：确认工具类型（信/家访/家长会/阅读写作/调查）与目标家长群体（高配合/抵触/忙碌）
2. **策略匹配**：选择激活策略（降低门槛/同伴影响/孩子反向带动/书信影响）
3. **模板生成**：输出结构化模板（话术/信稿/流程/示例），标注关键节点与风险规避点
4. **安全审查**：排查高风险情境（暴力/心理危机/隐私侵犯），必要时建议转介
5. **执行清单**：提供"准备-实施-跟进"三步检查清单，确保工具可立即使用

## Output Format

- **场景判断**：目标家长特征、当前抵触点与核心痛点
- **工具类型**：家长公开信/分层家访方案/家长会脚本/阅读写作模板/学生调查工具
- **核心策略**：单理念传播/低门槛启动/同伴教育/孩子反向带动
- **具体模板**：完整话术/信稿框架（800字内）/流程脚本/观察点清单（含[占位符]与填写指引）
- **分层适配**：高配合家长深化方案 vs 抵触/忙碌家长简化方案
- **风险提示**：隐私边界、法律合规（家庭教育促进法）、情绪冲突预警
- **执行检查清单**：准备事项（材料/时间/场地）- 执行步骤（开场-主体-结束）- 跟进要点（反馈收集/故事沉淀）

## Examples

- **家长公开信**：新班主任首封信聚焦"我们是一伙的"，附单理念选择指南（三选一：关系优先/习惯养成/错误是学习机会）与完整信稿框架
- **分层家访**：入户家访开场话术（"我是来向您了解孩子在家闪光点的"）、观察点清单（学习空间/整理习惯/家庭互动）、结束建议模板（"可以尝试的一个小微调"）
- **家长会组织**：60分钟流程脚本（教师10分钟+家长分享20分钟+分组讨论20分钟+共识10分钟），含家长讲师邀请话术与突发冷场应对
- **阅读激活**："每天五分钟"启动方案，附书单规则（薄册子/故事性/正向结尾）与"不打卡只分享"的自愿交流话术
- **两句话日记**："看见-感受-期待-行动"四要素写作框架，附3个示例（鼓励型/修复型/期待型）与"写不出来时的备用句式"

## Common Issues

- **家长抵触"形式主义"**：避免"截图留痕""表格打卡"要求，强调自愿分享与故事传播，降低造假动机
- **教师负担顾虑**：提供"一次制作多次复用"的模板，明确单次工具准备时间不超过20分钟，复杂活动提供简化版
- **隐私边界模糊**：家访不评价家庭生活方式（如"你们太乱了"），聚焦可改进的具体行为（"给孩子一个固定书桌"）
- **难以持续**：每个工具配套"四周微习惯"跟进建议（第一周示范/第二周提醒/第三周展示/第四周沉淀），避免一次性热闹
