# 家校共育机制设计

## When to use
- 设计学期/学年家校共育路线图与制度化安排（频率、角色分工、产出物、传播渠道）
- 制定替代打卡考核的教师正向激励机制（故事传播、尊严感驱动、案例汇编）
- 构建"家长影响家长"的同伴教育生态与可持续运行制度
- 建立共育效果评价指标与学期复盘框架（量化+质性指标）
- 校级层面启动方案与家校共识建立（启动阶段制度设计）

## When not to use
- 需要具体家长公开信模板、家访话术脚本、家长会流程、阅读写作活动模板 → 转交 **family-school-toolkit-generator**
- 涉及学生心理危机、家庭暴力、虐待忽视、家长精神异常或法律诉讼风险 → 立即建议转介心理教师、德育部门或法律机构，不提供机制设计方案
- 需要学科知识辅导或具体教学指导 → 拒绝并说明聚焦家校关系与育人机制

## Instructions
1. **反形式主义原则**：拒绝设计"表格打卡、截图留痕、硬性次数指标"等易造假考核，坚持"正向激励+故事传播+可见成果"机制
2. **可持续导向**：每个方案必须包含"如何避免一阵风"的制度固化建议（频率、角色分工、产出物、传播渠道）
3. **分层适配**：区分高配合度家长（进入深水区做示范）与抵触/忙碌家长（低门槛入口+孩子反馈触发）的差异化策略
4. **法律边界**：严格遵守《家庭教育促进法》《未成年人保护法》，家访与信息收集遵循最小化原则，避免羞辱性评价

## Workflow
1. **角色与情境诊断**：确认用户身份（校长/德育主任/班主任）、学段（小学/初中/高中）、当前痛点（参与度低/形式化/冲突/教师抵触）
2. **机制框架设计**：确定共育三件事（提升教师专业尊严/影响家长方法行动/关爱孩子关系成长），设计"五件套"制度（阅读/写作/同伴影响/公开信/孩子反向带动）
3. **激励体系构建**：用"故事感+尊严感+专业发展机会"替代硬指标，设计教师分享会、案例汇编、示范样本传播机制
4. **节奏与固化**：制定学期路线图（启动-深化-固化三阶段），明确月度动作、产出物标准、复盘节点
5. **风险筛查**：识别教师集体抵触（劳动权益纠纷）、家长法律风险、教育惩戒边界争议，触发安全指引或转介

## Output Format
- **场景判断**：当前学校/班级的家校关系状态（紧张/形式化/缺位）、核心痛点与适用学段
- **处理方案**：
  - 核心目标（3个以内，符合"影响-激活-带动"逻辑）
  - 机制设计（四项核心制度：频率、角色分工、产出物、传播渠道）
  - 学期路线图（月度节奏表，标注关键里程碑）
  - 激励体系（正向激励替代考核的具体方式与教师动员策略）
- **可用话术**：家校共识建立的关键表达、教师动员的激励语言、边界说明话术
- **风险提示**：可能的形式主义反弹、法律边界、高风险情境识别、教师抵触预警
- **后续评估**：量化指标（参与率、冲突率、家访覆盖率）与质性指标（案例沉淀、故事传播、环境改善）

## Examples
**输入**：我是校长，学校家校关系紧张、家长会到场率低。给我一个学期家校共育启动方案。

**输出要点**：
- **场景判断**：家校关系紧张型学校，需从"对立"转向"合伙人"认知，重点解决教师抵触与家长形式化参与
- **处理方案**：
  - **核心目标**：①建立"家庭教育是学校重要补充"共识 ②启动"家长影响家长"同伴机制 ③用故事分享替代表格考核
  - **四项机制**：①双周教师/家长共育故事分享会（产出：案例汇编） ②每月校长公开信（单理念聚焦，产出：理念传播度） ③班级"家长两句话写作"成果册（产出：实体/电子文集） ④分层家访观察制度（产出：家庭环境改善建议）
  - **学期节奏**：第1月共识建立（校长信+教师 pledge）→ 第2-3月种子寻找（写作+故事会）→ 第4-5月深度激活（家长会转型+入户观察）→ 第6月固化表彰（案例集+故事传播奖）
  - **激励体系**：设立"共育故事奖"（非强制参与），获奖案例纳入教师专业发展档案与职称评审附加材料
- **可用话术**："我们不考核次数，只收集故事"；"家校不是甲方乙方，是合伙人关系"
- **风险提示**：警惕教师将故事会视为额外负担（需计入教研工时），避免家访变成"家庭检查"（需强调平等谈心）
- **后续评估**：每月追踪家长写作提交率、故事会家长自发分享数、家访后家庭环境改善案例数

## Common Issues
- **教师抵触视为负担**：避免增加额外工作量，将共育成果与教师专业发展（课题申报、讲学机会、职称评审）挂钩，而非扣绩效或强制次数
- **家长参与度分层处理**：高配合度家长做"种子"进入深水区（深度阅读+经验分享），抵触/忙碌家长用"孩子反馈"触发（学生调查带回）+ 最低门槛写作（每日两句话）
- **线上替代入户误区**：制度设计必须明确视频/电话仅适合"通报情况"，入户家访核心价值是"平等谈心氛围+观察家庭环境"，保留入户比例要求（如每学期至少一次入户）
- **评价指标造假风险**：放弃"家访次数、表格数量"硬指标，改用"可展示成果"（家长写作册、案例故事、环境改善前后对比照片）作为评价依据，降低造假动机
