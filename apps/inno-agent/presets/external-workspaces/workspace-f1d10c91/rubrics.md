# 评分维度说明

本评分量表适用于评估「小学科学实验意外现象探究引导助手」的回应质量，共包含五个维度，每个维度满分4分，总分20分。

---

## 维度一：场景识别与信息确认准确性

**评估重点**：助手是否准确识别并确认了教师描述的核心场景要素（学段、实验名称与预设结论、实际意外现象、课堂状态），并在信息不足时进行了恰当追问。

| 分值 | 描述 |
|------|------|
| 4 | 完整识别全部关键场景要素，【场景确认】部分清晰准确地复述了学段、实验、意外现象及课堂状态；信息不足时以简洁方式追问，不过度打断教师求助节奏 |
| 3 | 识别了大部分场景要素，偶有遗漏但不影响后续建议的针对性；追问方式基本合理 |
| 2 | 场景识别存在明显遗漏或误解，导致后续建议与实际情境部分脱节；追问方式冗长或缺失 |
| 1 | 未进行有效场景识别，直接给出通用建议，与教师描述的具体情境严重不符 |

---

## 维度二：七环节应对方案的完整性与可操作性

**评估重点**：助手是否按照七个环节提供了结构化建议，每个环节是否给出了教师可直接使用的话术或操作指引，而非抽象原则。

| 分值 | 描述 |
|------|------|
| 4 | 七个环节均有覆盖，每个环节提供了具体、可直接照读或执行的话术/操作指引；【即时应对话术】简洁有力，适合在30秒内使用 |
| 3 | 覆盖了六个以上环节，大部分环节有具体话术，个别环节仅给出原则性描述；整体仍具较强可操作性 |
| 2 | 仅覆盖四至五个环节，或多个环节停留在抽象建议层面，缺乏可直接使用的话术；教师需要大量二次加工 |
| 1 | 环节覆盖不足四个，或内容高度抽象、缺乏实际指导价值，教师难以在课堂中直接应用 |

---

## 维度三：学段适配性

**评估重点**：助手是否根据教师提供的学段（低/中/高年级）调整了观察任务的复杂度、提问话术的难度以及课后任务的可行性，体现出对不同年龄段学生认知特点的理解。

| 分值 | 描述 |
|------|------|
| 4 | 观察引导语、讨论任务、推理要求、课后任务均与所述学段高度匹配；低年级聚焦感官描述，中年级注重证据收集，高年级强调逻辑推理链，差异化明显 |
| 3 | 整体符合学段特点，偶有话术或任务难度与学段略有偏差，但不影响整体适用性 |
| 2 | 学段适配意识薄弱，话术或任务设计明显偏难或偏易，与所述学段学生认知水平不符 |
| 1 | 完全忽视学段差异，提供通用模板，无任何针对性调整 |

---

## 维度四：探究导向与教师行为准则遵守情况

**评估重点**：助手是否始终坚持将意外现象转化为探究机会的核心立场，严格遵守教师行为准则——不编造答案、不指责学生操作、不掩盖意外现象、不直接给出结论，并在教师不知道原因时提供坦诚表达未知的话术。

| 分值 | 描述 |
|------|------|
| 4 | 全程坚持探究导向，无任何违反行为准则的建议；明确提供了坦诚表达未知的话术；将意外现象定性为有价值的发现，而非失败或错误 |
| 3 | 整体符合探究导向，偶有表述略显权威性或结论性，但未出现明确违反准则的建议 |
| 2 | 存在一至两处违反行为准则的建议（如暗示重做实验、给出确定性解释、将意外归咎于学生），或探究导向不够一贯 |
| 1 | 多处违反行为准则，或整体倾向于直接告知答案、掩盖意外，与助手核心使命相悖 |

---

## 维度五：输出格式规范性与教师使用友好度

**评估重点**：助手的输出是否符合规定格式（包含【场景确认】【即时应对话术】【完整应对流程】三个模块），结构是否清晰易读，教师在课堂紧急情境下能否快速定位所需内容。

| 分值 | 描述 |
|------|------|
| 4 | 严格包含三个规定模块，层次清晰，标题醒目；【即时应对话术】与【完整应对流程】明确区分，教师可在紧急情况下快速找到30秒内可用的开场语；整体排版简洁，无冗余信息 |
| 3 | 包含三个模块，结构基本清晰，偶有格式不规范或模块边界模糊，但不影响整体阅读 |
| 2 | 缺少一个规定模块，或模块内容混乱、层次不清，教师需要花费较多时间梳理才能使用 |
| 1 | 格式混乱，缺少两个以上规定模块，或以大段连续文字呈现，严重影响课堂紧急场景下的使用效率 |
