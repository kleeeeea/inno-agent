# 试讲状态与节奏优化

## When to use

- **即时状态调控**：上场前60秒内需要快速拉起精神面貌（声音虚、眼神飘、身体塌）
- **节奏失控干预**：语速过快、句间无停顿、信息密不透风导致考官疲劳
- **表情动作僵硬**：需要替代"全程硬笑/机械手势"的自然化呈现方案
- **临场突发应对**：中途卡壳、节奏崩坏时的自救话术与控场动作

## When not to use

- **训练体系搭建**：若用户需要"手机自录对照方法、量化评分表、录-评-改闭环设计"，转交 **self-training-loop**
- **教学设计咨询**：涉及具体学科知识点讲解、教案设计、环节创新等"内容层"问题，使用普通问答
- **高风险场景**：用户描述严重焦虑躯体化（手抖失控、呼吸困难）或寻求违规操作建议（误导考官），触发安全提示并建议专业干预

## Instructions

1. **先像老师，再讲内容**：优先校准状态与节奏，再谈教学设计
2. **量化可执行**：所有指令必须落到身体动作（如"句末停0.5秒"、"首句音量提高至胸腔有明显振动感"）
3. **默认位原则**：设定手部/表情的"静止基准线"（腹前自然相叠/眼神温和+嘴角微提），避免全程硬撑
4. **锚点触发**：区分"全程保持"与"关键节点触发"（仅问候、肯定、总结时微笑）

## Workflow

1. **场景识别**：判断用户处于"上场前即时/自主练习期/考中突发"哪个阶段
2. **问题归类**：归入状态低迷/节奏过快/动作僵硬/表情困难四类之一
3. **方案生成**：输出≤60秒可执行的量化脚本（含身体指令、锚点规则、默认位设定）
4. **安全校验**：检查是否存在过度训练或焦虑躯体化迹象，必要时插入风险提示

## Output Format

```markdown
**场景判断**：[上场前60秒/开口前10秒/中途卡壳/全程调控]

**即时调控脚本**（对应Rubric：状态调控可操作性）：
- 站姿/眼神：[具体动作]
- 气息/音量：[量化指令，含身体参照]
- 启动口令：[首句话术示例]

**节奏锚点规则**（对应Rubric：节奏控制可量化）：
- 句间停顿：[0.3-0.8秒硬规则]
- 段落切分：[讲完概念/提问/板书后停1秒]
- 语速档位：[紧张时启用"刹车机制"话术]

**表情动作指令**（对应Rubric：表情动作自然度）：
- 默认位：[手部位置/表情基准线]
- 允许动作：[列举/对比/强调三类手势描述]
- 触发节点：[微笑/手势出现的具体时机]

**风险提示**（场景化）：
- 考前场景：[失眠/食欲减退等信号及应对]
- 练习场景：[嗓子刺痛/肌肉酸痛等信号及应对]
- 考后场景：[失声/心理倦怠等信号及应对]

**唯一改进点**（对应Rubric：训练闭环与自检工具）：[本轮只改一项，避免信息过载]
```

## Examples

**示例1：上场前60秒状态拉起**
```markdown
**场景判断**：上场前60秒，声音虚、身体塌

**即时调控脚本**：
- 站姿：双脚与肩同宽，重心略前倾（激活状态）
- 眼神：平视考官眉心连线中点，不飘不垂
- 气息：深吸4秒-屏息2秒-慢吐6秒（1轮）
- 音量基线：首句"各位考官好"提高至能让最后一排听清的力度（感受胸腔振动），落点在最后一个字

**节奏锚点规则**：
- 问候后停0.8秒再开始试讲内容（给考官反应时间）
- 每句话尾字落下后默数"1"（约0.5秒）再启下句

**表情动作指令**：
- 默认位：双手腹前自然相叠（非捂肚脐，距腹部一拳距离）
- 触发节点：仅在说完"以上是我的试讲"时嘴角微提15度（肌肉记忆点）

**风险提示**：
- 练习场景：如试嗓时发现高音破裂或刺痛，立即停止发声，改为默念练习
```

**示例2：中途卡壳自救**
```markdown
**场景判断**：讲到一半突然忘词，节奏崩坏

**即时调控脚本**：
- 动作：自然侧移半步（制造" intentional pause"错觉）
- 眼神：保持看向考官，不躲闪不翻白眼
- 话术："请同学们观察黑板上的关键词/我们把这个概念再强调一遍"（争取3秒缓冲）

**节奏锚点规则**：
- 卡壳后重启首句必须慢50%，句末停0.8秒重建控制感

**唯一改进点**：下次练习时预设2个"缓冲话术"作为安全网

**风险提示**：
- 考后场景：若考后出现持续失声或心理倦怠，暂停所有复盘，优先休息48小时
```

## Common Issues

- **过度手势**：删除"切菜式"、"上楼梯式"等培训模板，仅保留"列举（数手指）、对比（双手分展）、强调（单手下切）"三类
- **全程硬笑**：用"眼神温和+嘴角微提"替代大幅微笑，将微笑锚定在"问候、肯定学生、总结"三个固定位置，避免分散认知资源
- **停顿恐惧**：明确告知"停顿显示掌控力而非忘词"，0.5秒停顿在考官感知中仅为自然呼吸，不会显得拖沓
