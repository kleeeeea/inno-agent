# 试讲自训闭环工具

## When to use

- 用户身边缺乏懂试讲的反馈者（同学/老师/家人无法提供有效评价），需要建立客观自我评估标准
- 用户需要设计可重复的训练闭环（录制→评估→改进→验证），形成迭代提升机制
- 用户需要量化评分表、自检清单或复盘记录格式，以追踪多轮训练进展
- 用户需要制定周期性训练计划（如考前7天/30天迭代方案），合理分配改进优先级
- 用户需要手机自录的具体操作指南（机位、取景、光线、时长）以获取有效训练素材

## When not to use

- 用户询问具体的状态调控技术（如句间停顿怎么停、微笑怎么做、站姿怎么调整、卡壳如何自救）→ 应路由至 **demo-state-rhythm-optimizer**
- 用户需要上场前即时调控方案（60秒状态拉起、开口前10秒准备）→ 应路由至 **demo-state-rhythm-optimizer**
- 用户出现严重焦虑躯体化症状（手抖无法控制、呼吸困难、失眠严重）→ 建议暂停训练并寻求专业心理支持，不提供训练方案
- 用户计划过度训练（如每天录制20遍以上、嗓子已疼痛仍要求高强度发声训练）→ 插入安全提示并建议终止或减量，不提供强化方案
- 用户要求提供违规操作建议（如如何伪装教学经验欺骗考官、替考技巧）→ 拒绝并终止对话

## Instructions

- 提供手机自录的**可操作参数**（机位高度与眼平齐、距离2-3米、全身入镜含板书区、横屏、光线充足），确保视频可用于状态/动作分析
- 设计**四维量化评分表**（状态、节奏、表情、动作），每项配具体可观察指标（如"句末是否有0.5秒停顿"而非"节奏感好不好"），总分20分制
- 坚持**单点突破原则**：每轮训练只改1个最突出问题（评分最低维度），避免同时调整多个变量导致无法归因
- 建立**最小闭环**：录制3-5分钟关键片段 → 自评打分 → 定位唯一改进点 → 针对性微练习 → 重录验证对比
- 提供**复盘记录模板**：包含日期、本轮改进点、四维分数变化、下一轮目标，形成可视化进步曲线

## Workflow

1. **环境诊断**：确认用户训练环境（独居/宿舍/教室）、可用设备（手机/支架/书本替代）、时间约束，排除过度训练风险
2. **录制配置**：指导设置手机参数（高度、距离、取景、光线），建议录制完整10分钟或关键片段（开场3分钟/互动环节）
3. **基线建立**：完成首轮录制，使用评分表进行四维评估，确定当前基准分数与唯一最低分项
4. **对照分析**：如提供示例视频，指导进行"状态-节奏-动作"三维度对比，标记具体差距时间点；如无示例，使用文字版优秀特征清单
5. **方案生成**：针对唯一改进点设计微练习（如节奏差则练习"每句末默数1秒"），明确下轮录制验证标准
6. **迭代规划**：填写复盘记录表，设定下一轮训练目标（达到4分/5分即晋升下一维度），建议每日录制不超过3-5遍

## Output Format

必须包含以下固定字段：

- **场景判断**：当前训练环境、设备条件、可用时间、经验水平
- **录制方案**：机位设置（高度/距离/角度）、取景范围、光线要求、建议时长（完整/片段）
- **评分量表**：四维评分表（状态/节奏/表情/动作）+ 具体观察点 + 总分计算方式 + 等级划分（15+进阶/12-重修状态）
- **本轮唯一改进点**：基于评分结果确定的具体、可量化改进目标（如"句末停顿0.5秒"而非"注意节奏"）
- **对照方法**：与优秀示例或上一轮视频的对比分析步骤（同步播放时间点标记法或清单对照法）
- **复盘记录表**：结构化表格（日期/改进点/四维分数/总分/下一轮目标）
- **风险提示**：过度训练预警（每日≤5遍）、声带保护提醒、心理疲劳识别
- **迭代建议**：下一轮训练重点提示或终止条件（达到目标分数/考试日期临近切换模式）

## Examples

**用户输入**："我身边没人懂试讲，只有手机。你给我一个'自录—对照—打分—改一条'的训练闭环和评分表。"

**输出示例**：

- **场景判断**：独居备考，仅手机无支架，缺乏外部反馈，需建立自我迭代的客观评价标准
- **录制方案**：手机用书堆叠至站立时眼睛高度，置于正前方2-3米，横屏录制确保全身（含板书区域）入镜，避免逆光，每次录制开场3分钟片段（节约复盘时间）
- **评分量表**：
  - 状态（眼神/音量/站姿）：1-5分，观察点：首句音量是否饱满、眼神是否定焦前方、有无塌肩驼背
  - 节奏（停顿/语速）：1-5分，观察点：句末是否有0.3-0.8秒停顿、段落间是否有呼吸感、是否连珠炮
  - 表情（自然度/微笑）：1-5分，观察点：面部肌肉是否放松、微笑是否刻意僵硬、有无表情突变
  - 动作（手部/走位）：1-5分，观察点：手部是否始终捂腹/背手、手势是否过多杂乱、移动是否自然
  - 总分：20分，15分以上进入细节打磨，12分以下优先处理状态维度
- **本轮唯一改进点**：节奏维度得分最低（3分），确定改进目标为"在每个提问后强制停顿1秒（默数1001）"
- **对照方法**：如提供示范视频，使用"静音对比法"：同时播放自己与示范视频，仅观察画面节奏（点头/换气/手势停顿点），标记差异时间点
- **复盘记录表**：
  | 日期 | 本轮改进点 | 状态 | 节奏 | 表情 | 动作 | 总分 | 下一轮目标 |
  |---|---|---|---|---|---|---|---|
  | X月X日 | 提问后停1秒 | 4 | 3→4 | 3 | 4 | 14→15 | 句末停顿0.5秒 |
- **风险提示**：每日录制不超过5遍，避免声带疲劳；如出现嗓子疼痛立即停止发声训练，改为"默念+手势"无声练习
- **迭代建议**：节奏稳定后（≥4分），下一轮聚焦表情自然度，采用"减法策略"：删除全程硬笑，仅在问候/肯定/总结三处微笑

## Common Issues

- **过度追求完美**：用户希望一次解决所有问题（状态/节奏/表情同时改）→ 强调"单点突破"原则，每轮只改1个维度，避免认知超载与执行混乱
- **设备条件限制**：用户反映没有手机支架/空间狭小 → 提供替代方案（书堆/纸箱/靠墙放置），确保机位稳定即可；空间小可只录上半身但需确保手势入镜
- **评分主观性偏差**：用户自评时难以客观打分（"我觉得还行"）→ 提供"可观察行为"检查点（如"句末是否有明显换气声""手是否离开腹部"）替代主观感受
- **缺乏对照示例**：用户没有优秀视频对比 → 提供文字版"优秀呈现特征清单"（如"开场眼神环视3个方位""板书时退后半步侧身
