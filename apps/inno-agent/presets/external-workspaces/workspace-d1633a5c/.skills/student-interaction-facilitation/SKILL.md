# 生生互动引导策略

## Instructions

- 本技能专为小学科学实验课堂设计，服务对象为新手教师，聚焦"学生回答后如何组织全班互动"这一具体困境。
- 教师角色定位为**组织者**，不急于评价对错，让学生之间的评价、补充、质疑成为课堂主线。
- 根据教师描述的具体困境（冷场 / 单线交流 / 汇报后沉默 / 错误回答处理）匹配对应话术类型，不泛泛给出通用建议。
- 话术须贴近真实课堂语言，教师无需二次加工即可直接使用。
- 若教师未说明年级或具体场景，优先追问：**年级、实验课题、当前卡住的具体环节**。

## Workflow

1. **识别困境类型**：判断教师遇到的是以下哪类场景——
   - A. 学生回答后全班沉默 / 教师只会说"对/不对"
   - B. 小组汇报后冷场，无人质疑或补充
   - C. 课堂整体冷场，学生不愿开口
   - D. 学生回答错误，教师直接否定打击积极性
2. **匹配话术模板**：按场景输出对应的具体话术（含追问句式）。
3. **说明使用时机**：标注每条话术适用的课堂节点，帮助教师判断何时使用。
4. **附角色提示**：提醒教师在生生互动阶段克制"直接给答案"的冲动，保持组织者身份。

## Output Format

按困境类型分块输出，每块包含：

**【场景】** 一句话描述适用情境
**【话术模板】** 2-3 条可直接朗读的话术，用引号标注
**【使用说明】** 1-2 句说明使用时机或注意事项

结尾附一条**教师角色提示**，强调组织者定位。

## Examples

**场景 A：学生回答后引发全班评价**
> "××同学说了他的想法，其他同学有没有不同意见？"
> "谁能用自己的话，把他的意思再说一遍？"
> "你同意他的说法吗？为什么？"
使用说明：学生回答后立即使用，不先表态对错，将评价权交给全班。

**场景 B：小组汇报后引导互动**
> "请汇报小组问大家一个问题，看看有没有疑问或补充。"
> "哪个小组的结果和他们不一样？说说你们的发现。"
使用说明：汇报结束后立即接入，避免教师独自总结导致其他学生游离。

**场景 C：课堂冷场激活沉默学生**
> "先和旁边同学说说你的想法，30秒后我请人分享。"
> "我们来投票：认为答案是A的举手，认为是B的举手。"
> "你们小组讨论出来的，谁来代表小组说一句话就好。"
使用说明：同伴讨论和选择型提问可显著降低发言门槛，适合全班沉默时使用。

**场景 D：学生回答错误后保护积极性**
> "这个想法很有意思，我们来用实验检验一下，看看结果是不是这样。"
> "你是怎么想到这个答案的？说说你的理由。"
> "还有没有同学有不同的猜测？"
使用说明：不直接说"不对"，通过实验验证或转移焦点推进思考，保护发言积极性。

## Common Issues

- **话术太生硬**：若教师反映话术不自然，建议结合自己的口头语调整，保留核心句式结构即可。
- **学生仍不回应**：先用"同伴讨论30秒"破冰，再请人分享，避免直接点名冷场学生。
- **教师忍不住直接给答案**：提醒自己"沉默5秒是正常的"，等待比填充更有价值。
- **不同学段适配**：低年级可用举手投票、贴图选择等具象方式替代口头互评；高年级可要求学生用"我同意/不同意，因为……"句式规范表达。
