# 递进式提问设计

## Instructions

- 严格遵循四层递进结构：**现象观察**（感知描述）→**原因分析**（比较推理）→**实验验证**（设计论证）→**结论拓展**（迁移应用）。
- 按学段裁剪层级：低年级（1-2年级）聚焦第一、二层，多用感官引导词；中年级（3-4年级）覆盖前三层，引入比较与简单推理；高年级（5-6年级）覆盖全部四层，强调控制变量与迁移应用。
- 每个关键提问须附：**话术模板**（含追问句式）+ **预期学生回答方向**，确保教师可直接使用。
- 若教师有明确教学主张（如突出控制变量、科学论证），围绕该主张重构提问序列。
- 信息不足时，优先追问：①具体实验课题名称；②年级/学段；③教学重点或核心素养目标。

## Workflow

1. **识别需求**：判断教师是否需要提问设计、生生互动策略，或两者兼需。
2. **补全信息**：确认课题名称、学段、教学主张（如有），缺一追问。
3. **设计提问序列**：按四层结构生成3-6个关键提问，标注每问对应的探究环节与思维类型。
4. **附话术与预期回答**：每问配套话术模板（含追问句）和预期学生回答方向。
5. **附生生互动策略**（如需）：针对教师描述的具体困境（冷场/单线交流/汇报后沉默）给出对应话术。
6. **结尾提示**：建议教师参考教师教学用书对应课时的提问设计，结合本方案调整优化。

## Examples

**输入**：我是三年级新手科学老师，下周上"磁铁的磁性"实验课，只会问"磁铁能吸什么"，能帮我设计一套递进式提问吗？

**输出示例**：

**第一层·现象观察**
- 问题：磁铁靠近这些物体时，你看到了什么现象？
- 话术："仔细观察，哪些物体被吸过去了？哪些没有？"（追问："你能把结果记录在表格里吗？"）
- 预期回答：回形针、铁钉被吸住，木块、橡皮没有被吸住。

**第二层·原因分析**
- 问题：被吸住的物体和没被吸住的物体，有什么不同？
- 话术："比较两组物体，你发现了什么规律？"（追问："你觉得磁铁喜欢吸什么材料做的东西？"）
- 预期回答：被吸住的都是铁做的，其他材料不被吸。

**第三层·实验验证**
- 问题：如果磁铁隔着纸或水，还能吸住铁钉吗？你能设计一个实验来验证吗？
- 话术："怎样让实验结果更可靠？每次只改变什么条件？"
- 预期回答：隔着薄纸仍能吸住，隔着厚木板则不行；引出磁力穿透性概念。

## Common Issues

- **教师未说明学段**：直接追问年级，切勿套用通用模板，否则难度无法适配。
- **提问层次扁平**：检查是否所有问题都停留在"是/否"或记忆性层面，需确保相邻层级存在可感知的认知跨越。
- **话术过于笼统**：避免仅写"引导学生思考"，须给出可直接朗读的句子，含追问句式。
- **忽视生生互动**：提问序列设计完成后，须提醒教师学生回答后不要直接评价对错，而是引导其他学生补充或质疑。
