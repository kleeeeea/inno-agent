# 弹性学业路径设计

## When to use

- 学生/家长已明确表达"休学/弹性学制/阶段性脱产投入项目"的诉求，且兴趣价值已通过前期评估确认
- 需要为"非传统学习路径"制定可落地的执行方案，包含学业底线、时间盒、里程碑与可逆退路
- 家校双方陷入"上学还是项目"的对立，急需转化为"如何设计可控实验"的结构化共识

## When not to use

- **兴趣价值未验证**：当用户询问"这个兴趣是否值得投入"或"如何判断是不是三分钟热度"时，应转交 `interest-3d-assessment` 技能
- **常规学业规划**：仅询问学科补习、提分策略或常规选科建议时，使用普通学业规划工具
- **需专业转介**：涉及学生抑郁/焦虑症状、家庭关系冲突或法律/财务风险时，转介心理咨询师或专业顾问
- **违法违规场景**：未成年人高风险商业活动、违反学籍管理规定的操作请求

## Instructions

1. **底线优先原则**：任何方案必须首先明确学籍合规、关键学科（语数外）最低学习量、身心健康监测三项不可突破的底线
2. **时间盒思维**：所有非传统路径必须设定6-12个月的明确周期，禁止"无限期休学"或"先试试看"的模糊承诺
3. **可逆性设计**：方案必须包含明确的复学时间点、补学方案、成绩缺口解释预案，确保项目失败后可平稳回归主流路径
4. **共识书面化**：通过非法律性质的"三方共识备忘录"固定目标、里程碑、复盘节点与退出机制，避免口水战

## Workflow

1. **冲突诊断与底线设定**
   - 确认当前学业段位、关键学科基础、中考/高考时间节点
   - 明确学籍合规要求、监护人支持度、资金/导师资源可用性
   - 设定身心健康监测指标与关键学科最低学习量

2. **路径分支选择**
   - **分支A（学业为主）**：课余/周末/假期集中块，学校提供选修课转化、导师制匹配、校内成果展
   - **分支B（弹性学制）**：阶段性休学，需明确时间盒周期（建议6-12个月）、每月验收点、复学触发条件

3. **时间盒与里程碑设计**
   - 将周期拆分为4-8周为一个复盘节点
   - 每个节点设定SMART化产出目标（作品迭代、用户数据、技术验证等）
   - 设定黄牌预警（调整策略）与红牌退出（启动复学）的量化标准

4. **可逆性方案制定**
   - 复学时间点与流程（如何对接原年级/下一年级）
   - 补学方案（关键学科缺口弥补计划、学习时长安排）
   - 成绩缺口叙事（解释信结构：中断原因+学习能力证明+补学计划）

5. **共识备忘录生成**
   - 起草三方非法律协议，明确目标、里程碑、复盘节点、退出机制、风险责任边界

## Output Format

## Examples

**输入**：初三学生想休学一年投入AI创业项目，已有初步原型和种子用户，家长担心中考和"落后同龄人"，学校担忧学籍合规。

**输出**：

### 场景判断
- 冲突类型：制度冲突（中考升学 vs 创业窗口期）
- 建议路径：分支B 弹性学制（9个月时间盒）
- 适用周期：初三上学期至中考报名前（可逆节点）

### 方案详情
- **学业底线**：语文/数学/英语每周最低6小时学习（线上课程+自测），保留中考报名资格
- **时间盒规划**：9个月分三阶段（3月产品迭代→3月市场验证→3月决策期），每月产出：用户增长率、功能迭代版本、营收数据
- **支持机制**：学校提供"创新实践"选修学分认定，匹配信息技术教师作为校内导师，季度校内答辩展示
- **可逆性设计**：若第6个月用户增长<20%即启动复学，预留3个月冲刺中考；保留补报名通道；准备《中断学业期间学习能力证明》文档

### 风险控制清单
- 学籍合规：办理"因病休学"或"其他"类别（依当地政策），确保不注销学籍；避免签订劳动合同（保持学生身份）
- 身心健康监测：每月一次心理量表评估，每周运动时长不低于3小时
- 资金风险：明确未成年人股权代持限制，建议以竞赛/研究名义而非公司运营

### 家校沟通话术
- 对家长："这不是'辍学'，而是'可控的职业实验'。9个月后要么带着验证过的项目进入高中（申请科技特长生），要么带着对商业的深度理解回归学业，两种结果都比现在盲目选择更有价值。中考可以晚一年，但创业窗口不等人。"
- 对学校："学校不承担创业成败责任，只提供'项目制学习'的平台和学分认定，学生仍以学习者身份积累成果，符合综合实践活动课程要求。"

### 共识备忘录模板
- 目标：9个月内验证产品市场匹配度（PMF），达成1000名活跃用户或获得天使轮融资
- 复盘节点：第3个月（原型测试）、第6个月（可逆决策点）、第9个月（最终评估）
- 退出机制：连续两个月用户零增长或学业底线未达标（周学习<6小时）立即复学
- 权责边界：学生负责项目执行与学业底线维持；家长负责资金与监护；学校负责学籍合规与学分认定

### 后续评估机制
- 绿色：达成PMF→继续弹性学习并准备科技特长生/国际申请路径
- 黄色：部分达成→转为分支A（学业为主），项目转为周末/假期推进
- 红色：未达关键指标→第6个月启动复学，执行补学方案

## Common Issues

1. **家长"落后焦虑"僵化**：反复使用"早学晚学一年差别不大，但职业窗口转瞬即逝"框架，将焦虑从"时间"转移到"风险控制证据"上
2. **学校学籍操作困惑**：明确区分"休学"（保留学籍）与"辍学"（注销学籍），建议咨询当地教育局具体流程，不提供违规操作指导
3. **项目失败后的学业缺口**：提前准备《学习能力证明》模板（项目期间自主学习记录、技术文档产出），用于解释成绩不连续，避免复读 stigma
4. **时间盒边界模糊**：坚决禁止"先休学看看"的无限期方案，必须在启动时明确最长周期（不超过12个月）和硬性复学触发条件
