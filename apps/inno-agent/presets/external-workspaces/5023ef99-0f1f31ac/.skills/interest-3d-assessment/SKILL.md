# 职业兴趣三维评估

## When to use
- 家长质疑"这是三分钟热度还是真兴趣"需要证据化判断时
- 学生提出休学/减课投入项目，需先评估兴趣强度与证据链时
- 区分短期竞赛冲动与长期职业方向时
- 已有作品集/项目成果，需评估是否值得调整学业节奏时

## When not to use
- **已确认兴趣价值**，询问具体休学/弹性学制操作流程→转交 `flexible-path-design`
- **询问学科补习/提分计划**→转交常规学业规划工具
- **涉及心理抑郁/焦虑症状**或家庭关系严重冲突→转介专业心理咨询师
- **需要法律/财务/商业合规审查**→转介专业顾问

## Instructions
- 严格依据三维指标量化评估，拒绝"感觉不错"式主观判断
- 区分"兴趣存在"与"值得学业让路"两个阈值层级
- 未通过评估时，必须提供"学业为主+课外深化"的保守方案
- 所有结论须指向可验证证据，不自作升学成功率承诺

## Workflow
1. **持续性扫描**：复盘6-12个月时间轴，记录每周投入时长、间断模式与深度学习痕迹
2. **深度证据审查**：列出可验证产出（代码/原型/专利/用户数据/论文/竞赛），评估技术/创意/社会价值维度
3. **自驱性测试**：执行"外力移除假设"（无家长催促/老师要求时的投入度），验证内在动机强度
4. **综合判定**：三维均通过→建议进入路径设计评估；任一维度薄弱→建议学业为主方案
5. **风险标注**：标记心理健康/合规风险/学业底线等需转介事项

## Output Format
- **场景判断**：冲突类型（时间冲突/认知冲突/证据不足）
- **三维评估结果**：
  - 持续性：[时间跨度/投入记录/间断分析]
  - 深度证据：[产出清单/可验证性/价值维度评分]
  - 自驱性：[外力移除测试结果/内在动机证据]
- **评估结论**：[建议进入弹性路径设计 / 建议学业为主+课外深化]
- **关键证据清单**：可用于升学档案的具体材料（附四段式描述建议）
- **风险提示**：心理健康/合规/学业底线等注意事项
- **下一步动作**：转交路径设计或制定课外深化计划

## Examples
**案例1（通过评估）**
- 持续性：14个月持续投入，GitHub连续提交记录，寒暑假每日4小时
- 深度证据：开源项目获200+Star，解决特定技术问题，有用户Issue反馈
- 自驱性：父母反对期间仍凌晨调试代码，校内未选信息技术课
- 结论：建议进入弹性路径设计，同时准备学业底线方案

**案例2（未通过评估）**
- 持续性：仅赛前3个月集训，赛后无自发练习
- 深度证据：仅有竞赛奖状，无独立作品/用户/代码库
- 自驱性：停止家长报班后无自主练习行为
- 结论：建议学业为主，课余通过选修课/周末工作坊深化兴趣

**案例3（风险转介）**
- 评估发现学生伴随失眠、自我否定陈述及家庭高压冲突
- 结论：暂停决策流程，建议优先进行心理健康评估与家庭咨询

## Common Issues
- **混淆获奖与持续**：将"竞赛奖状"等同于长期投入，忽视备赛周期外的真实投入记录
- **误判自驱性**：把"家长积极推动"当作学生内在动机，未做"外力移除测试"
- **忽视可验证性**：仅凭"孩子很喜欢"判断，缺乏作品/数据/用户等第三方证据
- **阈值混淆**：将"有兴趣"直接等同于"值得休学"，未区分兴趣存在与学业让路的高标准差异
