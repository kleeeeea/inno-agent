# 提供顺延申请指导
## When to use
*   当用户询问培训顺延申请的流程、步骤或具体操作方法时。
*   当用户需要了解申请培训顺延所需准备的材料或文件时。
*   当用户咨询培训顺延申请过程中需要注意的事项或常见问题时。
*   当用户请求关于培训顺延申请的通用性指导信息时。

## When not to use
*   当用户明确要求根据其个人信息生成一份培训顺延申请书时，应路由至 `generate-postponement-application` 技能。
*   当用户要求提供法律咨询或政策解读时。
*   当用户要求查询或修改其申请的审批状态时。
*   当用户要求提供其所在机构的内部审批流程细节（本技能仅提供通用指导）。
*   当用户请求生成非培训顺延类的申请时。

## Instructions
*   确认用户是寻求指导信息而非生成申请。
*   清晰、准确地提供培训顺延申请的流程、所需材料和注意事项。
*   强调所提供信息为通用指导，建议用户以其所在机构的最新规定为准。

## Workflow
1.  **理解需求**: 接收并解析用户请求，判断其意图是获取指导。
2.  **信息提供**: 直接提供关于培训顺延申请的流程、所需材料和注意事项的详细说明。
3.  **输出结果**: 将指导内容以结构化、清晰简洁的方式呈现给用户。

## Output Format
```
场景判断: 用户请求培训顺延申请指导。
处理方案: 提供培训顺延申请的通用流程、所需材料和注意事项。
可用话术:
您好！关于培训顺延申请，您可以参考以下通用指导：

**一、申请流程：**
1.  确认您所在机构关于培训顺延的具体规定和截止日期。
2.  准备所需材料。
3.  填写培训顺延申请表或撰写申请书。
4.  提交申请至相关部门（如党委教师工作部、人事部门等）。
5.  等待审批结果。

**二、所需材料：**
1.  个人身份信息（姓名、工号、部门等）。
2.  无法参加培训的原因说明（如因公出差、身体不适、家庭紧急事务等）。
3.  期望顺延的培训期次或具体安排。
4.  其他可能需要的证明材料（如医院证明、出差证明等）。

**三、注意事项：**
1.  **及时性**: 务必在规定时间内提交申请。
2.  **真实性**: 确保提供的信息和原因真实有效。
3.  **规范性**: 申请书或申请表应符合机构要求，内容完整、表述清晰。
4.  **机构规定**: 请务必以您所在机构的最新规定和具体要求为准。

风险提示: 本指导为通用信息，不构成法律意见或政策解读。请用户务必核实其所在机构的具体规定。
后续评估: 询问用户是否还有其他关于申请指导的问题。
```

## Examples
*   "培训顺延申请需要准备哪些材料？"
*   "申请顺延的流程是怎样的？"
*   "申请顺延有什么注意事项吗？"
*   "我想了解一下培训延期申请怎么办理？"

## Common Issues
*   用户可能误以为本技能可以查询其申请的审批状态，但本技能无法提供此服务。
*   用户可能要求提供特定机构的内部审批流程细节，但本技能只能提供通用指导。
*   用户可能要求提供法律或政策解读，但本技能不具备此能力。
*   用户可能混淆生成申请书与获取指导的需求，需要进行明确的路由判断。
