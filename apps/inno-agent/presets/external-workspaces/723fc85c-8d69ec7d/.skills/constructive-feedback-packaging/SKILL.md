# 建设性话术包装

## When to use

- 已有具体观察内容（如"AI视频干扰数学思维"），仅需润色表达以适应权威场域
- 需要将学生中心的批判性观察转化为尊重授课教师的专业话术
- 担心观点过于尖锐，需要"徒弟身份"的得体表达方式
- 需引用课标理论支撑观点，避免"AI套话"和空泛表述

## When not to use

- 尚未确定观察角度，需从冷门维度寻找不重复观点 → 交给 **unconventional-dimension-scan**
- 询问如何记录前面老师发言或课堂观察方法 → 使用普通问答
- 意图为"强行不同"而刻意挑刺，或评价教师个人特质（如性格急躁）→ 安全转介，阻止并提醒聚焦教学行为

## Instructions

1. 严格遵循"肯定意图+专业观察+小措施"三段式结构
2. 引用具体课标条款（如"基于教学评一体化理念"）支撑观点，禁用"充分体现新课程理念"等空泛AI腔
3. 将"教师教得怎样"转化为"学生学得怎样"的表达方式，维护授课者尊严
4. 小措施必须具体可操作（如"调整等待时间至5秒"），避免笼统建议
5. 语气保持"徒弟"身份的谦逊与专业，既指出问题又体现建设性

## Workflow

1. 接收用户提供的原始观察点（如"学生操作电脑时看不到教师演示"）
2. 识别授课教师的教学设计意图（如"技术融合的创新尝试"）
3. 将观察点转化为学生中心视角（如"从学生视角看，屏幕遮挡影响了关键步骤的观察"）
4. 匹配课标理论依据（如《义务教育课程方案》"教学评一体化"或学科课标相关条款）
5. 生成具体改进措施（如"建议采用分屏展示或操作前明确视线引导"）

## Output Format

- **场景判断**：当前权威场域压力等级（高/中/低）及授课教师情绪敏感度提示
- **肯定意图**：对授课教师设计初衷的正面解读（1句话）
- **专业观察**：基于课标的学生中心观察（1-2句话，引用具体理论）
- **小措施**：具体可操作的教学调整建议（1条，含实施细节）
- **风险提示**：需规避的表达禁忌（如避免使用"完全""明显"等绝对化词汇）
- **备用话术**：若教研组长追问时的简短回应版本

## Examples

**输入**："我发现前面的老师都说AI视频很生动，但我观察到学生看AI视频时很兴奋，但问到数学概念就懵了"

**输出**：
- **肯定意图**：本节课在情境创设上积极探索技术融合，AI动画的设计体现了激发学习兴趣的意图。
- **专业观察**：从学生认知负荷角度看，动态视觉刺激可能分散了对"凑十法"核心数量关系的注意力，部分学生在视频结束后出现概念提取困难，这提示我们需要关注多媒体呈现与数学思维过程的匹配度（参照数学课标"情境服务于知识建构"理念）。
- **小措施**：建议在AI视频播放后增加一个"闭眼复述关键画面"的静默思考环节，或采用定格画面让学生用手指比划数量变化，确保视觉兴奋转化为数学理解。
- **风险提示**：避免直接说"视频干扰学习"，改用"视觉刺激与思维过程的平衡"表述；禁用"充分体现信息素养"等套话。
- **备用话术**：如果追问，可以说："我观察到视频播放后学生的复述准确率有差异，可能需要增加一个思维缓冲环节。"

## Common Issues

- **AI腔调陷阱**：用户可能要求使用"充分体现了...理念"等套话，需坚决替换为具体课标条款（如"参照数学课标'情境创设与问题提出'要求"）
- **过度委婉**：为维持和谐而模糊表达，导致失去专业价值（如"可能也许大概"），需保持观察的具体性
- **小措施空泛**：建议"加强关注"改为"在巡视时优先停留学困生座位5秒以上"；"提高效率"改为"将操作步骤分解为3个关键词板书"
- **身份错位**：忘记"徒弟"身份使用指导者语气，需加入"个人浅见""观察所得"等谦逊表达，但保持专业底气
