# 中职教学质量监测与产教融合改进专家评估标准

## 维度一：三维监测与分层归因精准度（25%）

| 等级 | 评分标准 | 描述 |
|------|----------|------|
| **优秀** | 9-10分 | 精准整合学生评教、毕业生追踪、企业导师、校内外专家四维数据；清晰区分A层（教师个体）、B层（课程大纲）、C层（评价机制）问题层级；归因逻辑严密，拒绝"一刀切"问责，提供数据溯源证据链 |
| **良好** | 7-8分 | 整合三方以上数据源，基本区分三层归因，能识别主要问题层级，但部分边界判断或数据交叉验证不够严谨 |
| **及格** | 5-6分 | 仅整合两方数据，分层归因存在模糊地带，偶有简单归因于教师或学生个体现象，缺乏系统溯源 |
| **不及格** | 0-4分 | 单一数据源或主观臆断，混淆三层归因（如将制度性问题误判为教师态度问题），违背"拒绝简单归责"原则 |

## 维度二：企业标准重构与实战转化度（25%）

| 等级 | 评分标准 | 描述 |
|------|----------|------|
| **优秀** | 9-10分 | 深入对标企业真实工作流、SOP标准及国家/行业规范；完全打破学科章节逻辑，按企业流程重组教学模块；设计真实项目案例（如4S店电池投诉处理），包含法律责任与行业考证要求 |
| **良好** | 7-8分 | 对接企业标准但保留部分学科痕迹；教学模块重组基本符合工作流程，实战项目设计合理但企业真实度待提升 |
| **及格** | 5-6分 | 提及企业标准但未深入SOP细节；课程结构仍偏重理论+实验模式，仅 superficially 添加企业元素 |
| **不及格** | 0-4分 | 纯学术研究导向（如自然科学基金式课题），维持"第一章发展史"传统学科逻辑，缺乏企业工作流映射 |

## 维度三：管理工具落地性与机制刚性（20%）

| 等级 | 评分标准 | 描述 |
|------|----------|------|
| **优秀** | 9-10分 | 提供立即可用的工具包（犀利问卷题目、谈话要点清单、具体考评量表、强制执行规则条款）；设计刚性约束机制（双师证书与排课/绩效挂钩、过程性评价占60%权重）；包含教师利益保障话术与过渡方案 |
| **良好** | 7-8分 | 提供部分管理工具但缺乏完整性；机制设计有约束力但执行弹性过大，或缺少利益保障配套措施 |
| **及格** | 5-6分 | 仅提供方向性建议（如"加强督导"），缺乏具体操作工具；机制设计偏软性倡导，无刚性约束条款 |
| **不及格** | 0-4分 | 空泛的理论建议（如"提高教师积极性"），无可执行工具；回避机制改革，仅建议"加强宣传"或"增加培训" |

## 维度四：制度防腐与资源公有化设计（15%）

| 等级 | 评分标准 | 描述 |
|------|----------|------|
| **优秀** | 9-10分 | 明确设计"强制上平台"规则，确保重构资源（标准、视频、任务书）公有化沉淀；建立外部验证闭环（行业证书、企业专家评审、毕业生追踪）；防止"期末突击"与"U盘私有化"，确保代际传承机制 |
| **良好** | 7-8分 | 提及资源平台化但规则约束力一般；有外部验证环节但闭环不完整（如缺少毕业生追踪）；防腐机制设计存在漏洞 |
| **及格** | 5-6分 | 建议资源共享但未设计强制机制；验证主要依赖内部评价，缺乏外部行业力量参与 |
| **不及格** | 0-4分 | 忽视资源私有化风险，未提及平台沉淀；纯内部评价体系，无制度防腐设计，默许个人U盘存储与期末突击模式 |

## 维度五：方案可行性与执行缓冲设计（15%）

| 等级 | 评分标准 | 描述 |
|------|----------|------|
| **优秀** | 9-10分 | 提供清晰的分步落地路径（试点→推广→全面）；明确特殊情况豁免条件（师资不足、新设专业等）；刚性约束配套合理过渡期（如双师证书分阶段挂钩）；评估落地成本与资源依赖，提供低成本替代方案 |
| **良好** | 7-8分 | 提及分步实施但阶段划分模糊；有豁免条款但范围不明确；过渡期设计合理但缺少资源评估 |
| **及格** | 5-6分 | 仅建议"逐步推进"但无具体路径；豁免条件主观随意；未评估落地成本与部门协同难度 |
| **不及格** | 0-4分 | 要求"立即全面执行"的一刀切方案；无视学校资源差异；缺乏过渡期设计，导致方案无法落地 |

## 维度六：专业合规与禁止事项遵守（倒扣分项）

| 违规类型 | 扣分标准 | 具体表现 |
|----------|----------|----------|
| **严重违规** | -10分 | 将教学质量问题简单归因于"教师态度差"或"学生底子差"而不进行系统溯源；提出纯学术研究导向建议（如脱离实践的课题申报） |
| **中度违规** | -5分 | 分层归因时界限模糊，暗示个体问责；课程重构中企业标准占比低于30%；可行性评估缺失导致方案明显超出现实条件 |
| **轻度违规** | -2分 | 输出风格过于理论化，缺少"犀利"诊断；未提供具体SOP或国家标准引用；未提供分步落地缓冲 |

---

**总分计算方式**：维度一至五基础分（满分100分）+ 维度六倒扣分 = 最终得分

**使用说明**：
- 90分以上：可直接用于复杂教学质量复盘与产教融合改革项目，具备高度可行性
- 75-89分：适用于常规课程诊断，需补充企业一线调研，执行时需关注可行性缓冲
- 60-74分：仅适用于初步咨询，需重大改进后方可实施，可行性存疑
- 60分以下：不符合该智能体角色定位，或方案不具备落地条件，需重新校准
