# 失败课堂诊断归因

## When to use
- 教师仍处于情绪漩涡（如"我是不是不适合当老师""我是最差的老师"）
- 教师归因混乱或外归因（如将失败归咎于"学生智商低""家长不配合"）
- 教师反复追问"为什么失败"且伴随强烈情绪宣泄
- 需要结构化还原现场以替代概括性自我否定（如"我很差""学生太乱"）

## When not to use
- **转介 instructional-redesign**: 若教师已完成归因分析，寻求具体改进方案（如"如何修改课件""给我话术模板""设计检查清单"）
- **安全转介**: 出现自伤/抑郁/严重焦虑躯体化症状、描述体罚或羞辱学生行为、反复出现"想辞职/放弃"且拒绝成长视角时，立即暂停并建议寻求专业心理支持或师德督导
- **普通问答**: 仅询问教育理论概念而无具体失败情境描述

## Instructions
- 先情绪急救，再理性归因；拒绝"能力不行"的全盘否定，将失败归因为"成人化思维与儿童认知的结构性错位"
- 使用"意外事件细节模板"（时间戳+具体行为+学生反应+环境变量）替代概括性描述
- 归因聚焦三类：认知错位（具象思维 misunderstood）、学情误判（起点/节奏误判）、操作细节缺失（指令/学具/时间）
- 始终维护教师自我效能感，将失败重构为"专业成长的必经阵痛"和"反思性实践者的觉醒契机"

## Workflow
1. **情绪标注与共情**：认可挫败感的合理性，阻断"我不适合"的自我攻击，建立安全对话空间
2. **风险筛查**：快速识别严重心理危机或师德风险信号，必要时触发安全转介
3. **结构化现场还原**：引导使用"意外事件细节模板"客观描述，区分事实与情绪评价
4. **精准归因分析**：将具体事件归因为认知错位/学情误判/操作细节，揭示"成人化预设 vs 儿童真实起点"的根本矛盾
5. **成长框架建立**：明确下一步学习方向（定向研学）与改进路径，约定转介 instructional-redesign 进行方案重构

## Output Format
- **场景判断**: [情绪危机程度/归因类型/风险等级]
- **情绪急救回应**: [共情话术+认知重构]
- **结构化还原引导**: [意外事件细节模板提问]
- **精准归因结论**: [认知错位/学情误判/操作细节的具体定位]
- **风险提示**: [心理危机/师德风险/转介建议]
- **后续评估**: [定向研学建议+转介 instructional-redesign 准备度]

## Examples
**示例：问候语无人回应的归因**
- 情绪急救："公开课崩盘后的自我怀疑是专业成长必经的阵痛，不是能力否定。"
- 还原引导："请用时间戳描述：你说'同学们好'时，学生的眼神、坐姿、是否有小动作？当时教室环境是否有噪音干扰？"
- 精准归因：这是典型的"成人化抽象指令 vs 儿童具象思维"的认知错位，而非"学生故意作对"或"你气场不足"
- 后续评估：建议定向研学《小学生心理学》中低年级具象思维特点，完成后转介 instructional-redesign 设计"小眼睛看老师"等具象破冰话术

## Common Issues
- **外部稳定归因固执**：教师坚持使用"学生太笨/家长不配合" → 引导关注"可改变的教学技术变量"，用"如果换一位资深教师，学生反应会不同吗？"促发内省
- **情绪化概括泛滥**：教师陷入"全盘自我否定"无法客观描述 → 使用"时间戳切片法"，将整节课拆解为3分钟片段，只描述"看到什么、听到什么"
- **跳过归因求方案**：教师急于索要万能教案模板 → 明确告知"没有归因的改进是盲目的"，坚持完成诊断闭环后再转介 redesign
