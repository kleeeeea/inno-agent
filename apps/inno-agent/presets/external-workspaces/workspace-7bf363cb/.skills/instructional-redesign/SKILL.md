# 教学方案重构优化

## When to use

- 教师主动索要具体改进工具："请帮我设计导入环节话术""如何修改课件让一年级学生听懂"
- 针对特定教学卡点寻求重构：问候语无人回应、学具操作失控、内容节奏失衡、小组讨论冷场、多媒体故障等
- 需求明确指向"学具准备检查表""时间轴工具""无PPT平替方案"等具象化产出
- 已度过情绪危机期，进入"如何改进"的技术寻求阶段

## When not to use

- **应转介至 failure-diagnosis**：教师仍处于情绪漩涡（反复追问"为什么我总是失败"）、归因混乱（归咎于学生智商/家长不配合）或伴随严重自我否定
- **应转介至 failure-diagnosis**：未经历归因分析直接要求"给我万能教案模板"等逃避反思的请求
- **应安全转介**：方案涉及体罚、羞辱学生或报复性教学等师德风险
- **应普通问答**：高中及以上学段纯学科知识答疑（非低年级认知适配问题）

## Instructions

1. **视角转换原则**：所有方案必须体现从"教为中心"到"学为中心"的范式转变，将成人抽象思维转化为儿童具象行动
2. **单步拆解原则**：将复杂指令拆解为6-8岁可执行的单步动作，避免并列动词（如"观察并记录"改为"第一步：眼睛看这里"）
3. **隐形助手显性化**：强制预设学具可操作性、环境变量、突发干扰，建立可勾选检查清单而非依赖临场应变
4. **目标锚定原则**：应急预案必须围绕核心教学目标设计替代路径，而非仅解决技术故障（如"多媒体坏了换板书"→"无动画如何呈现空间概念"）

## Workflow

1. **场景诊断**：确认具体失效环节（导入/探究/练习/总结）及归因类型（指令抽象/节奏失衡/细节缺失/支架不足）
2. **儿童化重构**：基于皮亚杰具象思维理论，将抽象教学语言转化为具身动作、视觉提示或童谣口令
3. **工具固化**：生成可复用的检查清单、话术卡片、时间轴或学具预检表
4. **应急预案**：设计基于深层教学逻辑的平替方案（如"若投影失效，如何用实物操作达成相同认知目标"）
5. **迭代约定**：设定平行班再实践时间节点与观察反馈机制

## Output Format

```
【场景判断】
失效环节：导入/新授/练习/总结
归因标签：指令抽象/节奏失衡/学具障碍/支架缺失/技术故障

【儿童化重构方案】
原成人指令："同学们好/开始讨论"
具象化话术："小眼睛看老师/手指点到最喜欢的段落读一遍"
动作支架：配合手势/视觉锚点/童谣节奏

【隐形助手检查清单】
□ 学具：拉链顺滑度/独立操作能力预检（让学生课前试开一次）
□ 动词：每个指令包含唯一可观察动作
□ 时间：每个环节预设±2分钟弹性区间
□ 空间：小组活动半径/物品传递路径预设

【应急预案】
触发条件：多媒体故障/学具失效/学生突发质疑
平替策略：围绕核心目标（如"建立长度概念"）设计无实物/纯口述/黑板手绘替代路径
话术储备："设备遇到问题，我们用另一种方法探索"

【应用示例】
场景还原：...
教师行为：...
学生预期反应：...

【风险提示】
- 避免过度设计导致教师机械执行
- 防止支架过密限制学生自主空间
- 需配合 failure-diagnosis 完成前置情绪处理
```

## Examples

**示例1：一年级问候语重构**
- **原指令**："同学们好"（抽象社交礼仪）
- **具象化方案**："小眼睛——看老师"（视觉聚焦）+"小嘴巴——闭闭紧"（身体控制）+"小手——放膝盖"（具身定位）
- **检查清单**：课前带学生演练一次"口令-动作"配对，确认40人均能条件反射

**示例2：尺子测量环节学具管控**
- **隐形助手清单**：课前15分钟让学生试开铅笔盒（拉链/按扣）一次；尺子预先贴好姓名贴防争抢；备用20把尺子应对损坏
- **应急预案**：若一半学生无法取尺，改为"目测-手势比划-教师示范尺"三级递进，核心目标（长度比较）不中断

**示例3：15分钟内容3分钟讲完的时间拆解**
- **范式转变**：从"我要讲完PPT"到"学生需要多少次触摸/比划/错误尝试"
- **拆解方案**：原"讲解读写姿势"拆为"找颈椎骨（触摸）→ 比 triangle（手势）→ 同桌互查（社交）→ 错误示范辨析（判断）"，每步3分钟，填充原空置时间

## Common Issues

1. **过度设计陷阱**：提供的话术过于繁琐，教师背诵困难。应控制在10字以内口诀，配合固定手势。
2. **忽视学生操作能力**：假设学生能独立完成"打开学具/翻书到指定页"等动作，未设计"预演环节"或"同伴互助支架"。
3. **应急预案流于形式**：仅建议"准备备用U盘"等技术备份，未设计"教学目标不变-媒介转换"的深层替代逻辑。
4. **混淆难度与支架**：将"学生听不懂"简单理解为"降低难度"，而非"增加认知台阶（支架）"，导致目标缩水而非路径优化。
5. **遗漏情绪后处理**：直接给出技术方案，未提示"此方案需在情绪平复后使用，建议先完成 failure-diagnosis 阶段"。
