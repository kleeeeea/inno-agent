# 班级反霸凌干预助手 评分量规

## 评分维度

---

### 维度一：干预方案的结构完整性与可操作性
**评估重点**：输出的干预方案是否覆盖完整的五步流程（阻断定性→家校沟通→责任闭环→主题班会→动态追踪），每步是否包含具体操作步骤、可直接使用的话术示例及常见误区提示。

| 分数 | 描述 |
|------|------|
| 4 | 完整输出五步流程，每步均包含操作步骤、话术示例和误区提示；步骤逻辑清晰，班主任无需额外解读即可执行 |
| 3 | 覆盖大部分步骤，话术示例或误区提示偶有缺失，但整体可操作性较强 |
| 2 | 流程不完整，缺少1–2个关键步骤，或话术示例过于笼统，难以直接使用 |
| 1 | 仅提供原则性建议，缺乏具体操作指引，班主任无法据此直接行动 |

---

### 维度二：场景识别与信息收集的精准性
**评估重点**：助手是否准确判断输入属于"已发生/疑似霸凌事件"还是"预防/日常教育需求"，并在关键信息缺失时（如学段、霸凌类型、当前所处阶段）以不超过2个问题的方式高效补充，避免打断节奏或过度追问。

| 分数 | 描述 |
|------|------|
| 4 | 场景判断准确，信息充足时直接输出方案；信息缺失时精准提问，单次不超过2个问题，提问后立即推进 |
| 3 | 场景判断基本准确，提问数量合理，但偶尔提问与场景关联度不高，或在信息已足够时仍追问 |
| 2 | 场景判断出现偏差，或提问超过2个、打断节奏，导致班主任需要多次澄清才能获得方案 |
| 1 | 未进行场景判断，直接输出与需求不符的内容，或反复追问导致交互停滞 |

---

### 维度三：话术示例的适龄性与课堂真实感
**评估重点**：提供的教师话术是否符合目标学段（小学低段/高段/初中）的语言认知水平，是否贴近真实课堂语境，能否让班主任"开口即用"，而非需要大幅改写。

| 分数 | 描述 |
|------|------|
| 4 | 话术与学段高度匹配，语言自然、情境感强，班主任可直接使用；不同学段的话术有明显差异化处理 |
| 3 | 话术基本适龄，语言较为自然，但部分表达略显书面化或通用化，需少量调整 |
| 2 | 话术适龄性不足，语言偏学术或偏成人化，与真实课堂语境有明显落差 |
| 1 | 话术缺失或完全不适龄，班主任无法参考使用 |

---

### 维度四：对受害者保护与二次伤害风险的关注度
**评估重点**：方案是否在干预全程优先保障受害者的心理安全与隐私，是否明确标注二次伤害风险点（如公开点名、当众对质等），家校沟通顺序是否遵循"先受害者家长"原则，追踪机制是否兼顾双方心理状态。

| 分数 | 描述 |
|------|------|
| 4 | 全程体现受害者优先原则，关键风险点均以⚠️明确标注，家校沟通顺序正确，追踪机制覆盖双方心理状态 |
| 3 | 整体关注受害者保护，但部分风险点未标注，或追踪机制对施暴者关注多于受害者 |
| 2 | 仅在部分步骤提及受害者保护，二次伤害风险点未作提示，家校沟通顺序有误 |
| 1 | 未体现受害者保护意识，存在可能造成二次伤害的建议 |

---

### 维度五：班会方案的教育设计质量
**评估重点**：当输出主题班会方案时，是否包含认知/情感/行为三维目标，环节设计是否合理（含时间分配），是否涵盖行为红线说明与求助路径公示，课后跟进建议是否具体可执行，总长是否控制在800字以内。

| 分数 | 描述 |
|------|------|
| 4 | 三维目标明确，环节设计完整且时间分配合理，行为红线与求助路径均有涉及，课后跟进建议具体，总长符合规范 |
| 3 | 目标基本完整，环节设计合理，但时间分配或课后跟进建议略显笼统，总长基本符合规范 |
| 2 | 目标不完整或环节缺失（如无求助路径公示），课后跟进建议过于原则化，或内容严重超出800字 |
| 1 | 班会方案结构混乱，缺少核心环节，无法作为实际班会参考使用 |
