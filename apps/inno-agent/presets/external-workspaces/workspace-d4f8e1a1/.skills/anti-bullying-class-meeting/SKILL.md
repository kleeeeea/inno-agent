# 反霸凌主题班会设计

## Instructions
- 根据学段（小学低段/小学高段/初中）和触发原因（事后修复/日常预防）定制方案，语言与活动设计须符合学生认知水平
- 班会方案须包含认知、情感、行为三类目标，并覆盖导入→核心活动→行为红线说明→求助路径公示四个环节
- 话术示例用引号标注，可直接开口使用；⚠️ 标注二次伤害风险点（如公开点名受害者、当众对质等）
- 事后修复性班会不得点名当事人，应将事件转化为匿名案例或情境引入

## Workflow
1. **收集关键信息**（如未提供，最多问2个问题）：学段与班级规模、触发原因（事后修复/日常预防）、可用课时与资源条件
2. **确定班会类型**：事后修复型侧重共情与规范重建；预防型侧重认知提升与求助路径
3. **输出完整方案**，结构如下：
   - 班会目标（认知/情感/行为）
   - 适用学段与时长（建议40分钟）
   - 四环节设计（含时间分配与教师引导话术）
   - 课后跟进建议（问卷/观察记录/强化活动）

## Output Format
- 使用分级标题与分点列表，便于班主任快速定位操作
- 时间分配以括号标注在各环节标题后，如：**导入**（5分钟）
- 话术示例单独成行，用引号包裹
- 方案末尾附**效果评估提示**，说明如何判断班会成效
- 总长不超过800字，结构清晰优先于内容堆砌

## Examples

**输入**：初中班级上周发生言语霸凌事件，已完成家校沟通，现在需要开一节班会。

**输出片段**：

**班会目标**
- 认知：理解言语霸凌的定义与危害，明确校纪与法律后果
- 情感：体验被孤立与嘲讽的感受，激发共情与责任感
- 行为：掌握3条求助路径，承诺遵守班级反霸凌公约

**环节设计**

**导入**（5分钟）
呈现匿名改编案例（勿涉及本班细节）："有一个同学，每天被人叫外号、被排除在群聊之外……"
引导提问："如果你是他，你会有什么感受？"

**核心活动：角色扮演**（20分钟）
分组模拟"旁观者如何介入"场景，讨论：旁观者沉默是否也是一种伤害？
⚠️ 角色扮演中禁止还原真实事件情节，避免当事人二次受伤

**行为红线说明**（10分钟）
"以下行为在我们班零容忍：起侮辱性外号、散布谣言、孤立同学……"
说明违规后果：班规处理→家长约谈→情节严重移交学校

**求助路径公示**（5分钟）
公示三条路径：①直接告诉班主任 ②投入班级意见箱 ③拨打学校心理援助电话
全班签署《班级反霸凌公约》

**课后跟进**：一周后发放匿名安全感问卷，持续观察当事双方课间互动

## Common Issues
- **误区**：班会中点名批评施暴者 → 会引发当众羞辱，激化矛盾，应全程使用匿名案例
- **误区**：仅宣读规定、无互动活动 → 学生参与感低，规范内化效果差，须加入角色扮演或小组讨论
- **误区**：班会结束即视为干预完成 → 须配合课后追踪机制，至少观察4周
- **误区**：小学与初中使用相同话术 → 小学低段应用故事/绘本引入，初中可引入法律条文与真实新闻案例
