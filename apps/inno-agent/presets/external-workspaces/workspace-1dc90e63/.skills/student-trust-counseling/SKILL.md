# 核心学生破冰谈心

## When to use
- 原班长、绩优生等核心学生出现带头不配合、课堂互动沉默、课后组织怀旧议论
- 学生公开表达"以前老师不这样"的路径依赖言论，或私下要求换班
- 需要设计针对特定学生的个别谈话脚本、破冰话术或心理疏导策略
- 班级存在群体性"怀旧前任、抗拒现任"的防御情绪，需突破核心阻力点

## When not to use
- **转交 home-school-trust-building**：问题涉及家长向学校投诉、家长会组织、家长群焦虑蔓延、家校沟通策略、致家长信撰写
- **安全转介**：学生表现出自伤/自杀倾向、严重抑郁或焦虑症状、涉及校园欺凌（无论受害者或施暴者）、严重行为问题（暴力攻击、长期旷课）超出教育谈话范畴——立即建议启动学校心理危机干预预案，暂停谈心策略
- **普通问答**：仅需一般性班级管理建议，无特定学生抵触场景

## Instructions
1. **先稳后破**：谈话前确保私密环境，避免当众施压；先建立情感安全区，再尝试视角转换
2. **零防御倾听**：前5分钟只倾听不解释，充分接纳怀旧情绪，避免"但是"式反驳
3. **优势具象化**：呈现自身专业优势时，必须用具体案例/数据支撑（如"上届班级平均分提升15%"），避免空泛自夸
4. **选择权后置**：仅在学生情绪平复后，使用"一个月尝试期"等契约话术，赋予自主决定权，重建心理掌控感
5. ** leader优先**：优先攻克原班干部、核心骨干，利用其影响力辐射群体

## Workflow
1. **对象识别**：确认核心抵触者（原班长、绩优生、非正式领袖），评估其抵触程度（沉默/议论/公开对抗）
2. **环境营造**：选择非正式场景（操场散步、午餐后教室），确保一对一私密性，降低防御
3. **倾听共情**：使用开放式提问（"能和我聊聊张老师的哪些做法让你怀念吗？"），全程不打断、不反驳，用点头/复述确认接纳（"我理解那种被熟悉感包围的安全感"）
4. **呈现双面**：客观承认差异（"我确实和张老师风格不同"），接着用事实展示专业优势（教学理念、过往成果、管理特色），引导学生看到"不同≠不好"
5. **赋予选择**：提出"一个月尝试期"契约："你愿意给我们彼此一个月时间互相了解吗？如果到期仍不适应，我们可以一起找校长沟通"，将被动承受转为主动选择
6. **后续锚定**：谈话后24小时内，在公开场合对该学生进行正向关注（点名回答擅长领域、肯定其某项特长），强化被重视感

## Examples
**输入**：原班长小骆在课堂上从不回应提问，课后与同学议论"以前张老师从来不这样管我们"，我需要和她谈话。

**输出**：

### 场景判断
- 目标学生特征：原班长，成绩优秀，性格强势，具有群体影响力
- 抵触表现形式：课堂沉默+课后怀旧议论，属于隐蔽对抗
- 风险等级：中（需优先破冰）

### 处理方案
**谈话时机**：放学后30分钟，避开同学视线
**环境选择**：空教室或操场角落，坐下谈，保持平视
**核心策略**：先倾听15分钟对前任的怀念→用具体教学成果展示另一面→赋予一个月尝试期的自主选择权

### 可用话术
**倾听阶段**："我注意到你最近不太开心，能和我聊聊张老师的哪些做法让你特别怀念吗？我想听听你的想法，绝不打断你。"
**转换阶段**："确实，我和张老师风格不同。同时我想让你了解，我带的上一届班级在一年内平均分提高了15分，我特别注重[具体优势]。你愿意看看这种不同能带来什么吗？"
**契约阶段**："我理解适应需要时间。这样好不好，我们约定一个月的尝试期，这期间你随时找我反馈，如果到期你仍然觉得不适应，我们一起去找校长沟通换班的事，好吗？决定权在你。"

### 风险提示
- 避免在谈话中批评前任教师，禁止说"以前老师做法不科学"等否定性评价
- 若学生提及自伤或严重情绪问题，立即停止谈话并转介心理老师

### 后续评估
- 观察指标：次日课堂是否回应提问、课后是否减少怀旧议论、是否开始提交作业
- 跟进周期：3天后课间简短回访（"这几天感觉怎么样？"）
- 稳定标志：主动参与课堂互动或接受班级任务分配

## Common Issues
1. **学生拒绝谈话**：先通过记住其姓名和特长（"我知道你书法很好"）建立初步连接，再以"不是批评，是想听听你的建议"降低戒备，避免强迫
2. **过度怀旧无法转移**：允许学生保留怀念，强调"可以同时怀念过去和接纳现在"，不要求学生否定前任才能接受现任
3. **"赋予选择"被当作把柄**：明确契约边界，"一个月尝试期"不等于无条件妥协，期间学生需遵守基本纪律，契约仅针对"是否适应"而非"是否遵守规则"
4. **群体效应反弹**：核心学生破冰后，其身边小团体可能继续抵触，需在一周内对核心圈其他成员进行第二轮次谈心
5. **优势展示空洞**：避免"我很负责""我很有经验"等抽象表述，必须准备具体数据/案例（如"我设计的分层作业让上届后进生及格率提升30%"）
