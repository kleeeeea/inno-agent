# 家校共育专业宣示

## When to use
- 家长群体出现向学校投诉、年级组长反馈担忧、家长群内焦虑蔓延
- 需要设计/召开"携手共育，平稳过渡"主题家长会
- 应对态度激烈家长（威胁换班、质疑能力）的一对一沟通
- 撰写致家长信、家校协作备忘录等正式文本

## When not to use
- **转交 student-trust-counseling**：问题聚焦学生个体抵触（课堂沉默、核心学生带头对抗、怀旧言论）、设计与学生谈话话术、班级建议箱管理
- **转交普通问答**：仅需常规家校沟通技巧（非过渡危机场景）
- **安全转介**：学生出现自伤/自杀倾向、严重抑郁、校园欺凌、暴力攻击等心理危机，立即启动学校危机干预预案而非沟通策略

## Instructions
1. **先稳情绪，后展专业**：先接纳焦虑，再用教育规律与专业资质建立信任
2. **公开透明，双向互动**：家长会设置开放问答，避免单向说教
3. **分层沟通，重点突破**：群体会议后，对激烈家长必须一对一跟进
4. **契约精神，持续可见**：建立常态化沟通机制（线上答疑、定期反馈），用可验证承诺替代空头保证

## Workflow
1. **家长会设计**：构建"携手共育，平稳过渡"主题议程（开场说明→亲师信道原理→专业优势展示→开放问答→合作机制建立）
2. **专业宣示**：系统展示教育经历、教学成果、带班理念三类优势，提供可验证证据
3. **深度沟通**：会后对激烈家长执行"倾听诉求→展示诚意→提出行动"三步一对一沟通
4. **机制固化**：发布《家校共育平稳过渡弹性协作备忘录》，建立线上常态化答疑与定期反馈渠道
5. **动态监测**：每两周评估家长信任度回升率、换班诉求撤销率，调整沟通策略

## Output Format
- **场景判断**：当前家长危机等级（焦虑/质疑/对抗）与核心诉求识别
- **处理方案**：家长会流程或一对一沟通策略（含具体步骤）
- **可用话术**：关键节点发言脚本（开场、原理阐释、优势展示、冲突回应）
- **文档模板**：致家长信、会议纪要、协作备忘录等文本框架
- **风险提示**：可能引发家长反弹的敏感词与行为禁忌
- **后续评估**：信任重建指标监测与下一轮沟通时机

## Examples
- **示例1**：家长会上回应"孩子不适应新老师"质疑的完整发言要点（含亲师信道原理阐释+三类专业优势展示）
- **示例2**：应对"找校长换班"威胁家长的一对一沟通三步策略与话术

## Common Issues
- **误区1**：过度贬低前任教师以抬高自己，引发家长对教师职业操守的质疑
- **误区2**：家长会上回避尖锐问题，导致信任危机加剧
- **误区3**：承诺过于具体（如"下次考试必须提高10分"），一旦失败彻底失信
- **误区4**：忽视家委会核心成员的个体沟通，导致群体情绪持续发酵
