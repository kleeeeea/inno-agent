# 现场应急与证据固化

## Instructions
- **并行操作**：证据收集与送医急救同步进行，抢占黄金处置窗口
- **中立立场**：采用无引导式询问，禁止诱导性提问或责任预设
- **完整闭环**：确保证据链包含时间、地点、行为、结果四要素，可交叉验证
- **程序合规**：书面自述需学生手写"以上属实"并签名，录音需告知用途

## Workflow
1. **立即隔离**：分离肇事学生与目击学生，防止情绪激化与信息串供，指定专人看护
2. **交叉询问**：选取2名以上目击学生，按"你看到了什么→当时谁在场→事情先后顺序"中立提问，比对证词一致性
3. **证据固化**：获取书面自述报告（手写+签名+时间戳）或合规录音，标注取证地点、见证人、文件编号
4. **初步定责**：基于证据链客观还原事实经过，形成《现场情况说明》，为家校调解提供事实基础

## Output Format
- **证据清单表**：目击者姓名、年级、证词核心内容、取证形式（书面/录音）、取证时间
- **事实摘要**：时间轴（精确到分钟）+ 行为链（动作序列）+ 损伤点（部位与致伤物）三段式客观描述
- **风险提示**：标注证据缺口（如盲区角度）、潜在争议点及需补充的物证（如鞋印、物品损坏情况）

## Examples
**高质操作**：学生楼道碰撞骨折，班主任B在陪同送医途中，电话指令副班主任隔离双方并询问3名旁观学生，获取2份书面自述+1份录音，交叉验证"奔跑方向-碰撞部位-倒地姿势"一致性，形成闭环证据链，成功应对后续家长否认。

**低质操作**：教师仅口头询问"他是不是推你了"，未隔离双方导致串供，无书面记录仅依赖回忆，3日后家长否认时无据可查，陷入各执一词僵局。

## Common Issues
- **诱导询问**：使用"他是不是故意的"等暗示性语言，导致证据法律效力受损
- **延迟固化**：优先安抚家长情绪后补证据，错过24小时最佳取证窗口，目击学生记忆模糊或受家长影响
- **程序瑕疵**：未让学生手写"以上属实"并签名，或未经同意录音，证据被质疑真实性
- **单方采信**：仅收集受害方或肇事方单一证词，未交叉验证即形成结论，后期出现归因反转
