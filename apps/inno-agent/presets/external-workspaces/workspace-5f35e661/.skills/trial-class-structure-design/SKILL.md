# 首课结构设计与学情诊断

## When to use
- 需要设计40-60分钟一对一试听课整体时间轴与环节分配（学情沟通→聚焦教学→反馈规划）
- 准备首课5-7分钟学情沟通阶段的具体问题清单与信息收集逻辑（覆盖教材、成绩、失分点、偏好、兴趣）
- 判断学生是A类（意愿强/关注效率）还是B类（意愿弱/被动补课）并匹配对应教学风格与语言节奏
- 学情未知时制定"多备一手"备课策略，准备两套不同难度或题材内容以便现场切换

## When not to use
- 课堂中如何设计"1讲3练"具体题目、验证学生真实掌握度（区分"本来会"与"教后掌握"）→ 交给 `teaching-validation-feedback`
- 课后给家长的诊断-规划-预期管理反馈话术撰写 → 交给 `teaching-validation-feedback`
- 家长要求"保证一个月提XX分"等虚假承诺 → 明确拒绝并提醒遵守《中小学教师职业道德规范》
- 学生表现出严重厌学情绪或心理困扰 → 建议暂停学科教学，优先关注心理状态并转介专业心理咨询

## Instructions
- **首课核心目标**：呈现教师与学生的契合度（方法、风格匹配），而非讲完所有内容
- **三段式结构**：0-7分钟学情沟通→7-45分钟聚焦教学（1个点做透）→45-60分钟反馈规划
- **学情五维覆盖**：教材版本/年级进度、最近考试分数、最常失分板块、讲解偏好（先讲后练/边做边讲）、兴趣爱好（用于举例互动）
- **学生类型识别**：A类关注专业度与效率（节奏紧凑、逻辑清晰）；B类关注能否听懂+课堂趣味（内容基础、语言生活化、轻量幽默）
- **备课双轨制**：准备两套不同难度或题材内容（如语文备古诗+文言文；数学备基础+拔高），确保现场可根据学生反应30秒内切换

## Workflow
1. **生成三段式时间轴**：明确每阶段起止时间（0-7'/7-45'/45-60'）与核心目标（建立信任/诊断问题/呈现价值/规划后续）
2. **构建学情问题清单**：设计5-7分钟可落地问题，按"教材→成绩→失分点→偏好→兴趣"顺序收集，确保信息可转化为教学决策
3. **识别学生类型**：根据前3分钟互动表现判断A类（主动提问/关注提分/眼神专注）或B类（沉默/抗拒/低头），即时匹配教学风格与节奏
4. **制定备课预案**：准备两套内容（难度差异或题材差异），明确切换触发条件（如第一题卡壳即换基础版；进展顺利则上拔高版）

## Output Format
- **场景判断**：学生类型（A类/B类）与学情特征摘要（基础水平/学习意愿/互动表现）
- **时间轴设计**：40-60分钟三段式分配（各环节起止时间、核心目标、关键动作）
- **问题清单**：可直接照读的5-7分钟学情沟通问题（含追问逻辑与信息记录栏）
- **教学风格匹配**：A类（高效紧凑/专业术语/密集闭环）或B类（趣味参与/生活语言/小成功体验）的具体策略差异
- **备课方案**：两套备选内容（难度/题材差异）、同类练习题准备、切换触发条件与话术
- **风险提示**：学情信息不足时的应对建议、禁止虚假承诺的合规提醒、基础极差学生的预期管理

## Examples

### 示例1：标准60分钟首课结构
**场景判断**：学情未知，需通用框架，待现场识别学生类型

**时间轴设计**：
- 0'-7' 学情沟通：收集教材版本、最近分数、失分板块、讲解偏好、兴趣爱好，建立初步信任
- 7'-45' 聚焦教学：选1个代表性知识点（如阅读理解作用题）讲透→2-3道同类题练习→学生复述思路（梯度：中等→偏难→回基础）
- 45'-60' 反馈规划：向学生/家长呈现诊断结论+后续学习路径+预期管理（避免硬承诺提分）

**问题清单**：
1. "你现在用的教材是人教版/部编版还是其他？进度到哪里？"
2. "最近一次大考大概多少分？哪一块失分最多（阅读/作文/基础）？"
3. "你觉得最难的是哪类题型？自己分析过原因吗？"
4. "你喜欢老师先讲方法再练，还是边做边讲？"
5. "平时喜欢什么运动/音乐/游戏？（用于课堂举例）"

**教学风格匹配**：若学生主动问"老师你能帮我提分吗"→按A类执行（紧凑专业）；若学生低头说"我妈让我来的"→切换B类（基础+幽默）

**备课方案**：语文备"古诗鉴赏（中等）+文言文断句（基础）"，若古诗反应好则拔高，卡壳则换文言文建立信心

### 示例2：B类学生（意愿弱）专项设计
**场景判断**：初三学生，家长逼来补课，进门低头抗拒，基础薄弱

**时间轴调整**：压缩学情沟通至5分钟，快速进入低难度互动建立"小成功"体验

**教学风格匹配（B类）**：
- 语言：生活化、轻量幽默（"我们先做个超简单的，热热身"），避免高密度术语
- 节奏：内容更基础，多鼓励，重在建立课堂好感与参与感，而非知识密度
- 目标：让学生觉得"听得懂+不痛苦"，愿意继续上课

**备课方案**：准备"超基础版"（小学衔接知识点）和"常规版"（初三基础题），首题选择超基础版，确保学生能完成以建立信心

## Common Issues
- **误区1**：试图60分钟讲满证明专业，导致学生吸收差、体验累。→ 坚持"少讲多练"，聚焦1个代表性点做透即可
- **误区2**：仅听家长描述不与学生核对，诊断偏差。→ 必须亲自用问题清单问学生，验证失分点和自我感受
- **误区3**：未区分A/B类学生，对被动学生用高密度讲授。→ 首课前3分钟快速判断类型，B类学生必须先建立好感再谈知识
- **误区4**：学情未知只备一套内容，遇到基础极差学生无法应对。→ 必须"多备一手"，准备两套难度/题材，确保现场可切换
