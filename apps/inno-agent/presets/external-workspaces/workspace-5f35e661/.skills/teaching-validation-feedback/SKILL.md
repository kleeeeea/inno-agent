# 闭环教学验证与课后反馈

## When to use
- 课堂执行阶段：需落实"1道典型题讲透+2-3道同类题练习+学生复述思路"的少讲多练闭环
- 梯度选题决策：需根据学生反应执行"中等→偏难→回基础"的动态难度调整
- 真实掌握验证：需区分学生"本来就会"还是"教后掌握"，避免假性教学成果
- 家长沟通场景：课后需生成结构化反馈（诊断结论+学习规划+预期管理）支持续费决策
- 课堂突发调整：学生反应与预期不符时，实时切换A类（高效紧凑）或B类（趣味参与）教学模式

## When not to use
- 首课整体时间轴设计、学情沟通问题清单、学生类型预判与备课策略 → 请使用 `trial-class-structure-design`
- 仅需通用教学方法论而非试听首课特定场景 → 使用普通问答
- 学生表现出严重厌学情绪、心理困扰或需专业心理咨询 → 建议暂停教学并转介专业心理支持
- 家长要求虚假分数承诺或违反《中小学教师职业道德规范》的不当要求 → 明确拒绝并提供合规话术替代

## Instructions
1. **少讲多练执行**：每讲1个核心方法，必须配2-3道同类题即时练习，强制学生复述解题思路，完成"输入-输出-验证"闭环
2. **梯度诊断逻辑**：首题选中档难度试探，做对了上偏难题挑战，卡住了回基础题补漏，通过"试探-验证-回退"定位真实水平
3. **真伪掌握识别**：观察学生讲题时的逻辑完整性vs答案正确性；要求复述"第一步看什么/为什么这么做"，区分记忆答案vs理解方法
4. **动态风格切换**：A类学生（意愿强）压缩闲聊、增加思维密度；B类学生（意愿弱）降低起点、嵌入兴趣例子、强化小成功体验
5. **预期管理底线**：禁止承诺具体提分幅度或排名，使用"可见进步指标"（如掌握某类题型方法）替代结果承诺

## Workflow
1. **课堂执行验证**：讲透1道代表题 → 学生独立做同类题1 → 纠错讲解 → 同类题2巩固 → 强制复述思路 → 判断真实掌握度
2. **梯度动态调节**：根据首题正确率与耗时，决定下一题升难度（做对了且思路清晰）、保同级（做对但迟疑）或降难度（做错/超5分钟无思路）
3. **实时风格校准**：观察学生眼神/回应速度，A类保持快节奏逻辑推进，B类插入生活化例子/幽默/鼓励性语言降低焦虑
4. **课后反馈生成**：诊断结论（具体薄弱点+学习习惯观察）→ 阶段规划（先补基础X点→再突破Y题型）→ 预期管理（避免虚假承诺，强调方法习得）
5. **续费决策支持**：明确后续课程路径设计（基础补漏→能力提升），提供1-2节后续课的具体内容预览，增强家长对长期价值的信心

## Output Format
- **场景判断**：当前课堂阶段（教学中段/练习验证/课后沟通）与学生实时状态（流畅/卡顿/抗拒）
- **处理方案**：具体执行动作（选题难度/教学风格切换/反馈结构）与决策依据
- **可用话术**：教师可直接使用的引导语、提问句、家长沟通模板
- **风险提示**：需规避的教学误区（如过度承诺、假性掌握误判）与合规边界
- **后续评估**：本环节达成的验证目标与下一环节衔接建议

## Examples

**场景：家长要求"把10道错题全讲完"**
- **场景判断**：家长误解首课目标为"讲题数量"而非"呈现匹配度"
- **处理方案**：重构为"精讲1道代表题+同类验证"模式，说明盲目刷题无法验证教学效果
- **可用话术**："这10道题里我挑最有代表性的1道把方法讲透，然后让孩子练2道同类型的，看是不是真的掌握了方法。这样比赶时间讲完10道更有价值。"
- **风险提示**：避免被家长节奏带偏，坚持"少讲多练"原则，防止课堂变成单向灌输
- **后续评估**：观察学生针对同类题的正确率，作为续费沟通时的教学有效性证据

**场景：学生做题全对，判断是"本来就会"**
- **场景判断**：学生完成快、无迟疑、但无法清晰复述"为什么选这个思路"
- **处理方案**：立即上梯度偏难题，或更换题型本质相同但外壳变化的题目，测试迁移能力
- **可用话术**："这道题做得很快，说明你基础不错。我们换个难一点的/变个形式考考灵活性，看看这个方法能不能迁移。"
- **风险提示**：勿将"本来就会"误判为"教得好"，否则后续课程会暴露真实水平导致家长信任崩塌
- **后续评估**：若偏难题仍做对且思路清晰，确认学生水平高于预期，调整后续规划为拔高方向

**场景：课后家长反馈话术模板**
- **诊断结论**："通过今天的测试，发现孩子在**文言文虚词辨析**上存在系统性薄弱（具体表现：...），但**现代文阅读**的抓取能力较好。"
- **学习规划**："接下来4节课我们先集中突破**虚词6大高频考点**（基础补漏），再进入**文言文整体阅读策略**（能力提升）。"
- **预期管理**："不提具体分数承诺，但我们可以设定**可见目标**：第3节课结束时，孩子能独立辨析这6类虚词，作业错误率从现在的X降至Y。"

## Common Issues
- **只讲不练**：教师担心"讲少了显得不专业"，导致60分钟单向输出，学生无验证机会，家长看不到"教会"的证据
- **题目过易陷阱**：选择学生本来就会的简单题，造成"全会了"的假象，续费后正课暴露真实水平引发退费
- **虚假承诺压力**：家长施压要求保证月考提分，教师为成交而妥协，违反职业道德且埋下退费纠纷隐患
- **风格错配**：对B类被动学生使用A类高密度逻辑轰炸，或对A类高效学生过度闲聊，导致体验错位
- **反馈空洞**：课后只说"孩子挺好的/基础薄弱"，缺乏具体诊断点和清晰路径，家长无法评估续费价值
