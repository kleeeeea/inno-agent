# 行为观察记录与证据采集

## When to use

- 家长沟通前缺乏具体行为数据，仅有"不专注""爱说话"等主观印象，需建立事实基础
- 需设计可量化行为记录系统（频次、时长、时间点）以应对"他说他没有"类否认
- 需将抽象品质标签（如"懒惰""态度差"）转化为可观察动作（如"抠键盘、转头说话"）
- 需采集客观证据（监控、邻座反馈）避免"老师我看到"的主观性争议
- 处于师带徒保护期的新教师需要建立专业自信的事实基础

## When not to use

- **已处于与家长沟通对抗阶段**，需要应对"他说他没有说话"类否认话术（应转至 `factual-communication-strategist`）
- **需处理家长以学历/社会地位施压**的权威边界建立或心理止损策略（应转至 `factual-communication-strategist`）
- **发现高风险信号**：学生身体虐待痕迹、自伤自杀表达、家长暴力威胁、教师严重心理崩溃（立即上报学校管理层/德育处/心理老师，禁止仅提供沟通技巧）
- **涉及法律敏感事项**：校园欺凌定性、法定监护权争议、特殊教育鉴定（立即转介校级领导，超出教师个人权限）

## Instructions

1. **行为分解**：将"不专注"转化为3-5个具体可观察动作（如"抠键盘、藏鼠标、转头说话"），剔除"态度差"等品质标签
2. **频次量化**：记录具体数据（如"第3节课干扰同桌4次，每次持续2-3分钟"），避免"有时""经常"等模糊表述
3. **证据客观化**：优先利用教室监控、邻座学生证言，避免仅依赖"老师我看到"的主观观察
4. **时间戳记录**：标注具体时间点（如"10:15-10:25期间"），确保可与家长核对
5. **区分维度**：仅记录课堂行为（干扰他人、持续说话），不涵盖学业成绩评价或品德定性

## Workflow

1. **情境确认**：确认当前处于**课前准备阶段**（尚未与家长沟通），非对抗中
2. **行为分解**：将主观评价转化为具体动作清单，建立量化标准（频次/时长）
3. **模板设计**：制定行为日志模板（时间/行为描述/持续时长/客观证据来源）
4. **证据采集**：获取监控片段、邻座学生反馈等客观佐证材料
5. **风险筛查**：检查是否存在身体虐待、自伤等需立即上报的安全危机信号

## Output Format

- **场景判断**：当前处于课前准备/证据采集阶段，距离计划沟通时间
- **行为分解表**：抽象标签→具体动作（3-5项）+ 量化标准（频次/时长）
- **记录模板**：包含时间戳、行为描述、持续时间、客观证据来源的标准化日志
- **风险筛查结果**：是否存在需立即上报的安全/法律风险（是/否，具体描述）
- **转介建议**：证据充足后转至事实性沟通策略的具体标准（如"连续3天记录达5次以上"）

## Examples

**正面示例（行为分解）：**
- 原标签："上课不专注"
- 具体动作："数学课10:15-10:25期间转头与后排说话3次，每次20-30秒；用手指抠键盘5次，持续10秒/次；邻座学生XX可证实"

**负面示例（避免）：**
- 主观评价："我觉得他态度不好，故意捣乱"
- 模糊记录："他经常爱说话，有时影响别人"

## Common Issues

1. **记录过于笼统**：使用"有时""经常"而非"3次/每节课"等具体频次
2. **混入主观评判**：记录"故意对抗""态度恶劣"等无法观察的心理动机
3. **证据单一**：仅依赖教师个人观察，未采集邻座证言或监控佐证
4. **标签混淆**：将行为问题（说话、走神）与学业成绩、品德评价混为一谈
5. **忽视时效**：记录缺乏具体时间戳，无法与家长核对具体事件
