# 事实性沟通与边界建立

## When to use
- 家长否认学生课堂行为问题（如"他说他没有说话""幼儿园老师都夸他"）
- 面对高学历（教授/医生/公务员）家长以学术权威质疑教师观察
- 沟通陷入"你说你的我听我的"无效循环，需用事实数据突破防御
- 需将"希望家长多关注"转化为可验证的具体配合事项
- 教师感到心理弱势或 intimidation，需建立专业权威与止损策略

## When not to use
- **转介 behavior-observation-recorder**：尚未建立行为日志、缺乏具体频次数据支撑，需先进行课前证据采集
- **安全转介**：发现学生身体虐待/自伤、家长暴力威胁/扬言起诉、严重精神异常或法律敏感事项（校园欺凌定性、监护权争议），立即上报学校管理层/德育处/心理老师，不提供沟通话术
- **普通问答**：仅需一般性教育建议或非冲突性家校沟通

## Instructions
- **用事实替代评价**：陈述"第3节课转头说话4次，每次2-3分钟"，而非"他不专注"
- **引入第三方权威**：联合班主任/资深教师共同反馈，形成专业合力背书
- **具体化协商**：将模糊要求转为可验证行动，如"每天检查数学书笔记是否完成（5分钟）"
- **建立专业边界**：明确"我们反馈在校具体行为，不是评价家庭教育"，区分幼儿园与小学环境差异
- **心理止损**：接受"讲不通"的现实，区分"尽力沟通"与"必须说服"，建立最低限度管理策略

## Workflow
1. **情境诊断**：判断家长否认类型（完全否认/部分承认/转移话题/身份压制）
2. **事实陈述**：出示行为记录（频次/时间点/持续时间/具体动作），应对"他说他没有"类否认
3. **权威建构**：引入资深教师/班主任共同参与，应对高学历家长的学术权威压制
4. **方案协商**：设定可验证改进目标（如"本周举手发言而非插话，周五反馈数据"）
5. **边界设定**：明确教师职责范围，接受形式化妥协，避免过度消耗，必要时启动"最低限度管理"

## Output Format
- **场景判断**：当前沟通阶段（开场/否认/僵持/收尾）与家长防御类型
- **处理方案**：具体应对策略（事实呈现/权威引入/协商技巧/止损点）
- **可用话术**：基于行为观察的陈述模板、应对否认的回应、边界建立语句
- **风险提示**：是否触发安全转介（暴力/法律/心理危机信号）
- **后续评估**：可验证的跟进指标（行为改善数据、家长配合度分级、教师心理状态）

## Examples

**输入**：家长说"他是教授说没有说话，幼儿园每个老师都夸他非常好，是不是你们老师要求太高？"

**输出**：
- **场景判断**：沟通对抗阶段，家长使用"过往评价否认+身份压制"双重防御
- **处理方案**：出示具体行为记录，引入班主任权威，区分环境差异，设定可验证目标
- **可用话术**：
  - 事实陈述："本周数学课上，10分钟内转头与同桌说话3次，这是课堂观察记录的具体时间点"
  - 权威引入："班主任（资深语文教师）也观察到类似情况，我们一起讨论如何帮助他适应小学规则"
  - 边界建立："幼儿园和小学的环境要求不同，我们反映的是在校具体行为表现，需要一起帮助他适应"
  - 协商具体化："能否每天检查数学书笔记是否完成？这只需要5分钟，本周五我会反馈记录"
- **风险提示**：无（属典型否认型沟通，未触及安全红线）
- **后续评估**：目标行为减少50%以上（深度配合），或接受换座位等形式化妥协（最低限度管理），教师心理状态：区分"尽力"与"说服"，避免内耗

## Common Issues
- **主观评价陷阱**：使用"不专注""态度差"等抽象标签，易被家长否认且无法反驳
- **证据不足时强硬**：仅凭"我看到"指责，遭遇反驳时无法回应，导致权威崩塌
- **接受形式化承诺**：接受"会积极配合"的空头承诺，未设定可验证的下一步行动（如未明确"每天检查笔记"的具体要求）
- **过度消耗**：陷入"必须说服"执念，忽视"放下助人情节，尊重他人命运"的心理止损原则
- **权威倒置**：因家长高学历/社会地位产生 intimidation，未能坚持专业观察事实
