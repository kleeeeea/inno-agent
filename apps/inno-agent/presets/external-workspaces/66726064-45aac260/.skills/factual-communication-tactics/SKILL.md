# 事实性沟通策略执行

## When to use
- 家长否认学生行为问题（"他在家不这样""他说他没有说话"），需用已采集的行为数据（频次、时间、具体动作）进行事实性回应
- 家长以学历/地位质疑教师权威（"我是教授，你经验不足"），需重构专业边界并建立"用事实说话"的沟通基调
- 家长表面承诺配合但无实际行动（"会积极配合"却无后续），需将模糊承诺转化为可验证的具体行动方案

## When not to use
- **数据缺失时**：若仅有主观印象（"他很皮""总是捣乱"）而无"行为-频次-时间-环境"四维记录，必须先使用 **behavioral-fact-recording** 完成证据采集，遵循"无数据不沟通"原则
- **高风险情形**：遇家长法律威胁（投诉教育局、起诉）、人身攻击（辱骂、肢体冲突倾向）、或职业报复风险（家长为校领导/教育主管部门人员），立即停止沟通策略建议，转介学校法务/工会/上级行政，优先保护教师安全
- **儿童保护危机**：学生透露家暴、虐待痕迹或严重自伤，转介德育处/心理老师，停止仅做行为记录

## Instructions
- **事实锚定**：所有回应必须基于已验证的观察数据（监控、同伴证言、时间戳），严禁在沟通中临时发明细节或假设诊断（如"可能有多动症"）
- **非对抗立场**：区分"反馈在校行为"与"评价家庭教育"，明确"我们共同帮助孩子适应小学规则"，避免将家长置于被审判位置
- **可验证承诺**：将"希望家长多关注"转化为"每天检查数学书笔记是否完成（5分钟）"，设定量化目标（"本周举手而非插话，每日记录周五反馈"）
- **止损机制**：接受"讲不通"的现实，区分"尽力沟通"与"必须说服"，必要时转为最低限度管理（如换座位）并保持记录，避免情绪内耗

## Workflow
1. **事实陈述**：出示具体行为记录（"本周数学课10分钟内转头说话3次"），附证据（监控时间点/同桌反馈），避免"我觉得""我认为"等主观开场
2. **边界重构**：当家长以身份施压时，引入第三方权威（资深班主任/师傅共同沟通），明确"我们反映在校具体表现，非家庭教育评价，小学与幼儿园环境要求不同"
3. **方案协商**：提出具体配合事项（非抽象大道理），设定可量化的本周目标与反馈机制（每日记录卡/周五汇报）
4. **固化与止损**：家长否认时坚持"我们会继续记录并反馈，这是教师职责"；若沟通无效，记录过程后转为课堂管理调整，保护教师心理能量

## Output Format
- **场景判断**：当前家长反应类型（否认/质疑/敷衍/配合）
- **核心事实**：引用已掌握的3条具体行为数据（行为-频次-时间）
- **沟通脚本**：分"开场事实陈述→边界澄清→具体方案→应对话术"四段
- **预设应对**：针对家长可能的3种反驳（"他在家不这样""你针对他""我很忙没时间"）的即时回应
- **可验证目标**：本周具体行动项、衡量标准、反馈时间节点
- **风险提示**：是否存在权力不对等/报复风险/法律威胁，是否需要第三方在场
- **教师自保**：心理边界设定（"尽力即可，非必须说服"）与后续止损方案

## Examples
**用户输入**：家长是大学教授，说"我家孩子以前都被老师夸，是你们要求太高"，否认孩子上课说话。

**输出**：
- **场景判断**：地位压制型否认（以过往评价否定当前适应问题）
- **核心事实**：①周二数学课9:05-9:15转头与同桌说话4次 ②周三信息技术课藏同桌鼠标1次（监控9:23）③同桌书面反馈"他总是转过来说话"
- **沟通脚本**：
  - 开场："这是本周课堂观察记录，周二9:05-9:15，XX转头与同桌说话4次，每次持续2-3分钟，这是同桌的书面反馈和监控时间点。"
  - 边界："幼儿园和小学课堂规则不同，我们反馈的是适应新环境的具体行为，不是评价您的教育成果。班主任X老师（资深）也观察到同样情况。"
  - 方案："本周目标：如需发言先举手而非直接插话。我会记录每日次数，周五放学前反馈给您，您只需每天问他'今天举手了几次'，1分钟即可。"
  - 应对否认："他说他没有说"→"这是当时监控时间，同桌也写下了具体对话内容。我们不必争论动机，只关注如何让这个行为减少。"
- **可验证目标**：本周举手次数从0次增至每日≥2次，每日记录，周五16:00短信反馈
- **风险提示**：家长为教授，存在学术权威压制，建议邀请班主任共同沟通，避免一对一
- **教师自保**：若家长坚持否认，记录本次沟通内容，转为座位调整等课堂管理措施，无需说服家长承认，只需履行告知职责

## Common Issues
- **主观回潮**：沟通中不自觉使用"他态度不好""不专注"等抽象标签，导致家长否认时无法反驳。必须坚持"转头说话""离开座位"等可观察动作描述
- **证据不足时强硬**：在家长否认时无法出示具体频次/时间，仅凭"我看到"争论。无数据时应暂停沟通，返回记录阶段
- **接受空头承诺**：满足于家长"会积极配合"的表态，未约定"每天检查什么""何时反馈"。必须将承诺转化为具体可验证的行动项
- **情绪对抗**：被家长质疑经验后陷入自证或对抗。应转向事实数据，让行为记录而非教师资历成为权威来源
