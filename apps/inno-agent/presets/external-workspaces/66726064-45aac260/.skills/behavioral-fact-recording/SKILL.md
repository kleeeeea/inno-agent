# 行为事实记录与证据采集

## When to use
- 教师仅有主观印象（如"他不听话""最近又变差了"）需要转化为具体可观察行为时
- 需要将抽象标签（"态度不端""不专注"）转化为可量化数据（频次、持续时间、具体动作）时
- 设计标准化的课堂行为观察记录表或评估现有证据有效性时
- 采集客观证据（监控时间戳、同伴证言、具体行为序列）建立事实基础时

## When not to use
- **应转交 factual-communication-tactics**：已拥有具体行为数据（频次、时间、动作），询问如何与家长沟通或家长否认时的回应策略
- **应转交 factual-communication-tactics**：询问"该观察什么行为"之外的具体话术设计、边界重构或方案协商
- **安全转介情形**：发现儿童保护危机（家暴、虐待痕迹）、校园欺凌定性、疑似神经发育障碍（自闭症/ADHD），立即停止记录建议，转介学校德育处/心理老师/专业评估
- **普通问答**：仅需一般性教学建议而非行为观察专业工具时

## Instructions
1. **剔除主观标签**：将"不专注"转化为"转头说话""离开座位""注视窗外"等可观察动作；将"很调皮"转化为"插话次数""干扰同桌频率"等量化指标
2. **执行四维记录**：强制包含行为（具体动作）、频次（次数/持续时间）、时间（课程/日期/时段）、环境（座位位置、活动类型）四个维度
3. **证据有效性评估**：优先监控记录、时间戳、多来源同伴证言；避免单一主观印象；明确标注信息缺口并引导补充
4. **禁止诊断标签**：绝不使用"可能ADHD""疑似自闭"等医学推断，保持行为描述纯粹性

## Workflow
1. **主观标签识别**：识别用户输入中的主观评价词汇（如"态度差""不听话"）
2. **行为解构转换**：将抽象评价拆解为具体可观察动作（身体动作、语言内容、物品操作）
3. **四维数据补全**：补充频次（几次/几分钟）、时间（哪节课/哪天）、环境（座位/活动类型）缺失维度
4. **证据链评估**：评估当前证据有效性（是否有时间戳、第三方佐证），指出缺口并建议采集方式
5. **记录表生成**：输出标准化观察模板或具体记录示例

## Output Format
**场景判断**：当前处于数据缺失阶段/证据待完善阶段

**行为解构方案**：
- 原标签：[主观评价]
- 转化后：[具体可观察动作列表]

**四维记录模板**：
- 行为：[具体动作]
- 频次：[次数/持续时间]
- 时间：[课程/日期/具体时段]
- 环境：[座位位置/活动类型/触发条件]

**证据有效性评估**：[高/中/低] - [具体理由]
**信息缺口提示**：[需补充的数据维度]
**风险提示**：[如适用，提示避免医学诊断标签]
**后续动作**：[下一步应路由至 factual-communication-tactics/继续观察补充数据/安全转介]

## Examples

**示例1：主观标签转化**
- 输入："学生今天又不听课，怎么记录才客观？"
- 输出：
  - 原标签："不听课" → 转化："数学课上第3-5分钟注视窗外（持续2分钟）→ 第10分钟抠橡皮（持续3分钟）→ 第15分钟转头与后排说话（持续1分钟）"
  - 四维记录："行为：注视窗外、抠橡皮、转头说话；频次：3次干扰行为，累计6分钟；时间：2024年3月15日第2节数学课（9:00-9:40）；环境：靠窗第三排，独立练习时段"
  - 证据缺口：建议下次记录时添加同桌证言或监控时间点

**示例2：证据有效性评估**
- 输入："这样的观察算有效证据吗？我看到他总是爱说话"
- 输出：
  - 有效性评估：低 - 仅教师单方面主观观察，无时间戳、无频次统计、无第三方佐证
  - 改进方案：连续3天记录具体说话时间点（如"9:15插话1次，9:23与同桌交谈2分钟"），添加邻座学生反馈确认
  - 风险提示：避免使用"总是"等模糊词汇，需具体化

## Common Issues
1. **过度推断陷阱**：将"频繁离座"推断为"多动症倾向"或"家教不严" - 应坚持纯行为描述，剥离因果归因
2. **信息缺口忽略**：仅记录"上课说话"但缺失具体时间、频次、环境 - 应明确标注缺失维度并指导补充采集
3. **证据有效性误判**：将"我记得很清楚"视为有效证据 - 应要求客观佐证（监控、同伴证言、时间戳记录）
4. **事实编造风险**：为填补记录空白而假设具体频次（如"大概5次"） - 应标注为"待观察"，绝不虚构数据
