# 生成顺延申请信
## When to use
*   用户明确要求“生成”、“起草”或“撰写”一份培训顺延申请信。
*   用户提供了生成申请信所需的关键信息，如申请人、单位、培训名称和顺延理由。
*   用户请求一个包含占位符的培训顺延申请信模板。

## When not to use
*   用户要求从现有文档中“提取”信息（应路由至 `extract-postponement-info`）。
*   用户请求审批、评估申请的合理性或提供政策咨询。
*   用户请求生成非培训顺延类的其他申请信（如请假、报销等）。
*   用户请求修改或润色已有的申请信，而非从头生成。

## Instructions
1.  **明确申请要素：** 提供申请人的姓名、工作单位、申请顺延的培训名称以及详细的顺延理由。
2.  **指定日期（可选）：** 如有特定申请日期要求，请一并提供。
3.  **语言规范：** 确保提供的信息准确无误，以便生成正式、规范的申请信。

## Workflow
1.  接收用户生成培训顺延申请信的请求。
2.  识别并提取请求中包含的申请人姓名、工作单位、培训名称、顺延理由及可选日期。
3.  根据提取的信息，构建符合正式公文格式的培训顺延申请信。
4.  输出完整、规范的中文申请信文本。

## Output Format
```
场景判断：生成培训顺延申请信
处理方案：已根据您提供的信息生成以下申请信。
可用话术：
尊敬的[接收部门，如：党委教师工作部]：

我是[申请人姓名]，来自[工作单位]。因[详细顺延理由]，我无法参加原定于[原培训日期，若有]的“[培训名称]”。特此申请将该培训顺延至下一期或后续合适时间参加。给您带来的不便，深表歉意。恳请批准！

此致
敬礼！

申请人：[申请人姓名]
工作单位：[工作单位]
日期：[申请日期]
风险提示：请仔细核对信中信息，确保准确无误。
后续评估：请确认此申请信是否符合您的要求，如有需要可提供修改意见。
```

## Examples
*   “请根据以下信息生成一份培训顺延申请信：申请人李可骞，单位上海智能教育研究院，申请顺延'第一阶段集中课程培训'，原因是无法参加。”
*   “我需要一份正式的培训顺延申请模板，包含申请人、单位、培训名称和顺延理由的占位符。”

## Common Issues
*   **信息不全：** 缺少申请人姓名、单位、培训名称或顺延理由等关键信息，导致无法生成完整的申请信。
*   **意图混淆：** 用户请求的不是生成申请信，而是其他操作（如信息提取或政策咨询）。
*   **格式要求：** 用户对申请信的格式有特殊要求，但未在请求中明确说明。
