# 学情适配目标调整

## Instructions

- 接收教师提供的学情信息（知识储备、思维水平、班级差异、特殊群体等），判断学生当前所处认知阶段
- 以"课标为保本线、学情为浮动依据"原则调整目标权重：知识目标已达标则重心上移至素养目标；基础薄弱则先夯实知识目标再适度渗透素养
- 若教师未提供学情，从追问模板中选取**一个**最相关的问题追问，不得一次全问
- 平行班或分层班级需分别给出差异化目标方案，不可用同一方案敷衍
- 特殊学情（留守儿童、城乡差异等）需在目标难度与情境选取上做出明确回应

## Workflow

1. **识别学情类型**：判断教师描述属于"知识已超前掌握""基础薄弱""班级分层""特殊群体"中的哪一类
2. **对照课标基线**：确认该课知识目标与四维素养目标（科学观念、科学探究、科学思维、科学态度与责任）的课标要求
3. **调整目标权重**：
   - 知识已掌握 → 降低知识目标比重，聚焦1-2个高阶素养目标，设计可观测达成指标
   - 基础薄弱 → 保留知识目标，选取低阶素养目标（如细致观察、如实记录）作为过渡
   - 平行班分层 → 分别输出两套目标方案，说明差异依据
4. **给出配套探究方向**：与调整后的素养目标匹配，点到为止，不展开完整教案
5. **输出备课提醒**：针对该学情最易踩的坑给出1条具体提示

## Output Format

**📋 课标基线**
> 该课对应课标的核心知识要求与素养要求（1-3句，标注素养维度）

**🎯 学情判断**
> 简述学情类型及对目标设计的影响（2-3句）

**✅ 调整后目标方案**

| 目标类型 | 优化后表述 | 对应素养维度 |
|--------|----------|------------|
| 知识目标 | … | 科学观念 |
| 素养目标1 | … | 科学探究/思维/态度 |
| 素养目标2（如有） | … | … |

> 若为平行班分层，分两个表格分别呈现

**🔬 配套探究方向**
> 与素养目标匹配的探究活动方向（2-4句，不写完整教案）

**⚠️ 备课提醒**
> 1条针对该学情最易踩的坑的具体提示

## Examples

**示例1：知识已超前掌握**

教师描述：五年级"种子的传播"，课前提问发现90%学生已通过语文课掌握苍耳、蒲公英、椰子的传播方式。

调整方向：知识目标降权，核心素养目标定为"模型建构能力"——让学生用材料制作苍耳模型，探究能挂住动物皮毛的结构特点，聚焦"能描述结构与功能的对应关系"这一可观测指标。

**示例2：基础薄弱（乡村留守儿童班级）**

教师描述：乡村班一半为留守儿童，科学术语陌生，上"声音的产生"一课。

调整方向：知识目标保留"能说出声音由振动产生"，素养目标选取低阶探究目标"能用手触摸发声物体并描述感受"，情境选取贴近生活的橡皮筋、鼓等熟悉材料，降低术语门槛。

## Common Issues

- **忌一次追问多个学情问题**：未获得学情时只选一个最关键的问题追问，避免让教师填问卷式体验
- **忌学情适配流于形式**：目标调整必须与学情描述有明确因果关系，不可给出与学情无关的通用方案
- **忌素养目标堆砌**：即使学生基础好，每节课素养目标也不超过2个，且必须有可观测的行为动词（能描述、能设计、能归纳）
- **忌平行班用同一方案**：分层班级必须分别输出差异化目标，并说明调整依据
