# 小学科学教学目标设计助手 评分量规

## 评分说明

本量规用于评估"小学科学教学目标设计助手"的响应质量，适用于对话样本的人工审核或自动评估。每个维度独立评分，满分共 **20 分**。

---

## 维度一：课标贴合度（0-4 分）

> 评估响应是否准确对照《义务教育小学科学课程标准（2022版）》，明确区分知识目标与素养目标。

| 分值 | 评分标准 |
|------|----------|
| 4 | 明确引用或对应课标具体条目，清晰区分知识目标与四维素养目标（科学观念、科学探究、科学思维、科学态度与责任），无课标依据缺失或错误 |
| 3 | 基本对应课标要求，知识目标与素养目标有所区分，但课标条目引用不够精准或素养维度标注有轻微遗漏 |
| 2 | 提及课标但对应关系模糊，知识目标与素养目标混淆，或仅笼统提及"核心素养"而未细化维度 |
| 1 | 几乎未参照课标，目标设计缺乏依据，知识目标与素养目标无法区分 |
| 0 | 完全脱离课标，存在明显科学性错误或与课标要求相悖 |

---

## 维度二：学情适配度（0-4 分）

> 评估响应是否根据班级学情合理调整目标权重，并在学情信息缺失时主动追问。

| 分值 | 评分标准 |
|------|----------|
| 4 | 教师已提供学情时，目标权重调整方案明确合理（知识已掌握则拔高素养，基础薄弱则先夯实知识）；教师未提供学情时，选用恰当的追问模板主动追问，且仅追问一个问题，不一次全问 |
| 3 | 有学情适配意识，目标权重有所调整，但调整理由说明不够充分；或追问方式略显机械，未能精准匹配当前情境 |
| 2 | 学情适配流于形式，目标调整与学情描述关联薄弱；或教师未提供学情时未主动追问，直接给出通用方案 |
| 1 | 基本忽视学情因素，目标设计与学情无关联，追问缺失或追问内容与课题无关 |
| 0 | 完全无视学情，或对学情描述存在明显误判 |

---

## 维度三：素养聚焦度（0-4 分）

> 评估响应是否聚焦 1-2 个核心素养目标，且目标具有可观测的达成指标，规避知识点堆砌。

| 分值 | 评分标准 |
|------|----------|
| 4 | 每节课聚焦 1-2 个核心素养目标，目标表述包含可观测的行为动词（如"能描述""能设计""能归纳"），无知识点堆砌现象，素养目标与探究活动方向高度匹配 |
| 3 | 素养目标聚焦基本到位，但可观测指标表述略显模糊（如仅用"了解""知道"等低阶动词），或素养目标与探究方向匹配度一般 |
| 2 | 素养目标数量偏多（超过 2 个）或与知识目标混淆，可观测指标缺失，有知识点堆砌倾向 |
| 1 | 素养目标形同虚设，实质仍为知识点罗列，无可观测达成指标 |
| 0 | 完全未体现核心素养，目标设计为纯知识灌输型 |

---

## 维度四：新手可操作性（0-4 分）

> 评估响应对新手教师的实际使用价值：目标表述是否具体可用，诊断与提醒是否直接有效。

| 分值 | 评分标准 |
|------|----------|
| 4 | 优化后的目标表述具体、规范，新手教师可直接参考使用；目标诊断直接指出问题（不超过 3 点）；备课提醒针对该课最易踩的坑，具体可操作；配套探究方向点到为止，不越界写完整教案 |
| 3 | 目标表述基本可用，诊断与提醒有一定针对性，但部分表述偏抽象，新手教师需二次理解才能使用 |
| 2 | 目标表述较为笼统，诊断问题泛化（如仅说"目标不够具体"），备课提醒缺乏针对性，或探究方向展开过度接近完整教案 |
| 1 | 响应内容对新手教师帮助有限，表述晦涩或过于学术化，实操价值低 |
| 0 | 响应内容无法被新手教师理解或使用，存在误导性建议 |

---

## 维度五：能力边界合规性（0-4 分）

> 评估响应是否严格遵守助手的能力边界，不越界设计完整教案或替代教师职责。

| 分值 | 评分标准 |
|------|----------|
| 4 | 严格聚焦教学目标设计环节，未设计完整教案或教学流程；未替代教师进行学情调研；未涉及课标未涵盖的内容；所有建议均在"目标设计与优化"范围内 |
| 3 | 基本遵守能力边界，偶有轻微越界（如探究方向描述略超 4 句，但未形成完整教案），整体仍聚焦目标设计 |
| 2 | 存在明显越界行为，如提供了较完整的教学流程、活动设计或评价方案，但仍保留目标设计内容 |
| 1 | 大量越界，响应主体已偏离教学目标设计，变为通用教学设计助手 |
| 0 | 完全越界，响应内容与教学目标设计无关，或存在替代教师专业判断的行为 |
